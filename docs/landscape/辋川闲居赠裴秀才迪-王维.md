## 📜 原文
```
寒山转苍翠，秋水日潺湲。
倚杖柴门外，临风听暮蝉。
渡头余落日，墟里上孤烟。
复值接舆醉，狂歌五柳前。
```
## 🎯 主题
《辋川闲居赠裴秀才迪》主要展现了王维隐居生活的闲适与宁静。诗中通过寒山秋水、落日孤烟等自然景物的描绘，营造出淡远宁静的意境，体现了诗人对自然的热爱和对隐居生活的满足。诗人以"倚杖柴门，临风听蝉"的形象，表现了超脱世俗的闲适心态。最后通过与友人裴迪的相聚，以接舆和五柳先生自喻，表达了隐士之间的知遇之情和精神共鸣。整首诗流露出诗人追求心灵自由、与自然和谐相处的生活态度，体现了王维作为山水田园诗代表人物的思想特色。
## 🏗️ 结构
《辋川闲居赠裴秀才迪》是王维的五言律诗，其结构特点体现了严谨的章法布局和精心的层次安排。

从章法布局看，全诗可分为三个部分：首联写远景，勾勒秋日山水整体画卷；颔联和颈联写近景与诗人活动，展现具体环境和生活状态；尾联写人事交往，点明赠友主题。这种"景-人-情"的三段式结构，层次分明，过渡自然。

在层次安排上，诗歌采用了多维度的递进方式：

空间层次上，由远及近展开。首联"寒山转苍翠，秋水日潺湲"描绘远山秋水；颔联"倚杖柴门外，临风听暮蝉"将视线拉近至门外；颈联"渡头余落日，墟里上孤烟"则聚焦于更近的渡口和村落；尾联"复值接舆醉，狂歌五柳前"回归到诗人与友人的近距离交往。这种空间上的层层推进，使画面具有立体感。

时间层次上，全诗以傍晚时分的时间线连贯展开。从日暮时分的山水景色，到夕阳西下，再到村落炊烟升起，最后是夜晚与友人相聚，时间线索清晰，构成一个完整的傍晚场景。

意境层次上，从自然景色的客观描绘，到诗人闲适生活的展现，再到与友人交往的主观感受，意境逐渐深入。特别是尾联运用典故，以"接舆"指裴迪，以"五柳"自比，既点明赠友主题，又深化了诗歌内涵。

此外，诗歌在结构上还体现出对仗工整的特点。颔联"倚杖柴门外，临风听暮蝉"与颈联"渡头余落日，墟里上孤烟"形成工整的对偶，形式整齐，音韵和谐。同时，诗中动静结合，既有静态的景物描写，也有动态的活动描写；虚实结合，既有实景描绘，也有典故运用，使诗歌内涵更加丰富。

整体而言，这首诗通过严谨的章法布局和多维度的层次安排，将秋日傍晚的景色、诗人的闲适生活与友人交往的愉悦情感有机融合，体现了王维诗歌"诗中有画"的艺术特色。
## ✍️ 语言风格
《辋川闲居赠裴秀才迪》是王维山水田园诗的代表作，其语言风格体现了"诗中有画"的艺术特色，具有以下特点：

一、用词特点：
1. 简洁精炼：全诗仅四十字，却勾勒出完整的秋日傍晚山水画卷。每个字都经过精心选择，如"转"字表现山色随时间变化的动态，"余"字捕捉落日将尽的瞬间，"上"字描绘炊烟升起的景象。
2. 色彩鲜明："苍翠"一词既表现山的颜色，又暗示秋天特点，形成视觉上的强烈冲击。
3. 意象丰富：诗中包含"寒山"、"秋水"、"落日"、"孤烟"等典型意象，构建清幽淡远的意境。
4. 声形兼备："潺湲"一词既描绘水流形态，又表现水流声音，体现了王维诗歌对声音的敏感捕捉。

二、修辞手法：
1. 对仗工整：前两联和第三联均采用对仗手法，如"寒山"对"秋水"，"转苍翠"对"日潺湲"，"渡头"对"墟里"，"余落日"对"上孤烟"，使诗歌结构严谨，音韵和谐。
2. 借景抒情：通过描绘秋日傍晚的山水景色，含蓄表达诗人闲适淡泊的心境。
3. 用典精妙：尾联运用"接舆"（春秋楚国隐士）和"五柳"（陶渊明自号）的典故，以接舆比裴迪，以五柳先生自比，既点明赠诗对象，又暗示隐士身份和志趣。
4. 意象叠加：通过视觉（寒山、秋水、落日、孤烟）、听觉（潺湲、暮蝉）等多种感官意象的叠加，营造立体的艺术效果。

三、语言风格：
1. 清新自然：诗歌语言不事雕琢，自然流畅，如"倚杖柴门外，临风听暮蝉"一句，平实中见真趣。
2. 含蓄蕴藉：诗人通过景物描写间接表达情感，不直抒胸臆，体现了中国古典诗歌"含蓄"的美学特征。
3. 意境深远：通过简练的语言创造出开阔深远的意境，特别是"渡头余落日，墟里上孤烟"一联，空间开阔，意境深远。
4. 情景交融：诗人将个人情感与自然景物融为一体，达到"物我两忘"的艺术境界。

四、表现手法：
1. 动静结合：诗中既有静态的景物描写（如"寒山"、"柴门"），也有动态的刻画（如"转苍翠"、"日潺湲"），动静结合，使画面更加生动。
2. 远近结合：从远处的"寒山"、"秋水"到近处的"柴门"、"暮蝉"，再到远景的"渡头落日"和近景的"墟里孤烟"，空间层次丰富。
3. 虚实结合：实景描写与典故运用相结合，既有对眼前景物的真实描绘，又有通过典故表达的深层含义。

这首诗充分展现了王维作为"诗佛"的艺术风格，语言清新自然，意境空灵深远，情景交融，体现了唐代山水田园诗的最高成就。
## 📚 文化札记
《辋川闲居赠裴秀才迪》是王维在辋川别业隐居时写给友人裴迪的诗作。以下是诗中重要的文化背景注释：

辋川别业位于今陕西蓝田县，是王维晚年购置的别墅，也是他隐居生活的中心。这里风景优美，王维与裴迪等友人常在此吟咏山水，创作了大量诗作，后编为《辋川集》，成为唐代文人隐居生活的典范。

裴迪是王维的好友，也是一位诗人。两人交往密切，常有诗歌唱和。王维有多首诗是写给裴迪的，反映了两人在文学和思想上的深厚友谊。

"寒山转苍翠，秋水日潺湲"中的"寒山"指秋天的山，因秋天气温渐寒而得名。"苍翠"形容山色青翠，"潺湲"则描绘水流缓慢、连续不断的状态，共同勾勒出秋日山水的变化。

"倚杖柴门外，临风听暮蝉"中的"倚杖"表现诗人隐居闲适的生活状态；"柴门"指用柴木做的简陋门扉，象征隐居生活的简朴；"暮蝉"即傍晚时分的蝉鸣，蝉在中国古典文学中常象征高洁，也常用来表现秋天的到来。

"渡头余落日，墟里上孤烟"描绘了一幅宁静的乡村黄昏图景。"渡头"是水边用于渡船的地方；"余落日"指落日的余晖；"墟里"指村落；"孤烟"则是村中升起的炊烟。

"复值接舆醉，狂歌五柳前"两句包含两个重要典故："接舆"是春秋时期楚国的隐士，姓陆名通，字接舆。他因不满当时的政治，佯狂避世，被称为"楚狂"。《论语·微子》记载他歌而过孔子，劝其隐退。这里王维用接舆比喻裴迪，暗示裴迪也是一位不拘世俗的狂士。"五柳"指陶渊明，东晋著名诗人和隐士。他在《五柳先生传》中自号"五柳先生"，因其宅边有五棵柳树。陶渊明是中国文学史上隐逸诗人的代表，王维在此以陶渊明自比，表明自己隐居的志向。

这首诗体现了中国古典文学中的隐逸传统，从春秋时期的接舆，到东晋的陶渊明，再到唐代的王维，隐逸一直是中国文人重要的精神追求。王维作为唐代著名的山水田园诗人，也是佛教徒，他晚年受到母亲和佛教思想的影响，逐渐淡泊名利，追求隐居生活，在这首诗中得到了充分体现。
## 🌅 创作背景
《辋川闲居赠裴秀才迪》创作于唐玄宗天宝年间（约公元742-756年），地点在王维位于陕西蓝田县南的辋川别业。辋川别业是王维精心经营的隐居之所，山水环绕，风景优美，成为他晚年创作的重要场所。

当时王维已步入中年，经历了官场的起落。他曾在朝廷担任右拾遗、监察御史等职，但因政治环境复杂，尤其是安史之乱（755-763年）中被俘并被迫接受伪职的经历，使他对仕途心生厌倦。此时的王维选择在辋川过着半官半隐的生活，心境超然淡泊，追求精神上的自由与宁静。诗中"倚杖柴门外，临风听暮蝉"正是他闲适生活的真实写照。

历史背景上，天宝年间的唐朝表面繁荣，实则暗藏危机。唐玄宗沉迷享乐，朝政由杨国忠等人把持，政治腐败。社会动荡不安，许多文人对官场失望，纷纷选择隐居。王维的隐居既是对乱世的逃避，也是对理想生活的追求。

创作动机源于王维与友人裴迪的交往。裴迪是当时文人，与王维交情深厚。诗中"接舆"指春秋时期楚国隐士，"五柳"则指陶渊明，王维借此表达对古代隐士的向往。诗中描绘的寒山、秋水、落日、孤烟等景象，均取自辋川自然风光，而"接舆醉"、"狂歌"则表现了与友人畅饮的欢乐场景，展现了王维对闲适生活的满足和对友情的珍视。
## 💭 情感基调
这首诗的情感基调以闲适淡泊为主，兼有宁静超脱与亲切自然的特质。诗人通过描绘辋川秋日的自然景色和隐居生活，表现出一种远离尘嚣、安于现状的平和心态。与友人裴迪的互动更增添了诗中温暖的情谊，使整体情感基调既有隐士的孤高，又不失人间的温情。

氛围特点上，全诗营造出清幽淡远、古朴典雅的意境。"寒山"、"秋水"、"暮蝉"、"落日"、"孤烟"等意象组合，勾勒出一幅宁静悠远的山水画卷。诗中既有自然景色的和谐统一，又有人物活动的点缀，形成了一种恬静中见生机、淡泊中显情趣的独特氛围，充分体现了王维"诗中有画"的艺术特色和山水田园诗的典型风格。
## 🔍 赏析
王维的《辋川闲居赠裴秀才迪》是盛唐山水田园诗的代表作，充分体现了"诗中有画"的艺术特色。诗中通过"寒山"、"秋水"、"落日"、"孤烟"等意象，勾勒出一幅秋日傍晚的山水画卷。诗人采用动静结合的手法，如"转苍翠"的动态与"听暮蝉"的静态相结合，使画面生动而富有层次感。同时运用远近结合的构图法：近景是"倚杖柴门外"的诗人，中景是"渡头余落日"，远景是"寒山"和"墟里上孤烟"，形成空间层次分明的立体画面。

这首诗语言简练凝练，字字珠玑。"转"字生动表现山色随季节变化的动态，"余"字精准捕捉夕阳西下的瞬间，"上"字形象描绘炊烟升腾的景象。诗中对仗工整，如"寒山转苍翠，秋水日潺湲"和"渡头余落日，墟里上孤烟"，形式整齐，音韵和谐。诗人还巧妙运用典故，"接舆"和"五柳"分别指代春秋隐士接舆和陶渊明，既丰富了诗歌文化内涵，又含蓄表达了对隐逸生活的向往。

诗中表达了诗人对闲适隐逸生活的热爱和对自然的亲近之情。"倚杖柴门外，临风听暮蝉"展现悠然自得的生活状态，"复值接舆醉，狂歌五柳前"则表达了对隐逸传统的认同和对先贤的敬仰。主题思想是追求心灵自由和精神超脱，通过描绘辋川闲居的景色和生活，表达对官场纷扰的厌倦和对田园生活的向往，同时也蕴含着诗人作为"诗佛"的禅宗思想影响。

这首诗代表了盛唐山水田园诗的最高成就，对后世山水诗创作产生深远影响。王维将绘画技法融入诗歌创作，开创"诗画合一"的艺术境界，成为中国文学史上的重要创新。诗中体现的"隐逸"主题影响了后世文人的精神追求和生活方式，其融合儒、道、佛三家思想的特点，也展现了盛唐文化的包容性和融合性。至今，这首诗仍以其精湛的艺术技巧、丰富的思想内涵和深远的文化影响，成为中国古典诗歌中的经典之作。
