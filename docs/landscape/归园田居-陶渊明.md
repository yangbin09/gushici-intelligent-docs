## 📜 原文
```
少无适俗韵，性本爱丘山。
误落尘网中，一去三十年。
羁鸟恋旧林，池鱼思故渊。
开荒南野际，守拙归园田。
方宅十余亩，草屋八九间。
榆柳荫后檐，桃李罗堂前。
暧暧远人村，依依墟里烟。
狗吠深巷中，鸡鸣桑树颠。
户庭无尘杂，虚室有余闲。
久在樊笼里，复得返自然。
```
## 🎯 主题
《归园田居》主要表达了陶渊明对官场生活的厌倦和对田园生活的热爱，体现了诗人"返璞归真"的人生哲学。诗中通过"误落尘网中"、"久在樊笼里"等表述，将官场比作束缚人的牢笼，表达了对世俗生活的疏离感。而"性本爱丘山"、"守拙归园田"则表明诗人天性热爱自然，选择回归简朴的田园生活。诗中描绘的"方宅十余亩，草屋八九间"等田园景象，以及"狗吠深巷中，鸡鸣桑树颠"的乡村生活画面，展现了一种宁静祥和、与自然和谐共处的生活状态。最后"复得返自然"点明了全诗主旨，即摆脱世俗束缚，回归自然本真，追求精神自由和内心宁静。这首诗不仅是对个人生活选择的表达，也是对当时社会现实的一种批判，体现了诗人不与世俗同流合污的高洁品格。
## 🏗️ 结构
《归园田居》是陶渊明田园诗的代表作，其结构特点体现了严谨的章法布局和清晰的层次安排。

从整体章法布局看，这首诗呈现出典型的"起承转合"结构，共分为四个部分，层次分明，层层递进。

第一部分（起）为前四句："少无适俗韵，性本爱丘山。误落尘网中，一去三十年。"这部分开篇点题，表明诗人天性不适应世俗生活，本就热爱自然山水，但不幸落入官场"尘网"三十年。这为全诗奠定了基调，交代了回归田园的内在原因和时间背景。

第二部分（承）为第5-8句："羁鸟恋旧林，池鱼思故渊。开荒南野际，守拙归园田。"诗人通过"羁鸟"和"池鱼"的比喻，强化了对自由的渴望和对自然的向往。"开荒南野际，守拙归园田"则表明了实际行动，承接上文，深化了回归田园的主题。

第三部分（转）为第9-16句："方宅十余亩，草屋八九间。榆柳荫后檐，桃李罗堂前。暧暧远人村，依依墟里烟。狗吠深巷中，鸡鸣桑树颠。"这部分是全诗的转折，从抒情转向描写，具体描绘田园生活景象。空间安排上由近及远：从自己的住宅到房前屋后的树木，再到远处的村落和人烟，最后是乡村生活的声音，构成了一幅立体的田园生活画面。

第四部分（合）为最后四句："户庭无尘杂，虚室有余闲。久在樊笼里，复得返自然。"这部分总结全诗，再次表达对田园生活的满足和对自然的向往，与开篇形成呼应，点明主题。

在层次安排上，这首诗呈现出由抽象到具体再到抽象的特点：从抽象的情感表达（"性本爱丘山"）到具体的行动描述（"开荒南野际"），再到具体的景象描绘，最后又回到抽象的情感总结（"复得返自然"），形成循环结构。

此外，诗中巧妙运用对比手法，如"尘网"与"自然"、"樊笼"与"园田"的对比，突出了诗人对自由生活的向往和对官场生活的厌倦。在描写田园景象时，动静结合，既有静态的景物描写，也有动态的声音描写，使田园生活更加生动鲜活。

这种严谨的结构安排，使《归园田居》既有情感的深度，又有形象的生动，充分表达了诗人回归田园的喜悦和对自然的热爱，成为中国古典诗歌中结构艺术的典范。
## ✍️ 语言风格
陶渊明的《归园田居》体现了其独特的语言风格特点，整体上呈现出质朴自然、清新淡雅的艺术特色。这首诗语言平实无华，却意蕴深远，充分展现了诗人"自然"的审美追求和"归隐"的人生理想。

在用词方面，陶渊明倾向于使用日常化、口语化的词语，如"草屋"、"狗吠"、"鸡鸣"等，使诗歌贴近生活，真实可感。同时，他的用词简洁凝练，如"少无适俗韵，性本爱丘山"仅十字就概括了诗人的本性。诗中自然意象丰富，如"丘山"、"羁鸟"、"池鱼"、"榆柳"、"桃李"等，体现了诗人对自然的热爱。此外，对比性词语的运用，如"尘网"与"自然"、"樊笼"与"旧林"等，形成鲜明对比，突出了诗歌主题。

修辞手法上，这首诗运用了多种技巧。比喻手法随处可见，如"尘网中"比喻官场生活，"樊笼"比喻世俗束缚，"羁鸟"和"池鱼"比喻自己被困的状态。对偶句式的运用使诗歌节奏和谐，如"羁鸟恋旧林，池鱼思故渊"、"榆柳荫后檐，桃李罗堂前"、"狗吠深巷中，鸡鸣桑树颠"等。借景抒情的手法通过描绘田园景色表达情感，如"暧暧远人村，依依墟里烟"描绘的宁静乡村景象，表达了诗人对田园生活的向往。拟人手法如"依依墟里烟"中的"依依"一词，赋予炊烟以人的情感，使景物更加生动。对比手法通过前半部分描述官场生活的束缚，后半部分描述田园生活的自由，形成鲜明对比，突出主题。

句式特点上，全诗采用五言句式，整齐划一，节奏感强。句式简洁明快，不拖沓，如"方宅十余亩，草屋八九间"直接明了。诗歌结构层次分明，先述本性，再述经历，然后描绘田园生活，最后抒发情感，条理清晰。首尾呼应，开头"少无适俗韵，性本爱丘山"与结尾"久在樊笼里，复得返自然"形成呼应，使诗歌结构完整。

在意象运用方面，陶渊明运用了丰富的自然意象，如丘山、旧林、故渊、榆柳、桃李等，体现了诗人对自然的热爱。田园意象如草屋、南野、远人村、墟里烟、深巷、桑树等，描绘了一幅宁静祥和的田园图景。束缚意象如尘网、羁鸟、池鱼、樊笼等，象征官场生活的束缚和压抑。自由意象如开荒、归园田、无尘杂、有余闲、返自然等，象征归隐生活的自由和闲适。这些意象的运用，使诗歌意境丰富，情感表达更加深刻。

情感表达方式上，陶渊明既直抒胸臆，如"少无适俗韵，性本爱丘山"直接表达自己的本性；又借景抒情，通过描绘田园景色表达对归隐生活的热爱；还通过对比抒情，突出对自由的向往；同时情感表达含蓄而不直露，如"户庭无尘杂，虚室有余闲"暗含对简朴生活的满足。

总体而言，《归园田居》语言风格质朴自然，清新淡雅，用词日常化、口语化，修辞手法丰富多样，句式整齐匀称，意象运用丰富，情感表达真挚感人，充分体现了陶渊明独特的诗歌艺术风格，是中国古代田园诗的代表作之一。
## 📚 文化札记
《归园田居》是东晋诗人陶渊明的代表作，创作于他辞官归隐之后。这首诗体现了中国隐逸文化的精髓，以下是其中重要的文化背景注释：

"尘网"：比喻官场和世俗社会的束缚。在魏晋时期，政治黑暗，士人常以"网"喻指官场的复杂与险恶，表达对自由的渴望。

"羁鸟恋旧林，池鱼思故渊"：运用传统意象，被囚禁的鸟思念原来的森林，池中的鱼向往深水。这一典故源自《庄子》，后世文人常用以表达对自由的向往，成为隐逸文学的经典意象。

"守拙"：道家思想中的重要概念，出自《老子》"大巧若拙"，意为保持本真朴实的本性，不随波逐流。陶渊明以此表明自己宁愿保持淳朴本性，也不愿在官场中虚伪周旋。

"南野"：指南方的田野。陶渊明归隐之地在浔阳（今江西九江），位于长江以南，属南方地域。东晋时期，南方相对安定，许多士人南迁避乱。

"暧暧远人村，依依墟里烟"：描绘了典型的江南农村景象。"暧暧"形容远处村落模糊不清的样子，"依依"形容炊烟袅袅上升的形态，展现了一幅宁静和谐的田园画卷。

"狗吠深巷中，鸡鸣桑树颠"：农村生活的典型场景。桑树在古代农村极为重要，与养蚕业密切相关，是农耕文明的象征。狗吠鸡鸣的描写源自《诗经》，是表现乡村生活的传统手法。

"虚室"：暗用《庄子·人间世》中"虚室生白，吉祥止止"的典故，比喻心境空明，不为外物所扰。体现了陶渊明受道家思想影响，追求内心宁静的精神境界。

"樊笼"：关鸟兽的笼子，比喻官场或世俗的束缚。这一意象与开头的"尘网"呼应，强化了诗人对官场生活的厌恶和对自由的渴望。

"自然"：不仅指自然界，更指人的本真状态。这一概念源自道家哲学，尤其是老子的"人法地，地法天，天法道，道法自然"思想，体现了陶渊明追求与自然和谐共处的生活理想。

陶渊明作为田园诗派的创始人，其归隐思想深受魏晋玄学和道家哲学影响。东晋时期政治腐败，社会动荡，许多士人对现实失望，选择隐逸生活。陶渊明的诗歌不仅表达了个人的生活选择，更代表了中国传统文化中"达则兼济天下，穷则独善其身"的人生哲学，对后世文学产生了深远影响。
## 🌅 创作背景
《归园田居》创作于东晋义熙元年（公元405年）之后，是陶渊明辞去彭泽县令职务，回到家乡浔阳柴桑（今江西省九江市）隐居时的作品。当时陶渊明已年过四十，经历了多次出仕与归隐的反复，最终下定决心彻底离开官场。

归隐初期，陶渊明的生活相对清贫，但精神上却获得了极大的自由和满足。他亲自耕种，过着"方宅十余亩，草屋八九间"的简朴生活。诗中"久在樊笼里，复得返自然"表达了他摆脱官场束缚后的轻松和喜悦。他的心境平和而满足，虽然物质生活简朴，但精神世界丰富，正如诗中所描述的"户庭无尘杂，虚室有余闲"。

陶渊明生活在东晋末年，这是一个政治动荡、社会混乱的时期。东晋王朝已经衰弱，朝廷内部权力斗争激烈，士族门阀把持朝政，政治腐败。同时，北方已被少数民族政权占领，汉族政权被迫南迁，形成了南北对峙的局面。在这样的社会环境下，许多知识分子对现实感到失望，陶渊明选择了归隐田园，不愿与世俗同流合污。

陶渊明创作《归园田居》的动机主要源于他对官场生活的厌倦和对田园生活的向往。诗中"少无适俗韵，性本爱丘山"表明他从小就与世俗格格不入，本性热爱自然山水。"误落尘网中，一去三十年"则表达了他对自己进入官场的后悔。他的灵感来源于自己归隐后的真实生活体验，同时也受到了老庄思想的影响，追求自然无为的生活方式。此外，他对古代隐士的敬仰也是他创作的重要灵感来源，如诗中"守拙归园田"就体现了他对简朴生活的坚守。
## 💭 情感基调
《归园田居》的情感基调主要表现为解脱、满足与喜悦。诗人通过对官场生活的否定和对田园生活的赞美，流露出摆脱束缚后的轻松与释然。全诗洋溢着对自然的热爱和对简朴生活的珍视，体现出一种淡泊名利、返璞归真的精神追求。

氛围特点则呈现为宁静、恬淡与自然。诗中描绘的田园景象平和安详，远村、炊烟、鸡鸣、狗吠等意象共同构筑出一幅远离尘嚣的乡村画卷。诗人笔下的生活环境朴实无华却充满生机，营造出一种与自然和谐共处的悠闲氛围，使读者感受到一种超脱世俗的宁静之美。
## 🔍 赏析
《归园田居》是陶渊明田园诗的代表作，通过对比手法营造了鲜明的意境。诗中前半部分描述诗人在官场中的不适应和痛苦，后半部分则描绘归隐田园后的宁静与自在。"羁鸟恋旧林，池鱼思故渊"以鸟和鱼作比，生动表达了对自然的向往；"暧暧远人村，依依墟里烟"通过远景描写，营造出朦胧温馨的乡村氛围。诗人还运用动静结合的手法，既有静态的景物描写，也有"狗吠深巷中，鸡鸣桑树颠"等动态生活场景，使画面更加生动立体。

这首诗的语言质朴自然，不加雕琢，体现了陶渊明"平淡自然"的艺术风格。全诗用词通俗易懂，却蕴含深刻哲理。对比是全诗最突出的修辞手法，"尘网"与"园田"、"樊笼"与"自然"的对比，突出了诗人对两种生活的不同感受。比喻手法精妙，排比句式增强了诗歌的节奏感，对偶手法随处可见，使诗歌结构严谨，音韵和谐。

《归园田居》表达了诗人对田园生活的热爱和对官场生活的厌倦。"少无适俗韵，性本爱丘山"直接表明了诗人的本性不适合世俗生活，而热爱自然山水。"误落尘网中，一去三十年"则流露出对过去在官场中虚度光阴的悔恨。全诗贯穿着诗人对自由生活的向往和对自然本真的追求，"久在樊笼里，复得返自然"点明了主题，表达了摆脱世俗束缚、回归自然的喜悦和满足。诗中还蕴含着对简朴生活的赞美，反映了诗人对物质享受的淡泊和对精神自由的珍视。

这首诗是陶渊明田园诗的代表作，也是中国文学史上田园诗的开山之作。它开创了中国文学中的田园诗传统，对后世文人产生了深远影响，王维、孟浩然等唐代山水田园诗人，都受到陶渊明的启发。这首诗体现了魏晋时期文人的隐逸思想和追求个性解放的精神，在政治动荡、社会混乱的东晋时期，陶渊明选择归隐田园，保持独立的人格和精神自由，这种选择对后世知识分子产生了深远影响。此外，诗中所体现的人与自然和谐相处的理念，对当今社会仍有重要的启示意义，这正是陶渊明诗歌永恒魅力所在。
