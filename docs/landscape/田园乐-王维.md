## 📜 原文
```
其一
厌见千门万户，经过北里南邻。
官府鸣珂有底，崆峒散发何人。

其二
再见封侯万户，立身舍命何人。
博学已多岁月，犹向书窗辛勤。

其三
采菱渡头风急，策杖林西日斜。
杏树坛边渔父，桃花源里人家。

其四
萋萋春草秋绿，落落长松夏寒。
牛羊自归村巷，童稚不识衣冠。

其五
山下孤烟远村，天边独树高原。
一瓢颜回陋巷，五柳先生对门。

其六
桃红复含宿雨，柳绿更带朝烟。
花落家童未扫，莺啼山客犹眠。

其七
酌酒会临泉水，抱琴好倚长松。
南园露葵朝折，东谷黄粱夜舂。
```
## 🎯 主题
《田园乐》组诗主要表达了王维对官场生活的厌倦与对田园生活的向往。诗人通过对繁华都市和功名利禄的淡泊态度，展现了自己追求精神自由的人生理想。诗中描绘了田园生活的自然和谐、宁静闲适，如"桃红复含宿雨，柳绿更带朝烟"等画面，体现了诗人对简单纯朴生活的赞美。同时，王维通过引用颜回、陶渊明（五柳先生）等隐士典故，表达了对隐居生活的向往与追求。整组诗流露出诗人超然物外的人生态度，反映了他作为"诗佛"的精神境界，即远离尘世喧嚣，回归自然本真，在田园中寻找心灵的安宁与自由。
## 🏗️ 结构
王维的《田园乐》是一组七言绝句，共七首，展现了诗人对田园生活的向往和对官场生活的厌倦。这组诗在结构上具有鲜明的特点，下面从章法布局和层次安排两方面进行分析。

一、章法布局

1. 整体结构
《田园乐》七首诗构成了一个完整的组诗，整体上呈现出由现实到理想、由外到内的结构特点。这七首诗可以看作是一个有机整体，共同表达了诗人对田园生活的向往和对官场生活的厌倦。

2. 首尾呼应
第一首和第七首在内容上形成首尾呼应。第一首诗写诗人对官场生活的厌倦和对隐居生活的向往，"厌见千门万户，经过北里南邻。官府鸣珂有底，崆峒散发何人。"第七首则描绘了理想的田园生活场景，"酌酒会临泉水，抱琴好倚长松。南园露葵朝折，东谷黄粱夜舂。"从厌倦官场到享受田园，形成了一个完整的情感和思想脉络。

3. 中间过渡
中间的五首诗（第二至第六首）则是从不同角度描绘田园生活的各个方面，以及对官场生活的反思，形成了从现实到理想的过渡。其中第二首是对功名利禄的思考，第三首是对田园景色的描绘，第四首是对田园生活的赞美，第五首是对隐居生活的向往，第六首是对闲适生活的描绘。

二、层次安排

1. 情感层次
这组诗在情感上呈现出明显的层次递进。从第一首的厌倦官场，到第二首的反思功名，再到第三、四、五、六首对田园生活的描绘和赞美，最后到第七首对理想田园生活的享受，情感上经历了从厌倦到反思，再到向往和享受的过程，形成了一个完整的情感发展脉络。

2. 内容层次
在内容上，这组诗也呈现出清晰的层次安排。第一、二首主要写对官场生活的厌倦和反思；第三、四、五、六首主要写田园生活的各个方面，包括景色、生活状态、隐居理想等；第七首则是对理想田园生活的综合描绘。这种内容上的层次安排，使得整组诗在主题表达上更加丰富和全面。

3. 意象层次
在意象的运用上，这组诗也呈现出明显的层次。第一、二首主要运用官场相关的意象，如"千门万户"、"官府鸣珂"、"封侯万户"等；第三至第七首则主要运用田园相关的意象，如"采菱渡头"、"杏树坛边"、"桃花源"、"萋萋春草"、"落落长松"、"桃红"、"柳绿"、"酌酒"、"抱琴"等。这种意象上的层次变化，也体现了诗人从官场到田园的思想转变。

4. 表现手法层次
在表现手法上，这组诗也呈现出层次变化。第一、二首多用对比和反问的手法，如"厌见千门万户"与"崆峒散发何人"的对比，"再见封侯万户，立身舍命何人"的反问；第三至第七首则多用描绘和铺陈的手法，如"采菱渡头风急，策杖林西日斜"、"萋萋春草秋绿，落落长松夏寒"、"桃红复含宿雨，柳绿更带朝烟"等。这种表现手法上的层次变化，也体现了诗人情感和思想的变化。

三、艺术特色

1. 对比手法
这组诗大量运用对比手法，如官场与田园的对比，功名利禄与闲适生活的对比，繁忙与闲适的对比等。这种对比手法的运用，使得诗人对田园生活的向往和对官场生活的厌倦更加鲜明。

2. 意象丰富
这组诗运用了丰富的意象，既有官场意象，也有田园意象，既有自然意象，也有人文意象。这些意象的运用，使得诗歌的画面感更强，意境更加丰富。

3. 语言简练
这组诗的语言简练而富有表现力，每首诗只有四句，却能够表达丰富的情感和思想。这种简练的语言风格，也是王维诗歌的一大特色。

4. 情景交融
这组诗在描绘田园景色时，常常将情感融入其中，做到情景交融。如"桃红复含宿雨，柳绿更带朝烟"不仅描绘了春天的景色，也表达了诗人对田园生活的喜爱之情。

综上所述，王维的《田园乐》在章法布局上呈现出由现实到理想、由外到内的结构特点，在层次安排上则呈现出情感、内容、意象和表现手法上的层次递进。这种结构特点和层次安排，使得这组诗在表达诗人对田园生活的向往和对官场生活的厌倦时，更加丰富和全面，也使得这组诗在艺术上更加完美。
## ✍️ 语言风格
《田园乐》是王维的一组七言绝句，展现了诗人对田园生活的向往和赞美，体现了他"诗中有画"的艺术特色。这组诗在语言风格上具有以下特点：

一、用词特点
1. 自然意象丰富：诗中大量运用自然意象，如"春草"、"长松"、"桃红"、"柳绿"、"孤烟"、"独树"等，这些词汇清新朴素，没有过多雕琢，却能准确传达田园风光的恬静与美丽。如第六首"桃红复含宿雨，柳绿更带朝烟"，用简洁明快的色彩词勾勒出春日图景。

2. 生活化词汇运用：使用大量反映农村生活的词汇，如"采菱"、"策杖"、"牛羊"、"童稚"、"家童"等，这些词汇贴近生活，朴实无华，生动展现田园生活的真实场景。如第三首"采菱渡头风急，策杖林西日斜"，通过两个动作勾勒出田园日常画面。

3. 含蓄精炼：王维用词极为精炼，往往能用最简洁的词语表达最丰富的意境。如第五首"山下孤烟远村，天边独树高原"，仅用"孤烟"和"独树"两个意象，就营造出空旷宁静的氛围。

二、修辞手法
1. 对比手法：多次运用对比，突出田园生活的宁静美好与官场生活的喧嚣烦扰。如第一首将"千门万户"的繁华与"崆峒散发"的隐逸对比，表达对官场厌倦和对隐逸向往。

2. 典故运用：善于运用典故，增加诗歌文化内涵。如第五首"一瓢颜回陋巷，五柳先生对门"，引用颜回和陶渊明的典故，表达对古代隐士的敬仰和对田园生活的向往。

3. 借景抒情：通过描绘自然景物表达情感。如第六首描绘春日清晨景色，表达闲适恬淡的心境；第七首描绘在泉水边饮酒、松树下弹琴的场景，表达对闲适生活的享受。

4. 叠词运用：巧妙运用叠词，增强诗歌音乐性和表现力。如第四首"萋萋春草秋绿，落落长松夏寒"，"萋萋"形容春草茂盛，"落落"形容松树挺拔，增强节奏感，使形象更加生动。

三、语言风格特点
1. 清新自然：语言清新自然，没有过多雕琢和修饰，却能准确传达田园风光的美丽和生活的恬静。

2. 含蓄蕴藉：语言含蓄蕴藉，往往在平淡中见深意。如第一首表面上是问官府中鸣珂的是谁，崆峒山散发的是谁，实际上表达对官场厌倦和对隐逸向往。

3. 画面感强：具有"诗中有画"的特点，语言具有很强的画面感。如第五首仅用十四个字就勾勒出一幅空旷宁静的山水画；第三首通过两个场景构成生动的田园生活画卷。

4. 意境深远：语言简洁但意境深远，能引发读者无限遐想。如第四首表面描绘农村生活场景，实际上暗含对淳朴自然的田园生活的赞美和对世俗名利的超脱。

四、句式特点
1. 对仗工整：多采用对仗句式，使诗歌结构严谨，节奏鲜明。如第六首"桃红复含宿雨，柳绿更带朝烟"，结构对称，字数相同，形成工整对仗。

2. 节奏明快：诗歌节奏明快，读来朗朗上口，这与他善用短句、注重音律有关。如第三首两句都是四字结构，节奏鲜明，铿锵有力。

3. 虚实结合：句式往往虚实结合，既有对具体景物的描绘，又有对抽象情感的表达。如第五首前一句实写，后一句虚写，虚实结合，使意境更加深远。

总体而言，王维的《田园乐》在语言风格上清新自然、含蓄蕴藉，画面感强且意境深远，体现了诗人对田园生活的热爱和对超脱世俗的追求，展现了他作为"诗佛"的独特艺术风格。
## 📚 文化札记
《田园乐》是王维的代表作之一，这组诗充分体现了诗人对田园生活的向往和对官场的厌倦。以下是对诗中重要文化元素的注释：

其一文化背景：
"千门万户"指权贵豪门之家，也泛指繁华都市景象。"北里"在唐代特指长安城北的平康里，是当时著名的娱乐场所；"南邻"则泛指普通邻里。"鸣珂"是官员腰间佩戴的玉饰，行走时发出声响，象征官场身份。"崆峒"是道教名山，在今甘肃平凉，传说黄帝曾在此向广成子问道。"散发"指不束发的形象，象征不受世俗礼法约束的隐士生活，"崆峒散发"整体指代隐士，表达诗人向往隐居的心态。

其二文化背景：
"封侯万户"是古代最高级别的封赏，即封为侯爵，食邑万户，是古代士人追求的最高功名。"立身舍命"体现了儒家"舍生取义"的价值观念，指为了功名事业不惜牺牲生命。"博学已多岁月"反映了唐代科举制度下，士人需要长期苦读才能参加考试，追求功名的社会现实。

其三文化背景：
"采菱"是古代江南地区常见的水上劳动，多由女子进行，是田园生活的典型意象。"策杖"指拄着手杖行走，常用来形容隐士或老人的行走方式。"杏树坛"相传是孔子讲学的地方，在今山东曲阜，此处借用典故表示隐居之地也有文化气息。"渔父"典出《楚辞·渔父》，是屈原遇到的一位隐士，象征超然物外的智慧。"桃花源"典出陶渊明《桃花源记》，是理想中的隐居乐土，与世隔绝，生活安宁。

其四文化背景：
"萋萋"形容草木茂盛的样子，出自《诗经·小雅·出车》"春日迟迟，卉木萋萋"。"落落"形容松树高大挺拔的样子。"童稚不识衣冠"中，衣冠指官员的服饰和礼帽，这里表示乡村儿童不认识官场服饰，暗示乡村远离官场，保持纯朴自然的生活状态。

其五文化背景：
"一瓢颜回陋巷"典出《论语·雍也》，孔子称赞颜回"一箪食，一瓢饮，在陋巷，人不堪其忧，回也不改其乐"，赞美颜回安贫乐道的精神。"五柳先生"是陶渊明的自称，因其宅边有五棵柳树而得名。陶渊明是东晋著名隐逸诗人，代表作有《归园田居》《桃花源记》等，是王维推崇的隐士典范。

其六文化背景：
"宿雨"指昨夜的雨水，"朝烟"指早晨的雾气或炊烟，这两处意象营造出清晨田园的朦胧美感。"山客"指隐居山野的人，此处是诗人自指，表达自己作为隐士的身份认同。

其七文化背景：
"抱琴"是古代文人隐士的典型形象，琴象征高雅的生活情趣和精神追求。"露葵"指带露水的葵菜，古代常见的蔬菜。"黄粱"是小米，古代北方主要粮食之一。"夜舂"指晚上舂米，描绘农家日常生活的场景，展现田园生活的自然节奏。

这组诗充分展现了王维深厚的文化底蕴，他巧妙运用典故和传统文化元素，表达了对隐逸生活的向往和对官场的厌倦，体现了盛唐时期士人的精神追求和价值观。
## 🌅 创作背景
《田园乐》是王维在辋川别业隐居期间创作的一组七言绝句，一般认为创作于唐肃宗乾元元年（758年）至上元二年（761年）之间，即安史之乱后期。辋川别业位于今陕西省蓝田县南的辋川谷中，是王维精心经营的隐居之所，这里风景优美，环境清幽，成为诗人晚年创作的重要场所。

创作这组诗时，王维已年近六旬，经历了人生的起起落落。他早年仕途顺利，曾任右拾遗、监察御史等职，但安史之乱中被叛军俘获并被迫接受伪职，这段经历成为他心中难以抹去的阴影。乱平后虽得到赦免，被降为太子中允，后迁给事中，但他对官场生活已产生深深的厌倦，选择了半官半隐的生活方式。此时的王维心境复杂：一方面对官场的虚伪和险恶感到失望，向往自由自在的田园生活；另一方面又无法完全摆脱对世俗的关注和责任感。同时，他已潜心研究佛教，思想深受禅宗影响，追求心灵的宁静与超脱。

《田园乐》创作于唐朝中期的安史之乱后期，这场持续近八年的战乱给唐朝社会带来了巨大冲击，导致人口锐减、经济凋敝、社会动荡，朝廷权威下降。在这样的背景下，许多士人开始反思传统的价值观，对功名利禄的追求有所减弱，转而寻求精神上的慰藉和超脱。同时，这一时期也是佛教在中国发展的黄金时期，特别是禅宗思想在士人阶层中广泛传播，影响了王维等人的创作。此外，唐代中期的士人阶层普遍存在一种"吏隐"现象，即一方面担任官职，另一方面又向往隐逸生活，这种现象既是士人面对复杂政治环境的一种自我保护，也是他们对理想生活的一种追求。

王维创作《田园乐》的动机主要源于对官场生活的厌倦和对田园生活的向往，对隐逸传统的继承和发展，禅宗思想的影响，以及对自然美的深刻感受和艺术表达。诗中既有对官场的批判（如"厌见千门万户，官府鸣珂有底"），也有对隐逸生活的赞美（如"桃红复含宿雨，柳绿更带朝烟"）。他还化用了陶渊明等前代隐逸诗人的意象，如"杏树坛边渔父，桃花源里人家"，表现了对理想隐逸生活的追求。作为诗人和画家，王维在辋川别业隐居期间，有充足的时间观察和感受自然之美，并将这种感受通过诗歌表现出来，使这组诗具有了"诗中有画"的艺术特色。
## 💭 情感基调
《田园乐》组诗的情感基调主要表现为厌倦官场、向往田园、淡泊名利、闲适自得。诗人通过对繁华都市的"厌见"和对"崆峒散发"的向往，展现出对功名利禄的超脱态度。情感上既有对世俗生活的疏离，也有对隐逸生活的眷恋，整体呈现出一种恬淡宁静的心境。

氛围特点则以自然和谐、宁静悠远为主。诗中描绘了"萋萋春草"、"落落长松"、"桃红宿雨"、"柳绿朝烟"等自然景观，营造出清新淡雅的田园风光。同时，"牛羊自归"、"童稚不识衣冠"、"花落家童未扫"等生活场景，又增添了淳朴自然的乡村气息。整体氛围闲适恬淡，远离尘嚣，展现出一种与世无争的隐逸之美。

概括而言，这组诗的情感基调和氛围特点可简洁表述为：闲适恬淡，自然隐逸。
## 🔍 赏析
王维的《田园乐》组诗以清新淡雅的笔触勾勒出一幅幅田园生活的画卷。诗人通过对比手法，将官场的喧嚣与田园的宁静形成鲜明对照。如第一首中"厌见千门万户"与"崆峒散发何人"的对比，表现了对官场生活的厌倦和对隐逸生活的向往。诗人善于选取典型意象，如"采菱渡头"、"杏树坛边"、"桃花源里"、"山下孤烟"等，营造出空灵、淡远的意境，体现了王维"诗中有画"的艺术特色。同时，诗人还运用动静结合的手法，如"花落家童未扫，莺啼山客犹眠"，静态与动态形成对比，增强了画面的生动性。

《田园乐》的语言简洁明快，不事雕琢，却意蕴深远。王维擅长用白描手法，如"萋萋春草秋绿，落落长松夏寒"，寥寥数语便勾勒出四季变化的自然景观。修辞方面，诗人多用对偶和排比，如"桃红复含宿雨，柳绿更带朝烟"，不仅形式整齐，而且音韵和谐，增强了诗歌的音乐美。同时，诗人还善于用典，如"一瓢颜回陋巷，五柳先生对门"，引用颜回和陶渊明的典故，丰富了诗歌的内涵。王维还善于运用色彩词，使诗歌画面更加鲜明生动。

这组诗表达了诗人对田园生活的热爱和对官场生活的厌倦。第一首中"厌见千门万户"直接表明了诗人对官场生活的厌恶，而"崆峒散发何人"则流露出对隐逸生活的向往。诗人通过对自然景物的描绘，表达了对宁静、自由生活的追求。同时，诗歌也表达了诗人对简朴生活的赞美和对功名利禄的淡泊。此外，诗歌还体现了诗人对自然的热爱和对人与自然和谐相处的向往，如"牛羊自归村巷，童稚不识衣冠"，描绘了一幅人与自然和谐共处的画面。

《田园乐》是王维田园诗的代表作之一，体现了王维诗歌"诗中有画"的艺术特色，对后世山水田园诗的发展产生了深远影响。这组诗在思想内容上，继承了陶渊明的田园诗传统，但又融入了佛教的空寂思想，形成了独特的艺术风格。在艺术表现上，以其清新淡雅的语言、空灵深远的意境和精湛的艺术技巧，成为中国古典诗歌的典范。此外，《田园乐》所体现的对自然的热爱、对简朴生活的追求和对功名利禄的淡泊态度，也对中国传统文化中的隐逸思想产生了深远影响，成为后世文人精神追求的重要源泉。
