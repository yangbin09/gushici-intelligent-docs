## 📜 原文
```
厌见千门万户，经过北里南邻。官府鸣珂有底，崆峒散发何人。
再见封侯万户，立谈赐璧一双。讵胜耦耕南亩，何如高卧东窗。
采菱渡头风急，策杖林西日斜。杏树坛边渔父，桃花源里人家。
萋萋春草秋绿，落落长松夏寒。牛羊自归村巷，童稚不识衣冠。
山下孤烟远村，天边独树高原。一瓢颜回陋巷，五柳先生对门。
桃红复含宿雨，柳绿更带朝烟。花落家童未扫，鸟啼山客犹眠。
酌酒会临泉水，抱琴好倚长松。南园露葵朝折，东谷黄粱夜舂。
```
## 🎯 主题
王维的《田园乐》主要表达了对隐逸生活的向往与追求，体现了诗人淡泊名利、回归自然的人生理想。诗中通过对比繁华都市与宁静乡村，展现了对官场生活的厌倦和对田园生活的热爱。诗人赞美自然之美，描绘四季变换中的田园景色，表达了对简朴生活的向往。诗中还以颜回、陶渊明等古代隐士为榜样，崇尚精神自由和内心的宁静。整组诗流露出一种超然物外、与自然和谐共处的生活态度，反映了王维作为"诗佛"的精神境界和"诗中有画"的艺术特色，体现了中国传统文化中隐逸思想的精髓。
## 🏗️ 结构
王维的《田园乐》是一组由七首六言绝句构成的组诗，整体结构严谨而富有变化，体现了诗人精心的艺术构思。从章法布局来看，每首诗均采用四句六言的固定格式，形式整齐统一，形成一种节奏感和韵律美。整组诗以"厌见官场—向往田园—描绘田园—亲历田园"为线索，构成一个完整的情感发展脉络。

在层次安排上，这组诗呈现出明显的递进关系。第一、二首为第一层次，主要表达对官场生活的厌倦与否定，如"厌见千门万户"、"官府鸣珂有底"等句直抒胸臆，与"崆峒散发何人"、"耦耕南亩"形成鲜明对比，奠定全组诗的基调。第三至五首为第二层次，转向对田园生活的描绘与赞美，从"采菱渡头"、"杏树坛边渔父"到"山下孤烟远村"，视角由近及远，由具体到概括，展现田园生活的多样面貌。第六、七首为第三层次，进一步深入到诗人的亲身体验，如"花落家童未扫，鸟啼山客犹眠"、"酌酒会临泉水，抱琴好倚长松"，表现诗人融入田园生活的闲适与自在。

每首诗内部的层次安排也颇具匠心。大多采用前两句写景或活动，后两句写人或生活的结构，如第四首"萋萋春草秋绿，落落长松夏寒"写自然景色，"牛羊自归村巷，童稚不识衣冠"写田园生活，形成景与人的和谐统一。这种结构既保持了每首诗的独立性，又使各首之间相互呼应，构成一个有机整体。

在对比手法的运用上，这组诗通过官场与田园、繁华与宁静、功名与闲适的多重对比，强化了主题表达。如"再见封侯万户，立谈赐璧一双"与"讵胜耦耕南亩，何如高卧东窗"的对比，直接点明诗人的人生选择。

意象安排上，诗人巧妙运用自然意象（春草、长松、桃红、柳绿）、人物意象（渔父、山客、颜回、五柳先生）和活动意象（采菱、策杖、酌酒、抱琴），多角度展现田园生活的美好，使诗歌意境丰富而深远。

总体而言，《田园乐》的结构特点体现了王维"诗中有画"的艺术风格，通过严谨的章法布局和清晰的层次安排，将诗人对田园生活的热爱与对官场生活的厌倦表达得淋漓尽致，形成了一个完美的艺术整体。
## ✍️ 语言风格
王维的《田园乐》是一组展现田园生活的诗作，其语言风格体现了王维作为山水田园诗代表作家的独特艺术特色。

在用词方面，王维表现出简洁明快的特点。如"厌见"、"何如"等词语直接表达情感态度，不拐弯抹角。他善于运用色彩鲜明的词语，如"桃红"、"柳绿"，生动描绘自然景色。叠词的运用如"萋萋"、"落落"增强了诗歌的音乐性和形象感。动作性词语如"采菱"、"策杖"、"酌酒"、"抱琴"等，使画面更加生动，仿佛读者能亲眼看到诗中描绘的场景。

修辞手法上，王维运用了多种技巧。对比手法是最显著的，他将官场的繁华与田园的宁静形成鲜明对比，如"千门万户"与"崆峒散发"，"封侯万户"与"耦耕南亩"等，突显了诗人对田园生活的向往。用典是另一重要特点，诗中引用了"杏树坛"、"桃花源"、"颜回陋巷"、"五柳先生"等典故，不仅增强了诗歌的文化内涵，也表明诗人对古代隐士的敬仰。对偶手法如"桃红复含宿雨，柳绿更带朝烟"和"花落家童未扫，鸟啼山客犹眠"等，增强了诗歌的节奏感和美感。

王维的语言风格总体上清新自然，不事雕琢，如"花落家童未扫，鸟啼山客犹眠"表现了田园生活的自然状态。他的诗歌含蓄蕴藉，不直接表达情感，而是通过景物描写来寄托情感，如"山下孤烟远村，天边独树高原"等。这组诗具有很强的画面感，每首诗都如同一幅美丽的画卷，展现了诗人"诗中有画"的艺术特色。同时，诗歌也具有优美的音乐性，如"萋萋春草秋绿，落落长松夏寒"等，读来朗朗上口。

思想内容上，这组诗表达了诗人对田园生活的向往和对官场生活的厌倦，追求自然与自由，崇尚简朴生活，并通过引用古代隐士的典故，表达了对他们的敬仰和向往。整体而言，《田园乐》充分体现了王维清新自然、含蓄蕴藉的诗风，以及他作为"诗佛"的艺术风格和思想境界。
## 📚 文化札记
《田园乐》是王维创作的一组田园诗，共八首，表达了诗人对官场的厌倦和对田园生活的向往。以下是诗中重要的文化背景注释：

第一首中，"千门万户"指京城繁华景象，形容富贵人家住宅众多。"北里南邻"代指世俗社会。"鸣珂"是古代官员腰间佩玉，行走时相击发出声响，象征官职和权势。"崆峒"是山名，在今甘肃省平凉市西南，相传为黄帝问道于广成子的地方，后成为道家修仙的象征。"散发"表示不拘礼法，是隐居山林的生活方式。

第二首中，"封侯万户"是古代爵位制度，封为侯爵，食邑万户，是极高的荣誉。"立谈赐璧一双"典出《史记·平原君虞卿列传》，赵国平原君门客毛遂因言辞说服楚王合纵抗秦，获赐玉璧一双。"耦耕"指两人并耕，是古代农耕方式，也象征隐居耕种的生活。"南亩"泛指农田，出自《诗经·七月》"同我妇子，馌彼南亩"。

第三首中，"杏树坛"相传为孔子讲学的地方，在今山东曲阜，后泛指讲学或隐居之处。"渔父"典出《楚辞·渔父》，屈原被放逐后遇到一位渔父，渔父劝他与世推移，后成为隐士的象征。"桃花源"出自陶渊明《桃花源记》，描述了一个与世隔绝的理想社会，成为隐逸生活的象征。

第四首中，"萋萋"形容草木茂盛的样子，出自《诗经·周南·葛覃》"维叶萋萋"。"落落"形容松树高大挺拔。"童稚不识衣冠"指乡村儿童不认识官服，表示乡村远离官场，生活纯朴自然。衣冠象征官场和礼法。

第五首中，"一瓢颜回陋巷"典出《论语·雍也》，孔子称赞颜回"一箪食，一瓢饮，在陋巷，人不堪其忧，回也不改其乐"，形容安贫乐道的精神。"五柳先生"是陶渊明的自号，因其宅边有五棵柳树而得名，是隐士的代表人物。

第七首中，"抱琴"象征文人高雅的生活情趣。"露葵"指带露水的葵菜，古代常见蔬菜。"黄粱"是小米，古代重要粮食作物，与"黄粱一梦"典故相关，出自唐传奇《枕中记》，比喻人生如梦。

整组诗体现了王维深受道家思想影响的隐逸情怀，通过描绘田园生活的自然美景和闲适状态，表达了对官场生活的厌倦和对隐居生活的向往，展现了中国古代文人"达则兼济天下，穷则独善其身"的人生哲学。
## 🌅 创作背景
《田园乐》是唐代诗人王维在辋川别业隐居期间创作的作品，时间大约在公元741年至761年之间，尤其是安史之乱后的晚年时期。辋川别业位于今陕西蓝田县，是王维购置的庄园，也是他晚年生活和创作的重要场所。

当时王维虽仍担任尚书右丞等官职，但经历了安史之乱中被迫接受伪职的痛苦经历后，他对政治生活产生了深深的厌倦。这段经历使他内心充满矛盾与痛苦，转而向佛教寻求精神慰藉。在辋川别业，他过着半官半隐的生活，一方面无法完全脱离官场，另一方面则沉醉于山水田园之中，寻求心灵的宁静与超脱。

王维生活的时代是唐朝由盛转衰的时期。安史之乱（755年-763年）严重动摇了唐朝的统治基础，社会秩序遭到破坏，许多文人对政治失去信心，转而追求隐逸生活。同时，佛教在唐代非常盛行，特别是禅宗思想对文人影响深远。这种社会环境和文化氛围，为王维创作田园诗提供了土壤。

《田园乐》的创作动机主要源于王维对官场生活的厌倦和对田园生活的向往。诗中"厌见千门万户"、"官府鸣珂有底"等句直接表达了对官场的厌倦；而"耦耕南亩"、"高卧东窗"等句则表达了对田园生活的向往。灵感来源包括他在辋川别业的实际生活体验、对陶渊明等前代隐逸诗人的学习借鉴，以及佛教思想的影响，特别是禅宗追求自然、超脱世俗的思想，使王维能够以平和的心态欣赏田园生活的美好。
## 💭 情感基调
王维的《田园乐》组诗展现了鲜明的情感基调和独特的氛围特点。从情感基调看，整组诗流露出淡泊名利、超然物外的态度，诗人对官场生活表示厌倦，对田园隐居生活则充满向往和满足。诗中"厌见千门万户"、"讵胜耦耕南亩，何如高卧东窗"等句直接表达了对繁华世俗的疏离和对简朴生活的偏爱。

从氛围特点看，这组诗营造了恬静闲适、自然淳朴的意境。诗人通过"桃红复含宿雨，柳绿更带朝烟"、"山下孤烟远村，天边独树高原"等清新淡远的画面，展现了一个远离尘嚣、和谐宁静的田园世界。诗中描绘的四季景色、日常劳作、淳朴民风，共同构成了一幅与自然和谐共处的生活图景。

概括而言，《田园乐》的情感基调可归纳为：淡泊、超脱、自得、满足；氛围特点可概括为：恬静、自然、清幽、和谐。这组诗充分体现了王维"诗中有画"的艺术特色和其向往隐逸、追求精神自由的人生态度。
## 🔍 赏析
王维的《田园乐》组诗以其独特的艺术魅力展现了诗人对田园生活的向往与对官场的厌倦。在意境营造上，诗人善用对比手法，将"千门万户"的官场喧嚣与"崆峒散发"的隐逸生活形成鲜明对照，突显其精神追求。诗中多处运用白描技法，如"桃红复含宿雨，柳绿更带朝烟"，以简洁笔触勾勒出生动的自然画面，营造出清新淡远的艺术境界。同时，诗人通过时空交错的手法，如"萋萋春草秋绿，落落长松夏寒"，将四季景色融于一体，拓展了诗歌的时空维度。

语言方面，王维的文字清新自然，不事雕琢却意蕴深远。他善用对偶，如"山下孤烟远村，天边独树高原"，形式工整而意境开阔。诗中多处运用典故，如"一瓢颜回陋巷，五柳先生对门"，借颜回和陶渊明的形象，表达对简朴生活的推崇。色彩词的运用如"桃红"、"柳绿"使画面更加鲜明，而动静结合的手法如"花落家童未扫，鸟啼山客犹眠"，则形成了动静相宜的艺术效果。

情感表达上，这组诗流露出诗人对功名利禄的淡漠和对农耕隐逸生活的向往。"厌见千门万户"、"再见封侯万户"直接表明了诗人对官场的厌倦，而"耦耕南亩"、"高卧东窗"则表达了对田园生活的热爱。诗中体现了"天人合一"的哲学思想，表现了诗人追求心灵自由、回归自然的精神追求。

《田园乐》代表了唐代山水田园诗的高峰，对后世产生了深远影响。王维将主观情感与客观景物完美融合，创造了"诗中有画"的艺术境界，成为中国古典诗歌的典范。其对简朴生活的赞美和对官场的批判，影响了后世文人的生活态度和价值取向，对宋代苏轼等人以及后世无数文人产生了深远影响，成为中国文学史上不可或缺的重要组成部分。
