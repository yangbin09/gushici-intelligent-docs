## 📜 原文
```
移舟泊烟渚，日暮客愁新。
野旷天低树，江清月近人。
```
## 🎯 主题
《宿建德江》这首诗的主要主题思想是表达羁旅愁思与自然景观的交融。诗中通过"移舟泊烟渚，日暮客愁新"直接点明了诗人作为旅人的孤独感，黄昏时分更引发了新的思乡之情。后两句"野旷天低树，江清月近人"则以广阔的自然景象为背景，展现了诗人在壮丽自然前的渺小感受，同时"月近人"又透露出一种亲近自然的慰藉。整首诗将客居他乡的愁绪与对自然景物的细腻观察融为一体，表达了人在旅途中既感孤独又能在自然中寻求心灵慰藉的复杂情感，体现了孟浩然山水诗中以景抒情、情景交融的艺术特色。
## 🏗️ 结构
《宿建德江》作为一首五言绝句，其结构特点体现了唐代绝句的精巧构思和严密布局。从章法布局来看，全诗四句二十字，遵循"起承转合"的传统结构模式，层次分明，转换自然。

第一句"移舟泊烟渚"为起句，交代了诗人的行动和所处环境，是叙事性的开篇。这一句设定了诗歌的基本场景：诗人乘船停靠在烟雾弥漫的江边小洲上，奠定了全诗的基调。

第二句"日暮客愁新"为承句，由景入情，直接点明诗人的情感状态。"日暮"承接上句的时间背景，"客愁新"则揭示了诗人的内心感受，将外在景物与内在情感联系起来，为全诗定下了愁思的基调。

第三句"野旷天低树"为转句，视野突然拉远，描绘了广阔的野外景象。这一句在空间上形成了一个转折，从近处的烟渚转向远处的旷野，同时"天低树"的描写也创造了一种压抑感，与诗人的愁绪相呼应。

第四句"江清月近人"为合句，视线又拉回近处，描写清澈的江水和近在咫尺的月亮。这一句既是对前文的呼应，也是情感的升华，通过"月近人"的意象，含蓄地表达了诗人的孤独感和对亲近的渴望。

在层次安排上，诗歌呈现出明显的空间转换：从近处（烟渚）到远处（野旷天低）再到近处（江清月近），形成一种空间上的回环结构。同时，时间线索也从日暮推移到夜晚，自然流畅。

此外，诗歌还运用了多种对比手法：远近对比、高低对比（天低树）、明暗对比（日暮与月近），这些对比不仅增强了诗歌的立体感和画面感，也深化了情感的表达。

整体而言，《宿建德江》的结构严谨而富有变化，通过精心的章法布局和层次安排，成功地营造了一种孤寂、清冷的氛围，表达了游子思乡的愁绪，体现了孟浩然诗歌"意境高远，含蓄蕴藉"的艺术特色。
## ✍️ 语言风格
孟浩然的《宿建德江》展现了典型的盛唐山水田园诗风格，语言特点鲜明而独特。

在用词方面，全诗语言简洁朴素，仅用二十字便勾勒出一幅完整的江边夜景。诗人选用"移舟"、"泊"、"烟渚"等精准而朴实的词语，没有华丽辞藻的堆砌，却意境深远。"烟渚"一词尤为精妙，"渚"指水中小洲，"烟"则形容水汽缭绕的朦胧景象，二者结合既点明了停船地点，又营造出一种朦胧美感。"日暮"、"客愁新"等词语直白而含蓄，特别是"新"字，既点明了愁绪的突然涌现，又暗示了情感的微妙变化。

在修辞手法上，诗人运用了多种技巧增强艺术效果。首先，后两句"野旷天低树，江清月近人"采用了工整的对偶结构，"野旷"对"江清"，"天低树"对"月近人"，形式对称，音韵和谐。其次，诗人巧妙运用拟人手法，通过"低"和"近"两个动词，赋予了天和月以人的动作和情感，使自然景物生动活泼，富有灵性。此外，诗中还运用了对比手法，"野旷"与"天低树"形成空间上的对比，"江清"与"月近人"则构成远近对比，增强了画面的层次感和立体感。

情景交融是这首诗最突出的艺术特点。诗人不直接抒发情感，而是将内心感受巧妙地融入景物描写中。"日暮客愁新"一句，将外在的日落景象与内心的愁绪自然联系起来，达到了情景合一的艺术境界。整首诗通过"烟渚"、"日暮"、"野旷"、"天低树"、"江清"、"月近人"等意象，构成了一幅幅生动的画面，体现了"诗中有画"的艺术特色。

总体而言，孟浩然的《宿建德江》语言风格清新自然，含蓄委婉，既有盛唐诗歌的普遍特征，又带有诗人个人的艺术印记，展现了中国古典诗歌"言有尽而意无穷"的艺术魅力。
## 📚 文化札记
《宿建德江》是唐代山水田园诗派代表诗人孟浩然的代表作。这首诗虽短小，却蕴含丰富的文化内涵。

建德江位于今浙江省杭州市建德市，是钱塘江的一段，在唐代是重要的水上交通要道。孟浩然一生未仕，多游历山水，此诗应作于其漫游吴越(今江浙一带)期间。

"移舟泊烟渚"中的"烟渚"指被薄雾笼罩的江中小岛或岸边，这一意象在古典诗词中常用来营造朦胧、凄清的氛围。"渚"字本身就有文化渊源，《诗经》中就有"宛在水中沚"的描写，是古典诗歌常用的意象。

"日暮客愁新"一句中，"日暮"在古典诗词中常引发思乡之情，如马致远"夕阳西下，断肠人在天涯"。"客"字表明诗人身处异乡的身份，反映了中国古代"安土重迁"的文化传统，游子思乡是古典诗词的重要主题。"愁新"则表明诗人本已有愁，在黄昏时分又添新愁，情感层层递进。

"野旷天低树"描绘了建德江两岸开阔的地理特点。这种"天低树"的视觉描写，是中国古典诗词中常用的空间表现手法，通过远近视点的转换，创造出独特的空间感。

"江清月近人"中，建德江以水质清澈著称，月亮的倒影映在水中，仿佛离人很近。"月"在中国传统文化中是团圆的象征，也是思乡情感的载体，李白"举头望明月，低头思故乡"即是明证。"月近人"既有实景描摹，又有情感寄托，暗示诗人的孤独和对亲人的思念。

整首诗体现了孟浩然山水诗"诗中有画"的艺术特色，以及唐代文人"情景交融"的创作理念，通过简练的语言和精准的意象，表达了游子思乡的普遍情感，展现了中国古典诗歌的深厚文化底蕴。
## 🌅 创作背景
《宿建德江》是唐代著名山水田园诗人孟浩然的代表作之一，创作于唐玄宗开元十四年至十六年（726-728年）间，地点在今天的浙江省建德市附近的新安江（富春江上游）。此时孟浩然约三十八至四十岁，正处于人生的一个重要阶段。

孟浩然曾于开元十二年（724年）赴长安应试，但未能中第，这次失败对他打击很大。之后他开始漫游吴越地区，寻求心灵的慰藉和创作灵感。此时的孟浩然怀才不遇，内心充满了失意和惆怅，但同时也对自然山水有着深厚的热爱。这首诗正是他在漫游途中，夜宿建德江畔时所作，反映了作者孤独、思乡的复杂心境。

唐玄宗开元年间（713-741年）是唐朝的鼎盛时期，社会安定，经济繁荣，文化昌盛，史称"开元盛世"。这一时期，科举制度已经完善，成为知识分子入仕的主要途径。然而，科举竞争也异常激烈，许多有才华的士人如孟浩然都未能通过科举实现自己的政治抱负。在这样的社会环境下，一些失意的文人转而寄情山水，通过诗歌表达自己的情感和志向，形成了盛唐山水田园诗派的繁荣景象。

孟浩然创作这首诗的直接动机是他夜宿建德江时的所见所感。诗中"移舟泊烟渚，日暮客愁新"描述了傍晚时分泊舟靠岸的场景，流露出作者作为游子的思乡之情。"野旷天低树，江清月近人"则描绘了开阔的原野、低垂的天空和清澈的江水中映照的月亮，表现了作者对自然美景的细腻感受。这首诗的创作灵感来源于作者对自然景观的敏锐观察和内心情感的交融，反映了中国传统文人在仕途不顺时，常借自然景物抒发情感的特点。
## 💭 情感基调
《宿建德江》的情感基调主要表现为孤寂忧愁。诗中"客愁新"直接点明了诗人作为异乡客的愁绪，这种情感在日暮时分尤为强烈，流露出深沉的思乡之情和漂泊无依的孤独感。

氛围特点则是幽静空旷。通过"烟渚"、"日暮"营造出朦胧幽静的黄昏景象；"野旷天低树"描绘出辽阔空旷的自然环境，天地相接，视野开阔；而"江清月近人"则进一步强化了这种清冷宁静的氛围，江水清澈，月光倒映，使整个画面既空旷又静谧，与诗人内心的孤寂形成呼应，达到情景交融的艺术效果。
## 🔍 赏析
《宿建德江》是孟浩然山水诗的代表作，通过简洁凝练的语言勾勒出一幅江边夜景图，并融入诗人复杂的情感体验。

在意境营造方面，诗人采用远近结合、虚实相生的艺术手法，构建了一个层次分明的空间结构。首句"移舟泊烟渚"直接点明行动和地点，"烟渚"一词已暗示傍晚江边雾气缭绕的朦胧氛围。第二句"日暮客愁新"点明时间和心情，一个"新"字表明愁绪是刚刚涌上心头的。第三句"野旷天低树"将视线投向远方，展现广阔原野与低垂天空的对比，既写实又暗合诗人压抑心境。末句"江清月近人"将视线拉回近处，清澈江水中的月亮倒影给人以亲近感，虚实结合增强艺术感染力。

语言特色上，全诗仅二十字却意境深远，体现了"言有尽而意无穷"的艺术境界。诗人善用对比和拟人修辞，"野旷"与"天低"形成空间对比，"江清"与"月近"则与前面的"客愁"形成情绪对比。"天低树"和"月近人"运用拟人手法，赋予自然物以人的情感，增强生动性。意象叠加也是重要特色，"烟渚"、"日暮"、"野旷"等意象层层叠加，共同构建完整艺术世界。

情感表达上，诗歌直接点明"客愁"，展现诗人作为旅人的孤独感和思乡之情。然而诗人并非一味沉浸愁绪，而是通过与自然交流寻求心灵慰藉。广阔原野和低垂天空与诗人心境相呼应，清澈江水和近在咫尺的月亮则给予慰藉，体现了人与自然的和谐关系。

文学价值方面，这首诗是唐代五言绝句的经典之作，体现了盛唐山水田园诗的艺术特色。它对后世文学创作产生深远影响，所表达的旅人思乡之情及人与自然和谐相处的理念具有超越时代的普遍意义，成为中国古典诗歌宝库中的璀璨明珠。
