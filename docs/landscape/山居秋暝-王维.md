## 📜 原文
```
空山新雨后，天气晚来秋。
明月松间照，清泉石上流。
竹喧归浣女，莲动下渔舟。
随意春芳歇，王孙自可留。
```
## 🎯 主题
《山居秋暝》主要表达了诗人王维对隐居山林生活的热爱与赞美，展现了他追求心灵宁静、与自然和谐共处的生活理想。诗中通过描绘秋雨过后山间的清新景象、月光松影、清泉流淌等自然美景，以及浣女归家、渔舟穿行的生活场景，构成了一幅动静结合、人景交融的山水画卷。最后两句"随意春芳歇，王孙自可留"更直接表达了诗人随遇而安、不为外物所动的超然心境，体现了他顺应自然、安于山居的生活态度，反映了王维作为"诗佛"特有的禅意思想和对精神自由的追求。
## 🏗️ 结构
《山居秋暝》作为王维山水诗的代表作品，在结构上体现了严谨的章法布局和精妙的层次安排，充分展现了唐代律诗的艺术特色。

从章法布局看，此诗遵循律诗"起承转合"的经典结构。首联"空山新雨后，天气晚来秋"为"起"，点明时间、地点和总体氛围，以简练笔墨勾勒出秋日傍晚雨后山林的清新环境，为全诗奠定基调。颔联"明月松间照，清泉石上流"为"承"，承接首联展开静态自然景色的描写，通过工整对仗展现山居的宁静优美。颈联"竹喧归浣女，莲动下渔舟"为"转"，巧妙转换视角，由静态景物转向动态人文活动，引入生活气息。尾联"随意春芳歇，王孙自可留"为"合"，总结全诗并表达诗人情感，含蓄点明归隐之志。

在层次安排上，此诗呈现出清晰的递进层次。第一层次为环境背景，通过首联构建整体时空框架；第二层次为自然景观，颔联聚焦于月光、清泉等静态自然元素；第三层次为人文活动，颈联引入浣女、渔舟等动态人文元素；第四层次为情感哲理，尾联升华主题，表达诗人对山居生活的态度。这种由外而内、由景及情的层次安排，使诗歌内容丰富而不杂乱，结构严谨而不呆板。

此外，诗歌在空间布局上也颇具匠心。从高远空山到松间明月，再到石上清泉，继而到竹林莲池，形成由远及近、由高至低的空间转换，构建立体画面。在时间线上，诗中暗含从雨后到傍晚再到明月升起的时间推移，增强了诗歌的时空感和真实感。

特别值得注意的是，颔联与颈联的对仗工整严谨，"明月"对"清泉"，"松间"对"石上"，"照"对"流"；"竹喧"对"莲动"，"归浣女"对"下渔舟"，体现了律诗形式美的同时，也通过动静结合（颔联静、颈联动）的手法丰富了诗歌的表现力。

总体而言，《山居秋暝》结构严谨而自然，层次分明而流畅，章法布局合理有序，充分展现了王维"诗中有画"的艺术特色，也体现了唐代律诗的成熟之美。
## ✍️ 语言风格
王维的《山居秋暝》体现了诗人清新淡雅、意境幽远的语言风格，是唐代山水田园诗的代表作。其语言风格特点主要表现在以下几个方面：

在用词方面，这首诗选词精炼而富有画面感。"空山"、"新雨"、"明月"、"清泉"等自然意象的运用，构建了一个清新脱俗的山水世界。诗人没有直接使用色彩词，但通过"明月"、"清泉"、"莲"等词语暗示了明亮的白色、清澈的透明色和莲花的粉色，形成了一幅色彩和谐的画卷。同时，诗中既有"照"、"流"、"喧"、"动"等动态描写，又有"空山"、"石"等静态描写，动静结合，使画面更加生动。多感官的描写（视觉的"明月松间照"，听觉的"竹喧"，触觉的"新雨后"湿润感）使诗歌更加立体。

在修辞手法上，这首诗运用了多种技巧。全诗对仗工整，尤其是中间两联："明月松间照，清泉石上流"和"竹喧归浣女，莲动下渔舟"，形式整齐，音韵和谐。诗人通过借景抒情，表达了对隐居生活的喜爱和对自然的热爱，情景交融。"竹喧归浣女"中，以声响反衬出山林的宁静，体现了"蝉噪林逾静，鸟鸣山更幽"的艺术效果。诗中既有实景描写，又有虚写，虚实结合，意境更加深远。多个意象的组合（如"明月"、"松"、"清泉"、"石"）构成了一幅完整的山水画卷，增强了诗歌的画面感。尾联"随意春芳歇，王孙自可留"中，将"春芳"拟人化，赋予其意志，增强了诗歌的生动性。

在音律方面，全诗为五言律诗，格律工整，平仄相间，读来抑扬顿挫，富有音乐美。押"秋、流、舟、留"韵，音韵和谐，增强了诗歌的音乐性。

整体而言，这首诗营造出空灵、清幽、和谐、淡远的艺术境界，体现了王维"诗中有画，画中有诗"的艺术特色和禅意诗风。诗人以简约的语言勾勒出丰富的画面，在平淡中见深远，在自然中见禅意，展现了王维作为"诗佛"的独特艺术魅力。
## 📚 文化札记
《山居秋暝》是王维隐居终南山辋川别业时所作的代表作，体现了其"诗中有画"的艺术特色和"诗佛"的禅意境界。

"空山"指寂静无人的山林，暗合王维隐居的终南山环境。终南山位于今陕西西安南，自古就是隐士修行之地，道教文化圣地，王维在此购置辋川别业，过着半官半隐的生活。

"明月松间照，清泉石上流"中的"松"在传统文化中象征坚韧不拔、高洁不屈的品格，是文人雅士喜爱的植物。明月则象征高洁、清幽，这些意象共同营造出王维特有的空灵美学风格。

"竹喧归浣女，莲动下渔舟"描绘了山中傍晚的生活场景。"浣女"指洗衣的女子，在古代农村生活中，女子常到溪流或河边洗衣，是常见的田园意象。"莲"在中国传统文化中象征纯洁高雅，也与佛教有着密切联系，呼应王维的佛教信仰。

末句"随意春芳歇，王孙自可留"中的"王孙"是重要典故，出自《楚辞·招隐士》："王孙游兮不归，春草生兮萋萋。"原意是劝隐士回归仕途。王维在此反用其意，表示即使春天已过，花草凋零，作为隐士仍然可以留在这山林之中，不必回到世俗社会。这体现了王维对传统隐逸思想的继承和发展，也反映了他超脱世俗、回归自然的人生态度。

整首诗融合了自然景观与人文活动，展现了王维对自然的热爱和对隐居生活的满足，同时也反映了唐代文人的隐逸文化和佛教思想对王维创作的深远影响。
## 🌅 创作背景
《山居秋暝》创作于唐玄宗开元末年至天宝初年（约740-750年）之间，地点是王维位于终南山蓝田县的辋川别业。王维在开元二十八年（740年）左右购得此地产，成为他晚年隐居的重要场所。这一时期，王维虽仍在朝为官，担任右拾遗、给事中等职，但已经开始实行半官半隐的生活方式，更多时间在辋川别业度过。

当时王维的心境平和而超脱，深受佛教禅宗思想影响。他在官场与山林之间寻求平衡，既不完全脱离世俗，也不被功名利禄所束缚。这种"中道"的生活态度反映在他的诗歌创作中，形成了独特的"诗佛"风格。他对自然有着极为敏锐的观察力和深厚的感情，常以细腻的笔触描绘山水景物，寄托自己的思想情感。

从历史背景看，这一时期是唐朝由盛转衰的过渡阶段。表面上仍是"开元盛世"的延续，但朝政已开始腐败，李林甫等人专权，政治环境逐渐恶化。安史之乱（755年）虽未爆发，但社会矛盾已日益显现。在这样的大环境下，许多文人选择远离政治中心，寻求精神上的寄托，王维的半官半隐生活方式正是这种社会背景下的产物。

《山居秋暝》的创作动机主要源于王维对自然的热爱和对禅宗思想的追求。诗中描绘的秋雨初晴、明月松间、清泉石上、浣女归来、渔舟下莲等景象，都是他在辋川别业日常生活的真实写照。王维通过这些自然景物，表达了自己对宁静生活的热爱和对世俗的超脱态度。特别是"随意春芳歇，王孙自可留"两句，体现了他对世事变迁的超然心境和选择隐居生活的决心，展现了他"诗中有画"的艺术特色和深厚的禅宗思想底蕴。
## 💭 情感基调
《山居秋暝》的情感基调可概括为恬淡闲适，氛围特点表现为清幽宁静。

诗中通过"空山新雨后"、"明月松间照"等意象，营造出雨后山林的清新与空灵。诗人以平和的心态观察自然，"清泉石上流"、"竹喧归浣女"等句展现了动静结合的和谐画面，没有喧嚣，只有自然的韵律。尾联"随意春芳歇，王孙自可留"更体现了诗人超然物外、安于隐居的心境。

整首诗色彩淡雅，意境空灵，既有对自然美景的细腻描绘，又蕴含着诗人对隐居生活的满足与热爱，呈现出一种物我两忘的禅意境界。
## 🔍 赏析
《山居秋暝》是王维山水诗的代表作，体现了诗人"诗中有画"的艺术特色和禅宗思想。在意境营造上，诗人通过"空山"、"新雨"、"明月"、"清泉"等意象，构建了一个空灵宁静、清新淡远的艺术世界。诗中动静结合、视听交融的艺术手法尤为精妙：静态的"空山"、"明月"与动态的"清泉"、"竹喧"、"莲动"形成对比；视觉上的"松间照"与听觉上的"竹喧"相互配合，使意境更加立体丰富。这种以小见大的手法，通过局部景物的描写展现整体美感，创造了令人心旷神怡的艺术境界。

语言特色方面，这首诗清新自然，简洁凝练，没有华丽辞藻却能准确传达意境。诗人巧妙运用对仗手法，如"明月松间照，清泉石上流"和"竹喧归浣女，莲动下渔舟"，使诗歌节奏和谐，音韵优美。此外，意象叠加和并置的手法增强了诗歌的表现力，而借代和暗示的运用则使表达更加含蓄深远，如"竹喧"暗示浣女归来，"莲动"暗示渔舟下行，这种含蓄表达使诗歌韵味无穷。

情感表达上，诗人流露出对自然的热爱和对隐居生活的满足。"空山"之"空"不仅指物理空旷，更暗含禅宗"空"的哲理；最后两句"随意春芳歇，王孙自可留"直接表达了诗人对山居生活的留恋和对世俗的超脱，体现了他顺应自然、与自然和谐相处的生活哲学。诗中隐含着对官场生活的厌倦和对田园生活的赞美，反映了诗人对理想生活方式的追求。

从文学价值看，这首诗完美体现了"诗中有画"的艺术特色，融合了诗歌与禅理，既有艺术性又有哲理性。它不仅影响了后世山水诗创作，也成为中国传统文化中追求精神自由和心灵宁静的重要象征。在当代社会，诗中所倡导的回归自然、淡泊名利的生活态度，对于缓解现代人精神压力，寻找内心平静，仍具有积极的启示意义。
