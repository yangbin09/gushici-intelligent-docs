## 📜 原文
```
空山新雨后，天气晚来秋。
明月松间照，清泉石上流。
竹喧归浣女，莲动下渔舟。
随意春芳歇，王孙自可留。
```
## 🎯 主题
《山居秋暝》主要表达了诗人王维对自然山水的热爱与对隐居生活的向往。诗中通过描绘雨后秋山的清新景象、明月清泉的自然美景以及人与自然和谐共处的画面，展现了一种宁静淡泊、超然物外的生活理想。诗人以"空山"的意象暗含禅宗思想，表现出追求心灵空灵的精神境界。最后两句直接抒发了对山居生活的满足与留恋，即使春光已逝，这种远离尘嚣、与自然融为一体的生活仍值得珍惜。整首诗体现了王维"诗中有画"的艺术特色，以及他作为"诗佛"追求物我两忘、天人合一的哲学思想。
## 🏗️ 结构
《山居秋暝》是王维山水诗的代表作，在结构上体现了严谨的章法和精心的布局安排。

首先，从整体章法看，这首五言律诗遵循了传统的"起承转合"结构。首联"空山新雨后，天气晚来秋"为"起"，交代了时间（秋日傍晚）和环境（雨后的空山），奠定了全诗基调。颔联"明月松间照，清泉石上流"为"承"，承接首联的环境描写，进一步展现自然景观的静谧之美。颈联"竹喧归浣女，莲动下渔舟"为"转"，引入人物活动，使画面更加生动活泼。尾联"随意春芳歇，王孙自可留"为"合"，抒发情感，点明诗人的隐居志向，完成全诗的情感升华。

在层次安排上，全诗可分为三个清晰的层次：第一层次是环境背景层（首联），对宏观环境进行概括描述；第二层次是具体景象层（颔联和颈联），又可分为自然景象的静态描写和人文景象的动态描写；第三层次是情感抒发层（尾联），表达诗人的内心世界和人生志向。

空间结构上，诗歌采用了由远及近、由上到下的布局方式。从整个山居环境的宏观描述，到月光、松树、清泉、石头的具体描绘，再到浣女、渔舟的人物活动，最后聚焦于诗人内心世界的展现，形成了空间上的层次递进。

时间结构上，诗歌展现了从傍晚到夜晚的时间流动。"晚来秋"点明傍晚时分，"明月松间照"则表明已入夜，整个时间线索清晰连贯。

在表现手法上，诗歌动静结合，虚实相生。首联和颔联以静态描写为主，呈现雨后山林的宁静；颈联转为动态描写，"竹喧"和"莲动"增添了生机；尾联又回归静态，表达诗人内心的平静。前六句为实写，描绘具体景象；尾联为虚写，抒发情感志向。

此外，颔联和颈联的对仗工整，不仅符合律诗的格律要求，也增强了诗歌的节奏感和艺术美感。"明月"对"清泉"，"松间"对"石上"，"照"对"流"；"竹喧"对"莲动"，"归浣女"对"下渔舟"，对仗精妙，音韵和谐。

通过这种严谨的结构安排，王维成功构建了一幅人与自然和谐统一的山居秋暝图，展现了他"诗中有画"的艺术特色和追求宁静淡泊的人生境界。
## ✍️ 语言风格
王维的《山居秋暝》是唐代山水田园诗的代表作，其语言风格体现了诗人"诗中有画"的艺术特色和"诗佛"的精神境界。从用词特点来看，这首诗简洁凝练，全诗仅用40个字便勾勒出一幅完整的秋日山居图。如"空山"、"新雨"、"晚来秋"等词语，既简洁又富有表现力。诗中用词清新自然，多选取"明月"、"清泉"、"竹"、"莲"等自然景物，避免华丽辞藻，给人以清新之感。同时，诗人善于运用动静结合的词语，既有静态的"空山"、"明月"，也有动态的"流"、"喧"、"动"，使画面更加生动。在色彩表现上，虽未直接使用色彩词，但通过意象组合自然地在读者脑海中形成色彩鲜明的画面。

在修辞手法方面，这首诗运用了多种技巧。颔联"明月松间照，清泉石上流"和颈联"竹喧归浣女，莲动下渔舟"都是工整的对仗，结构对称，音韵和谐。诗中还巧妙运用拟人手法，如"竹喧归浣女"，将竹林的声音拟人化，仿佛竹林在欢送浣女归来。借景抒情是这首诗的重要特点，通过描绘自然景物，表达诗人对隐居生活的热爱和对自然的向往，尤其是尾联"随意春芳歇，王孙自可留"，表面上是说春天的花草可以随意凋谢，自己可以留在山中，实际上表达了诗人对隐居生活的满足和对世俗的超脱。此外，诗人还运用衬托和意象叠加等手法，如用"空山新雨后"衬托山中的宁静和清新，通过叠加山、雨、月、松、泉、石、竹、莲等意象，共同构成一幅完整的山水画卷。

从句式结构看，这首诗是一首标准的五言律诗，共八句，每句五字。全诗起承转合，结构完整，层次分明：首联"空山新雨后，天气晚来秋"为起，点明时间地点；颔联"明月松间照，清泉石上流"为承，描绘自然景色；颈联"竹喧归浣女，莲动下渔舟"为转，引入人物活动；尾联"随意春芳歇，王孙自可留"为合，表达诗人的情感和态度。诗中前后呼应，首联的"空山"与尾联的"王孙自可留"形成呼应，暗示了诗人在空山中找到心灵的归宿。颔联和颈联中，静态的景物描写与动态的人物活动相结合，使画面更加生动。

在意象营造上，这首诗通过精心选择的意象，营造出独特的艺术效果。诗中运用了大量的自然意象，如"空山"、"新雨"、"明月"、"松"、"清泉"、"石"、"竹"、"莲"等，这些意象共同构成了一幅清新自然的山水画卷。同时，诗人也巧妙融入人文意象，如"浣女"、"渔舟"、"王孙"等，将人的活动与自然景观融为一体，展现了人与自然和谐相处的理想境界。通过"新雨后"、"晚来秋"等词语，诗人营造出特定的时间和空间氛围，使读者仿佛置身于秋日傍晚的山中。诗中既有视觉意象（如"明月"、"莲动"），也有听觉意象（如"竹喧"），多感官的描写使画面更加立体生动。此外，诗中的意象既有实景描写，也有虚景想象，虚实结合，意境深远。

在音律节奏方面，这首诗平仄协调，作为一首律诗，诗中的平仄搭配协调，读来抑扬顿挫，富有音乐性。全诗押"ou"韵（秋、流、舟、留），韵脚整齐，增强了诗的音乐美。每句五个字，节奏鲜明，读来朗朗上口。诗中字词的声调搭配和谐，如"空山新雨后"（平平平仄仄），"天气晚来秋"（平仄仄平平），读来抑扬顿挫，富有韵律感。

整体而言，这首诗的风格清新淡雅，语言不追求华丽，而是以自然朴素见长，意境淡雅，充分体现了王维的艺术特色。诗人的情感表达含蓄蕴藉，不直接抒发，而是通过景物描写间接流露，如"随意春芳歇，王孙自可留"，含蓄地表达了对隐居生活的满足。诗中情景交融，景物描写与情感表达融为一体，如"空山"既是对环境的描写，也暗示了诗人内心的空明澄澈。作为一位虔诚的佛教徒，王维的这首诗还体现了禅意悠远的特点，通过对自然景物的描绘，表现了一种超然物外、与自然合一的精神境界，展现了"诗佛"的独特艺术风格。
## 📚 文化札记
《山居秋暝》是王维山水田园诗的代表作，体现了"诗中有画"的艺术特色。以下是重要的文化背景注释：

"空山"不仅指人迹罕至的山中，更蕴含禅宗"空"的思想，表现超然物外的心境。王维晚年笃信佛教，尤其是禅宗，这种思想使他的诗歌呈现出空灵、超脱的特质。

"山居"在中国传统文化中有特殊含义，代表一种超脱世俗、回归自然的生活方式。自魏晋以来，隐居山林成为文人雅士追求精神自由的选择。王维在终南山辋川别业（今陕西蓝田县）过着半官半隐的生活，这首诗正是他山居生活的写照。

"明月"、"清泉"、"竹"、"莲"等意象在中国传统文化中具有丰富象征意义：明月象征高洁清朗；清泉象征纯净流动；竹象征正直谦逊；莲象征纯洁高雅，"出淤泥而不染"。这些意象构成了一幅意境深远的山水画卷。

"浣女"指洗衣的女子，是古代乡村生活的常见场景；"渔舟"代表渔樵耕读中的渔夫生活，常被描绘为超脱世俗、亲近自然的形象。

"王孙"一词用典出自《楚辞·招隐士》："王孙游兮不归，春草生兮萋萋。"原意是召唤隐士归来，而王维反用其意，表示即使春天已过，王孙（也指自己）仍然可以留在山中，表达对隐居生活的热爱和满足。

秋天在中国传统文化中常与萧瑟凄凉联系，但王维笔下的秋天却呈现宁静清新的美感，反映了他超然物外的心境。"春芳歇"暗示时光流逝，但诗人以平和心态接受这种变化，体现禅宗"随缘不变，不变随缘"的思想。

王维（701-761），字摩诘，号摩诘居士，官至尚书右丞，世称"王右丞"。他是盛唐著名诗人、画家、音乐家，诗歌风格清新淡雅，意境深远，对后世影响深远，被誉为"诗佛"。
## 🌅 创作背景
《山居秋暝》创作于唐玄宗天宝年间（约742-756年），地点在终南山辋川别业。这是王维购置的一处山庄，位于今陕西省蓝田县终南山中，风景秀丽，环境幽静。

当时王维虽担任右拾遗、监察御史等官职，但大部分时间在此过着半官半隐的生活。这一时期，他已年过不惑，历经仕途起伏，心境趋于平和淡泊。王维深受佛教禅宗思想影响，追求心灵的宁静与超脱，对官场生活逐渐疏离，更向往在自然山水间寻找精神寄托。

从历史背景看，这一时期正处于唐朝由盛转衰的关键阶段。表面上是开元盛世的延续，社会安定，经济繁荣，文化昌盛；但实际上朝廷内部已暗流涌动，李林甫、杨国忠等人专权，安禄山势力崛起，政治环境日趋复杂。这种局势使得许多知识分子如王维选择远离政治中心，寻求精神上的慰藉。

王维创作此诗的动机源于对自然山水的热爱和对隐居生活的向往。诗中描绘了秋日傍晚山中雨后的清新景象，表现了诗人对自然的细腻观察和深刻感悟。灵感直接来源于他在辋川别业的日常生活体验，以及他对佛教禅宗"空寂"思想的理解。通过这首诗，王维不仅表达了对自然美的赞美，也展现了他超然物外、随遇而安的人生态度，反映了他追求心灵自由与宁静的精神追求。
## 💭 情感基调
《山居秋暝》展现了王维独特的山水田园风格，其情感基调宁静淡远，氛围清新幽雅。诗中通过"空山"、"明月"、"清泉"等意象，营造出一种超然物外的意境。诗人以平和的心态观照自然，人与自然和谐共处，如"竹喧归浣女，莲动下渔舟"所呈现的生机与静谧并存。整体氛围清新脱俗，恬淡怡然，体现了诗人对山居生活的热爱和随遇而安的心境，以及"王孙自可留"所表达的归隐之乐和超脱情怀。这首诗的情感基调和氛围特点可概括为：宁静、清新、闲适、和谐、超然。
## 🔍 赏析
《山居秋暝》是王维山水田园诗的代表作，体现了"诗中有画"的艺术特色。在意境营造上，诗人以"空山新雨后"勾勒出雨后秋山的整体氛围，通过动静结合的手法，既有"明月松间照"的静态美，又有"清泉石上流"的动态感，形成空灵清幽的意境。诗人巧妙运用远近视角转换，从远山到近景，再到人物活动，营造出立体空间感，同时视听结合，使读者如临其境。语言方面，王维用词精准简洁，"空"字既写山之空旷，又暗示内心空明；"新"字点出雨后清新感。诗中多处运用对偶，如"明月松间照，清泉石上流"，节奏鲜明，音韵和谐。意象叠加手法使画面层次丰富，意境深远。情感表达上，诗人流露出对自然的热爱和隐居生活的满足，"随意春芳歇，王孙自可留"直接表达了不愿离去的决心，体现了诗人受佛教思想影响的宁静心境和超脱态度。这首诗作为盛唐诗歌的典范，对后世文学创作产生深远影响，特别是"明月松间照，清泉石上流"成为千古传诵的名句。它所体现的隐逸思想和对自然的审美态度，成为中国传统文化的重要组成部分，其"情景交融"的艺术特点也成为古典诗歌美学的重要标准。
