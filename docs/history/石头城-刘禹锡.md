## 📜 原文
```
山围故国周遭在，
潮打空城寂寞回。
淮水东边旧时月，
夜深还过女墙来。
```
## 🎯 主题
《石头城》一诗通过描绘故都荒凉景象，表达了深沉的历史沧桑感和怀古之情。诗人以"山围故国"、"潮打空城"勾勒出石头城的寂寥现状，又以"旧时月"和"女墙"的意象，形成自然永恒与人事无常的鲜明对比。全诗流露出对历史兴衰的思考，展现了繁华不再的哀叹，同时也体现了诗人对时代变迁的敏锐感知和对历史长河中人类命运深沉的哲学思考。
## 🏗️ 结构
《石头城》作为刘禹锡的代表作之一，在结构安排上体现了精妙的章法布局和层次设计。这首七言绝句虽然仅有四句，却通过严谨的结构安排，展现了丰富的历史内涵和深刻的哲理思考。

从章法布局来看，此诗遵循了绝句典型的"起承转合"结构。首句"山围故国周遭在"为"起"，点明地理环境，奠定全诗基调；次句"潮打空城寂寞回"为"承"，承接首句的地理描写，进一步渲染荒凉氛围；第三句"淮水东边旧时月"为"转"，由空间描写转向时间维度，引入历史感慨；末句"夜深还过女墙来"为"合"，收束全诗，以永恒的月光与短暂的人类文明形成对比，完成对历史变迁的深刻思考。

在层次安排上，诗人精心设计了多重层次。空间层次上，从远山到城墙，再到淮水和女墙，形成由远及近、由大到小的空间转换，使读者如同镜头般逐渐聚焦于历史的细节。时间层次上，从现在的荒凉景象，到"旧时月"所连接的历史长河，再回到"夜深"的当下，形成时间的循环与交错，增强了历史感。意象层次上，山（永恒）→城（变迁）→潮水（循环）→月亮（见证者）→女墙（遗迹），意象层层递进，构成一个完整的意象系统。

此外，诗人还巧妙运用对比手法强化结构张力。"故国"与"空城"形成繁华与荒凉的对比；"周遭在"与"寂寞回"形成自然依旧与人事已非的对比；"旧时月"与"夜深还过"形成月亮的永恒与人类历史短暂的对比。这些对比不仅丰富了诗歌的内涵，也使结构更加紧凑有力。

视角转换也是此诗结构的重要特点。从空间视角（山围故国）到时间视角（旧时月），从宏观（山、城）到微观（女墙），从现实（潮打空城）到历史（旧时月）再回到现实（夜深还过），视角的多重转换使诗歌在短小的篇幅中呈现出广阔的历史画卷和深刻的人生感悟。

情感表达上，诗歌也呈现出递进结构：从客观描述（山围故国）到情感渲染（寂寞回），从历史感慨（旧时月）到深沉思考（夜深还过女墙来），情感由浅入深，由表及里，使读者在欣赏诗歌美景的同时，也能感受到诗人对历史变迁的深沉感慨。

总体而言，《石头城》通过严谨的章法布局和精心的层次安排，在短小的篇幅中构建了一个既有画面感又有历史深度的诗歌世界，展现了刘禹锡高超的艺术技巧和深刻的历史思考。
## ✍️ 语言风格
刘禹锡的《石头城》语言风格凝练含蓄，意境深远，体现了中唐诗歌特有的历史沧桑感和哲理思考。这首诗通过简洁的笔触勾勒出石头城的荒凉景象，抒发了对历史兴衰的深沉感慨。

在用词方面，这首诗表现出精炼而富有表现力的特点。全诗仅28字，却勾勒出一幅完整的历史画卷。"故国"与"空城"形成鲜明对比，直接点明历史变迁的主题；"寂寞"一词情感色彩浓厚，点明了诗人的情感基调；"旧时"则增强了诗歌的历史厚重感。这些词语的选择既准确又富有内涵，体现了刘禹锡高超的语言驾驭能力。

修辞手法上，诗人运用了多种技巧增强艺术效果。首先是拟人手法，"潮打空城寂寞回"中的"寂寞回"将潮水人格化，赋予其人的情感，使自然景物与人的情感产生共鸣。其次是对比手法，通过"故国"与"空城"、"旧时"与"现在"的对比，突显了历史的沧桑变迁。借景抒情也是这首诗的重要特点，诗人借石头城的荒凉景象，抒发了对历史兴衰的感慨，达到情景交融的艺术效果。此外，诗中的"月"象征着永恒，与人事变迁形成对比，增强了诗歌的哲理深度；而用自然的永恒（山、潮、月）衬托人事的变迁，则突显了历史的无情和人生的短暂。

句式结构上，这首诗对仗工整，前两句"山围故国周遭在，潮打空城寂寞回"形成对仗，结构严谨。全诗层次分明，先写山，再写水，后写月，由远及近，由大到小，层次清晰。首尾呼应也是其结构特点，首句写"山围故国"，末句写"月过女墙"，形成空间上的呼应，增强了诗歌的整体感。

在意象运用上，诗人巧妙地运用了多种意象：山象征着永恒和不变，与人事变迁形成对比；潮象征着时间的流逝和历史的变迁；空城象征着历史的遗迹和过去的辉煌；月象征着永恒和不变，见证着历史的变迁；女墙则象征着历史的遗迹和过去的防御工事。这些意象的运用不仅丰富了诗歌的内涵，也营造出了深沉的历史感和苍凉的意境。

总体而言，刘禹锡的《石头城》通过精炼的语言、丰富的修辞手法、严谨的结构和深刻的意象，成功地营造出一种历史的沧桑感和哲理的深度，体现了作者作为中唐杰出诗人的艺术成就和思想深度。
## 📚 文化札记
《石头城》是刘禹锡创作的一首著名的怀古诗，描写了六朝古都南京（古称石头城）的衰落景象，抒发了对历史兴衰的感慨。以下是这首诗中重要的文化背景注释：

石头城：位于今南京市，是六朝（吴、东晋、宋、齐、梁、陈）的都城，因城内有石头山而得名。它是长江下游的重要军事要塞，在历史上曾是中国南方的政治、经济和文化中心。到刘禹锡所处的中唐时期，石头城已失去都城地位，成为一座衰败的空城。

故国：指六朝时期的都城，这里特指已经衰落的六朝故都。六朝时期（公元222年-589年）是中国历史上政权更迭频繁的时期，六个朝代相继在此建都，使得石头城曾经非常繁华。

淮水：这里指秦淮河，而非安徽北部的淮河。秦淮河是南京的母亲河，六朝时期是繁华的商业区和文人雅士聚集地，两岸歌楼酒肆林立，是当时著名的娱乐场所，有"六朝金粉"之称。在唐诗中，秦淮河常常作为繁华与衰落的象征出现。

女墙：又称雉堞，是城墙上的矮墙，作为防御设施。在诗中，它象征着石头城曾经的辉煌和军事重要性，如今却只剩下月光越过它的景象，暗示了城市的衰落。

这首诗体现了中国传统怀古诗的特点，通过对古迹的描写，抒发对历史兴衰的感慨。诗中通过山、潮、月等自然元素与"故国"、"空城"等人事元素的对比，表达了自然永恒而人事易变的哲理思考，反映了中国古典诗歌中常见的历史意识和哲学思考。刘禹锡生活在唐朝由盛转衰的时期，对历史兴衰有着深刻的体会，这首诗也寄托了他对唐朝命运的忧虑。
## 🌅 创作背景
《石头城》创作于唐德宗贞元年间（约785-805年），地点在今江苏省南京市的石头城遗址。刘禹锡当时正值中年，可能因游历或任职途经此地。作为一位经历过政治挫折的文人，他曾因参与王叔文政治改革而被贬谪，个人遭遇与对国家命运的忧虑交织，形成了复杂的心境。

从历史背景看，安史之乱（755-763年）后，唐朝国力逐渐衰退，藩镇割据，中央集权削弱。刘禹锡生活的时代正是唐朝由盛转衰的关键时期。石头城作为六朝（吴、东晋、宋、齐、梁、陈）的都城，曾经繁华一时，但到唐代已成为废墟。这种历史的兴衰更迭，与唐朝当时的处境形成了呼应，引发诗人对历史和现实的深刻思考。

刘禹锡创作此诗的直接动机，是他面对石头城遗址时触景生情。诗中描绘的荒凉景象与永恒的自然形成鲜明对比，表达了对历史变迁、物是人非的感慨。通过对六朝故都的凭吊，刘禹锡实际上是在借古讽今，表达对唐朝国运的关切和思考，体现了文人忧国忧民的情怀和对历史规律的深刻洞察。
## 💭 情感基调
《石头城》的情感基调以怀古伤今为主，通过山、城、潮、月等意象，营造出一种深沉的历史沧桑感。诗中"空城"、"寂寞"等词直接点明了荒凉冷清的氛围，整体呈现出物是人非的哀婉情绪。诗人以静默的自然景物对照人世变迁，表达了对繁华逝去的无限感慨，情感内敛而深远，氛围凄清而寂寥，透露出一种历史的厚重感和时光流逝的无奈。
## 🔍 赏析
《石头城》是刘禹锡怀古诗的代表作，通过简练而深刻的笔触，描绘了南京（古称石头城）的荒凉景象，抒发了对历史变迁的深沉感慨。

在意境营造和艺术手法上，诗人采用时空交错的笔法，首句"山围故国周遭在"以山为背景，点明地点，"故国"二字即奠定历史厚重感。次句"潮打空城寂寞回"通过潮水拍打城墙的动态意象，表现时光流逝与城池荒凉，"寂寞"二字直接点出环境氛围。后两句将视角转向淮水和明月，"旧时月"与"女墙"形成古今对比，月光依旧照耀而人事已非，营造出苍凉孤寂的历史感。诗人巧妙运用对比、映衬等手法，将永恒自然与短暂人类文明并置，突显历史沧桑。

语言特色方面，这首七言绝句用词精炼含蓄，"故国"、"空城"、"旧时"、"女墙"等词语既点明地点时代，又蕴含丰富情感。修辞上，诗人运用拟人手法，如"潮打空城寂寞回"，赋予潮水以人的情感；同时通过象征手法，以"女墙"象征历史遗迹，以"月"象征自然永恒，增强了诗歌的艺术感染力。

情感表达上，诗人流露出对历史变迁的深切感慨和对故国衰落的哀思。"寂寞"一词直接表达情感基调，"旧时月"则暗示对往昔的追忆。主题思想方面，诗歌探讨了历史变迁与自然永恒的哲学命题，表达了人类文明在时间长河中的渺小和自然界的永恒不变。作为经历安史之乱后的中唐诗人，刘禹锡在诗中也寄托了对国家前途的忧虑和对民族文化的珍视。

文学价值上，这首诗展现了唐代怀古诗的艺术特色和思想深度，成为后世怀古诗的典范。它体现了中唐文人的历史意识和忧患意识，反映了那个时代由盛转衰的社会背景下文人的思想情感。对后世文学产生深远影响，其艺术手法和思想内涵为后世诗人提供了丰富借鉴。同时，诗中所蕴含的对历史变迁的思考和对文化传承的关切，超越了时代局限，具有普遍的人文价值和现实意义，至今仍能引发读者对历史、文明与自然关系的深刻思考。
