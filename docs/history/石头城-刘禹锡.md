## 📜 原文
```
山围故国周遭在，
潮打空城寂寞回。
淮水东边旧时月，
夜深还过女墙来。
```
## 🎯 主题
刘禹锡的《石头城》主要表达了历史兴衰的深沉感慨和物是人非的沧桑之感。诗中通过对南京（石头城）昔日繁华与今日荒凉的对比，展现了时间流逝中人事变迁的无奈。环绕的群山依旧，潮水依然拍打，月亮依然升起，这些都是永恒的自然景象；而城池已经空荡，昔日的繁华不再，形成了强烈的对比。诗人借景抒情，以自然之永恒反衬人事之短暂，既是对六朝兴亡的追忆，也是对中唐国势衰微的隐忧，体现了文人深沉的历史责任感和忧患意识。整首诗意境苍凉，语言凝练，通过简洁的意象勾勒出宏大的历史图景。
## 🏗️ 结构
《石头城》作为一首七言绝句，在结构上体现了严谨的章法布局和精妙的层次安排。从整体结构看，全诗遵循了绝句"起承转合"的基本章法，四句之间环环相扣，层层递进。

首句"山围故国周遭在"为"起"，点明地点和主题，以"山围"勾勒出石头城的地理环境，"故国"二字奠定了全诗的历史感。这一句从空间上为读者构建了一个宏大的背景。

次句"潮打空城寂寞回"为"承"，承接首句的地理描写，将视角从远处的山脉转向近处的江潮，同时通过"空城"和"寂寞"注入情感色彩，暗示昔日的繁华已逝。

第三句"淮水东边旧时月"为"转"，引入新的意象"月"，并将视角从空间转向时间维度。"旧时月"三字巧妙地将读者的思绪引向过去，形成时空的转换。

末句"夜深还过女墙来"为"合"，总结全诗，通过月亮依然照过女墙的景象，点明历史变迁而自然永恒的哲理，使全诗意境得到升华。

在层次安排上，这首诗呈现出多维度的结构特点：

空间层次上，诗人采用了由远及近、由大到小的布局方式。从远处的"山围"，到近处的"潮打"，再到"淮水东边"的中景，最后聚焦于"女墙"的近景，形成了一个空间上的递进关系。

时间层次上，诗人巧妙地交织了现在与过去。首句和次句描绘当前的景象，第三句通过"旧时月"引入历史维度，末句又回到现在的夜深时刻，形成时间上的回环结构。

意象层次上，诗歌从自然景观（山、潮）到人文景观（故国、空城、女墙），再到天象（月），最后又回到人文景观，形成了一种意象上的流动与呼应。

此外，诗中还存在多处对比与呼应：山与潮的静与动，故国与空城的繁华与寂寥，旧时月与现在夜深的时间跨越，这些对比与呼应增强了诗歌的张力，使短短四句诗意蕴丰富，余韵悠长。

通过这种严谨而富有变化的章法布局和多层次的结构安排，刘禹锡成功地在有限的篇幅内表达了深刻的历史感慨和哲理思考，展现了唐代绝句艺术的精湛成就。
## ✍️ 语言风格
刘禹锡的《石头城》是一首典型的怀古诗，语言风格简约含蓄，意蕴深远。全诗仅四句二十八字，却通过精心选择的词语和丰富的修辞手法，构建出一幅凄清萧瑟的历史画卷，表达了对六朝兴衰的深沉感慨。

在用词方面，诗人选择了富有历史感和情感色彩的词汇。"故国"、"空城"、"旧时"等词直接点明历史变迁的主题，暗示昔日繁华已逝；"周遭在"表明自然景物依然存在，与人事变迁形成对比；"寂寞回"中的"寂寞"一词，将潮水人格化，赋予情感色彩；"还过女墙来"中的"还"字，暗示月亮照临此地已久，见证了历史的沧桑。

修辞手法上，诗人运用了多种技巧。首先是对比手法，"山围故国周遭在"与"潮打空城寂寞回"形成鲜明对比，山依旧而城已空，突显历史变迁的无情。其次是拟人手法，"潮打空城寂寞回"将潮水人格化，使自然景物与人的情感相融合。再次是象征手法，月亮作为永恒的象征，与"旧时"相连，暗示历史的循环往复。此外，诗人还运用了借景抒情的手法，通过描写山、潮、月等自然景物，抒发对历史兴衰的感慨。

在意象构建上，诗中叠加了山、城、潮、淮水、月、女墙等多种意象，这些意象既有实景又有虚景，既有自然景物又有人文遗迹，构成一幅时空交错的历史画卷。特别是"旧时月"这一意象，既是实景描写，又是对过去的追忆，体现了诗人独特的时空观念。

句式结构上，前两句写景，后两句抒情，形成由实到虚、由景到情的过渡。每句七言，结构整齐，节奏感强。首句与次句、第三句与末句之间形成呼应，结构严谨而完整。

音韵方面，全诗押"回"韵（"在"、"回"、"来"），韵律和谐，符合七言绝句的平仄要求，读来抑扬顿挫，富有音乐美。

总体而言，《石头城》体现了刘禹锡诗歌沉郁顿挫、含蓄蕴藉的风格特点，通过简约的语言和丰富的修辞手法，成功营造了一种凄清萧瑟的氛围，表达了对历史兴衰的深沉感慨，是唐代怀古诗的代表作之一。
## 📚 文化札记
刘禹锡的《石头城》是一首咏史怀古的七言绝句，通过对南京古城的描写，抒发了对历史兴衰的感慨。以下是这首诗中重要的文化背景注释：

石头城：指今南京，因三国时期孙权在清凉山（又称石头山）筑城而得名。石头城依山傍水，地势险要，是古代重要的军事要塞和六朝都城的象征。南京作为"六朝古都"（东吴、东晋、宋、齐、梁、陈），在中国历史上具有特殊地位。

故国：指六朝时期的都城建康（今南京）。这里"故国"一词带有浓厚的怀旧色彩，暗示了昔日的繁华已经消逝。

淮水：此处的"淮水"并非指北方的淮河，而是指流经南京的秦淮河。秦淮河是南京的母亲河，六朝时期两岸繁华，文人雅士聚集，是当时文化娱乐的中心，有"十里秦淮"之称。

女墙：城墙上的矮墙，又称"雉堞"，是古代城墙防御系统的一部分。女墙不仅具有防御功能，也是城墙建筑的典型特征。

旧时月：月亮在中国古典诗词中常作为时间的见证者。这里"旧时月"指曾经照耀六朝繁华的月亮，与现在的荒凉形成对比，表达了物是人非的历史沧桑感。

这首诗创作于中唐时期，当时唐朝已经由盛转衰。刘禹锡通过描写石头城的荒凉景象，寄托了对唐朝命运的忧虑和对历史兴衰的深刻思考。诗中以山、潮、月、城等意象，营造出一种苍凉悲壮的氛围，体现了中国古典诗歌中常见的"怀古伤今"主题。
## 🌅 创作背景
《石头城》创作于唐德宗贞元末年至唐宪宗元和年间（约805年前后），当时刘禹锡途经或游览金陵（今南京）时所作。石头城是南京的古称，曾是三国东吴和六朝时期的都城，具有深厚的历史底蕴。

创作此诗时，刘禹锡正经历人生的重要转折。他早年参与"永贞革新"失败后，被贬为朗州司马，后虽被召回长安，但仕途始终坎坷。这一时期，他亲历了政治挫折，对国家命运和个人遭遇都有深刻思考。诗中流露的寂寞与沧桑，正是他此时心境的真实写照。

从历史背景看，唐朝经历了安史之乱后，国力由盛转衰，中央集权削弱，藩镇割据严重。虽然元和年间有短暂中兴，但已无法恢复盛唐气象。刘禹锡面对曾经繁华的六朝古都如今只剩下断壁残垣，联想到唐朝的处境，产生了强烈的历史感慨。

刘禹锡创作此诗的动机，一方面源于对历史兴衰的深刻感悟，另一方面也寄托了对国家前途的忧虑。当他站在石头城遗址，看到"山围故国"、"潮打空城"的景象，联想到昔日六朝的繁华与今日的荒凉，历史的沧桑感油然而生。诗中"淮水东边旧时月，夜深还过女墙来"的意象，既是对自然景观的描绘，也是对历史变迁的哲思，体现了诗人深沉的忧患意识和对历史的独特理解。
## 💭 情感基调
《石头城》的情感基调可概括为怀古、伤感、寂寥。诗人通过对故国遗址的描绘，流露出对昔日繁华的追忆和对当下荒凉的感慨，内心充满孤独与惆怅。

氛围特点则表现为荒凉、冷清、沧桑。诗中"空城"、"寂寞回"等词语直接营造了寂静无人的景象；"故国"、"旧时月"则暗示了历史变迁带来的沧桑感；而山围潮打、月照女墙的意象，更强化了时光流逝、物是人非的凄凉氛围。整首诗以静默的自然景物反衬人事变迁，形成一种深沉而凝重的历史感伤。
## 🔍 赏析
这首诗通过简练而富有意境的语言，描绘了六朝古都金陵石头城的荒凉景象，表达了诗人对历史变迁的深沉感慨。在意境营造上，诗人以"山围故国"起笔，勾勒出石头城被群山环绕的地理环境，"周遭在"三字暗示自然永恒与人事变迁的对比。继以"潮打空城寂寞回"，通过拟人手法，赋予潮水以情感，强化了城市的荒凉感。"淮水东边旧时月"一句，以月亮为历史的见证者，连接古今，而"夜深还过女墙来"则以静谧的夜景收束全诗，营造出苍凉孤寂的艺术氛围。语言特色方面，全诗仅二十八字，却意蕴深远。诗人善用对比手法，"故国"与"空城"、"旧时"与"现在"形成鲜明对照；同时运用象征技巧，"山围"象征永恒，"潮打"象征时间流逝，"旧时月"象征历史见证。语言平实自然，不事雕琢，却含蓄深远，体现了唐诗"言有尽而意无穷"的艺术特色。情感表达上，诗人通过对石头城荒凉景象的描绘，抒发了对历史变迁的感慨和对时光流逝的思考。繁华的六朝都城已成空城，唯有自然景物依旧，这种对比蕴含着深刻的"兴亡之感"和"无常之叹"。主题思想聚焦于历史变迁、时光流逝、繁华易逝与自然永恒的哲理思考，体现了中国传统文化中的历史观和人生观。作为唐代怀古诗的代表作，《石头城》在中国文学史上具有重要地位。它不仅以其深刻的思想内容和精湛的艺术表现成为经典之作，还对后世文学产生深远影响，如苏轼《念奴娇·赤壁怀古》等作品都可见其影子。同时，这首诗也具有重要的文化价值，记录了历史遗迹的变迁，反映了唐代文人的历史观和人生观，为理解中国传统文化提供了重要窗口。
