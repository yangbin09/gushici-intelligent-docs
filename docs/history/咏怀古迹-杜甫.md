## 📜 原文
```
群山万壑赴荆门，生长明妃尚有村。
一去紫台连朔漠，独留青冢向黄昏。
画图省识春风面，环珮空归月夜魂。
千载琵琶作胡语，分明怨恨曲中论。
```
## 🎯 主题
杜甫的《咏怀古迹》通过咏叹王昭君的悲剧命运，表达了深刻的历史感慨和个人情怀。诗中描绘了昭君因画师作假而被错嫁匈奴、最终客死他乡的不幸遭遇，展现了她从皇宫到荒漠的人生转折和死后孤坟的凄凉景象。诗人借昭君的故事，抒发了对个人命运与国家政治关系的思考，揭示了在历史大潮中个体的渺小与无奈。同时，诗中也流露出对美的毁灭的惋惜、对故土思念的共鸣，以及对历史真相被掩盖的深刻反思。最后以"怨恨"二字点明昭君千年不绝的悲情，也寄托了诗人对自身处境的感怀，体现了杜甫对历史与现实的双重关怀。
## 🏗️ 结构
杜甫的《咏怀古迹》是一首结构精巧的七言律诗，其章法布局和层次安排体现了诗人高超的艺术构思。从整体结构看，此诗遵循传统律诗"起承转合"的章法，四联各有侧重又紧密联系，形成完整的艺术整体。

首联"群山万壑赴荆门，生长明妃尚有村"为"起"，从地理环境入手，以壮阔的自然景观引出昭君故里。诗人先以"群山万壑"的宏大意象开篇，再聚焦到"明妃村"的具体地点，由远及近，由大到小，为全诗奠定基调。这种以景起兴的手法，既点明了昭君的出生地，又暗示了她命运的壮阔与悲凉。

颔联"一去紫台连朔漠，独留青冢向黄昏"为"承"，概括昭君一生的悲剧命运。此联在时间上跨越昭君的一生，从"一去紫台"的远嫁到"独留青冢"的死亡；空间上则从汉宫到匈奴，再到坟墓，形成广阔的时空跨度。"紫台"与"青冢"、"朔漠"与"黄昏"的对比，既工整对仗，又形成繁华与荒凉、生与死的强烈反差。

颈联"画图省识春风面，环珮空归月夜魂"为"转"，深入揭示昭君悲剧的具体原因和死后的哀思。诗人从历史事件转向情感抒发，点明昭君悲剧的直接原因是画师故意画丑，导致汉元帝未能识其真容；后句则写昭君死后魂魄只能在月夜中空自归来，充满哀婉之情。此联对仗极为工整，"画图"对"环珮"，"省识"对"空归"，"春风面"对"月夜魂"，形式与内容完美统一。

尾联"千载琵琶作胡语，分明怨恨曲中论"为"合"，总结昭君故事的历史意义和情感内涵。诗人将时间拉长到"千载"，从个人悲剧上升到历史评价，点明昭君的故事通过琵琶曲流传后世，其中的怨恨情感历久弥新。这一收束既呼应了开头，又升华了主题，使全诗在时空上形成完整的闭环。

在层次安排上，此诗呈现出多重维度的递进：空间上从群山万壑到明妃村，再到紫台、朔漠、青冢，最后超越具体空间；时间上从昭君出生，到远嫁、死亡，延伸到千载之后；情感上从客观描述到悲叹命运，再到揭示原因，最后升华为历史评价。这种多层次的安排，使诗歌内容丰富而深刻。

此外，诗中意象的选择和安排也极具匠心，如"群山万壑"、"紫台"、"朔漠"、"青冢"、"画图"、"环珮"、"琵琶"等，既有实景描绘，又有象征意义，共同构建了昭君悲剧命运的形象画面。这些意象按照从自然到人文、从具体到抽象的顺序排列，使诗歌层次分明，意境深远。

总体而言，杜甫的《咏怀古迹》结构严谨，层次分明，章法布局遵循传统又不拘泥，既有历史的厚重感，又有情感的感染力，体现了诗人高超的艺术造诣和深厚的历史情怀。
## ✍️ 语言风格
杜甫的《咏怀古迹》展现了其诗歌语言的典型风格，体现出沉郁顿挫、意境深远的特点。从用词方面看，诗中选词精炼而富有表现力，如"群山万壑"以宏大意象开篇，"赴"字赋予静态景物以动感，气势磅礴；"紫台"与"朔漠"形成鲜明对比，暗示昭君命运的巨大转折；"青冢"与"黄昏"的组合营造出凄凉氛围，象征昭君的孤独与凄苦。这些词语不仅描绘了具体形象，更承载着深厚的情感内涵。

修辞手法上，此诗运用了多种技巧增强艺术表现力。首联"群山万壑赴荆门，生长明妃尚有村"采用拟人手法，赋予山壑以人的行动；颔联"一去紫台连朔漠，独留青冢向黄昏"形成工整对仗，同时通过时空对比强化悲剧效果；颈联"画图省识春风面，环珮空归月夜魂"巧妙用典，暗指昭君因画像被误认的历史故事，并以"春风面"比喻昭君美貌，"月夜魂"营造幽冥氛围；尾联"千载琵琶作胡语，分明怨恨曲中论"则通过听觉意象，将昭君的怨恨与音乐融为一体，使情感得以跨越时空。

句式结构上，此诗为七言律诗，格律严谨，对仗工整，前两联侧重写景叙事，后两联转向抒情议论，形成由外而内、由景及情的结构层次。诗中多处运用对比手法，如"紫台"与"朔漠"、"一去"与"独留"、"省识"与"空归"等，通过这些对比强化了昭君命运的悲剧性。

整体而言，这首诗语言风格沉郁含蓄，情感深沉而不直露，通过精妙的意象组合和典故运用，将个人命运与历史背景紧密结合，既表达了对昭君遭遇的同情，也暗含对当时政治的批判，体现了杜甫诗歌"诗史"的特点和深厚的人文关怀。
## 📚 文化札记
《咏怀古迹》是杜甫所作的一首咏史诗，主题围绕汉代美女王昭君（明妃）的故事展开。

"群山万壑赴荆门"中的荆门，位于今湖北省宜昌市东南，是古代楚国的重要关隘，地势险要。

"生长明妃尚有村"中的明妃指王昭君，西汉时期著名美女，汉元帝宫女。村指王昭君的出生地，相传在今湖北省秭归县（一说兴山县）有昭君村。

"一去紫台连朔漠"中，紫台指汉代皇宫，因宫墙多涂紫色而得名，代指汉元帝的宫廷；朔漠指北方的沙漠，即匈奴所在地。

"独留青冢向黄昏"中的青冢是王昭君的坟墓，据说在今内蒙古呼和浩特市南郊，因墓草常青，故称"青冢"。

"画图省识春风面"引用了著名典故：汉元帝按图选妃，宫女们贿赂画师把自己画得美一些，唯独王昭君不肯行贿，被画师毛延寿故意画丑。后来汉元帝见到王昭君本人，发现她非常美丽，但已经答应将她嫁给匈奴呼韩邪单于，后悔莫及。

"环珮空归月夜魂"中，环珮是古代女子佩戴的玉饰，这里指王昭君的魂魄；空归月夜魂形容王昭君死后魂魄在月夜中徒然归来的情景，表达了她思乡而不得归的悲哀。

"千载琵琶作胡语"中，琵琶是古代弹拨乐器，相传王昭君出塞时携带琵琶，常弹奏以抒发思乡之情；胡语指匈奴的音乐语言。

杜甫写此诗时正值安史之乱后，唐朝国力衰退，他自己也漂泊异乡，因此对王昭君的遭遇有深刻共鸣，借古讽今，表达了对国事的忧虑和对故国的思念。
## 🌅 创作背景
杜甫的《咏怀古迹》创作于唐代宗大历元年（公元766年）左右，当时诗人寓居夔州（今重庆奉节）。这是杜甫晚年时期的作品，此时的他已经年近六旬，历经安史之乱的动荡，辗转流离后定居于此。

创作此诗时，杜甫的生活状况颇为困顿。他年老体弱，远离朝廷，政治抱负无法实现，生活较为贫困。在夔州期间，他虽然得到了当地官员的一些资助，但内心依然充满忧国忧民之情和对个人命运的感慨。这种复杂的心境使他常常回顾历史，借古讽今，通过咏叹历史人物和事件来抒发自己的情感。

从历史背景和社会环境来看，此诗创作于安史之乱（755-763年）之后。这场持续八年的叛乱严重削弱了唐朝的国力，导致社会动荡、经济衰退、边疆不稳。唐朝由盛转衰，中央集权减弱，藩镇割据现象严重。同时，与周边民族的关系也发生了变化，唐朝对边疆的控制力减弱，民族问题日益突出。

杜甫创作《咏怀古迹》的动机和灵感来源主要有几方面：首先，王昭君的故事是中国历史上著名的和亲案例，她作为汉元帝时期被送往匈奴和亲的宫女，成为民族团结的象征。杜甫通过咏叹昭君的命运，意在表达对唐朝当时与周边民族关系的思考。其次，杜甫借昭君的遭遇，抒发自己怀才不遇的感慨。昭君因画工毛延寿的恶意而未能得到皇帝的赏识，最终远嫁匈奴；而杜甫自己也因各种原因未能得到重用，无法实现自己的政治抱负。此外，杜甫还通过这首诗表达对唐朝命运的忧虑，借古讽今，反映对当时社会现实的关切。
## 💭 情感基调
杜甫的《咏怀古迹》情感基调主要表现为悲凉怨恨、孤独怀念。诗中通过王昭君的悲剧命运，流露出深沉的哀伤与同情。首联以壮阔山水引出明妃故里，已含追思之情；颔联"一去紫台连朔漠，独留青冢向黄昏"，以"一去"与"独留"的强烈对比，凸显了命运的悲凉与孤独；颈联"画图省识春风面，环珮空归月夜魂"中的"空归"二字，更添无奈与凄清；尾联"千载琵琶作胡语，分明怨恨曲中论"，将个人悲剧升华为千古怨恨，情感深沉持久。

氛围特点上，全诗笼罩在萧瑟苍凉、幽远哀婉之中。"黄昏"、"月夜"等意象营造出萧瑟之感；时空的巨大跨度，从昭君生前到千年之后，形成苍凉幽远的历史氛围；语言凄美而不失庄重，如"青冢向黄昏"等句，既有画面感又充满哀婉情调，使全诗在悲凉中透着一种凄美的艺术效果。
## 🔍 赏析
杜甫的《咏怀古迹》以王昭君为题材，通过凝练的语言和深远的意境，展现了一幅悲壮的历史画卷。在意境营造上，诗人以"群山万壑赴荆门"开篇，勾勒出雄浑壮阔的地理背景，随后通过"紫台"、"朔漠"、"青冢"、"黄昏"等意象，构建了时空交错的立体感。诗中运用对比手法，如"紫台"与"朔漠"、"春风面"与"月夜魂"的强烈反差，以及虚实结合的技巧，如"环珮空归月夜魂"中魂魄与环珮声的结合，营造出梦幻般的氛围，增强了诗歌的艺术感染力。

语言特色方面，杜甫的文字凝练而富有表现力，多用单音节动词如"赴"、"去"、"留"等，简洁有力地表达动作和状态。形容词的使用极为精准，如"青冢"中的"青"字，不仅表现了坟墓的颜色，还暗含了岁月流逝和生命消逝的意味。修辞技巧上，诗人运用借代手法，如"紫台"代指皇宫；运用拟人手法，如"青冢向黄昏"，赋予无生命的坟墓以人的情感；运用象征手法，如"琵琶作胡语"象征昭君的异化与孤独。对仗工整的对偶句增强了诗歌的节奏感和音乐美。

情感表达上，这首诗流露出对王昭君命运的深切同情和对历史悲剧的深刻思考。昭君作为和亲公主，远嫁匈奴，客死异乡，她的故事成为中国古代女性悲剧命运的典型代表。"分明怨恨曲中论"直接点明了全诗的情感基调——怨恨。这种怨恨不仅是个人的，也是时代的，是历史中无数被牺牲者的共同心声。诗人通过昭君的故事，表达了对个人命运无法自主的悲叹，对政治牺牲品的同情，以及对历史无情、人生短暂的感慨。

文学价值方面，这首诗作为杜甫咏史诗的代表作，将历史人物与现实情感相结合，既有对历史的客观描述，又有对历史的主观评价，体现了杜甫"诗史"的特点。它所塑造的昭君形象成为后世文学创作的重要素材，"怨恨"的主题也成为中国古代文学中常见的情感表达。此外，这首诗还具有重要的历史文化价值，是研究唐代社会历史、民族关系和文化交流的重要资料，展现了杜甫作为"诗圣"的人文关怀和艺术成就。
