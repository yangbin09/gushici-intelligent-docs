## 📜 原文
```
支离东北风尘际，漂泊西南天地间。
三峡楼台淹日月，五溪衣服共云山。
羯胡事主终无赖，词客哀时且未还。
庾信平生最萧瑟，暮年诗赋动江关。

摇落深知宋玉悲，风流儒雅亦吾师。
怅望千秋一洒泪，萧条异代不同时。
江山故宅空文藻，云雨荒台岂梦思。
最是楚宫俱泯灭，舟人指点到今疑。

群山万壑赴荆门，生长明妃尚有村。
一去紫台连朔漠，独留青冢向黄昏。
画图省识春风面，环佩空归月夜魂。
千载琵琶作胡语，分明怨恨曲中论。

蜀主窥吴幸三峡，崩年亦在永安宫。
翠华想像空山里，玉殿虚无野寺中。
古庙杉松巢水鹤，岁时伏腊走村翁。
武侯祠屋常邻近，一体君臣祭祀同。

诸葛大名垂宇宙，宗臣遗像肃清高。
三分割据纡筹策，万古云霄一羽毛。
伯仲之间见伊吕，指挥若定失萧曹。
运移汉祚终难复，志决身歼军务劳。
```
## 🎯 主题
《咏怀古迹》这组诗通过咏叹历史古迹和古人，表达了多重主题思想。首先，诗人借庾信、宋玉等历史人物的遭遇，抒发自己在乱世中的漂泊之苦和怀才不遇的悲愤，体现了个人命运的感慨。其次，通过对王昭君、刘备、诸葛亮等历史人物和事件的咏叹，表达了对国家命运的忧虑和对时局的关切。第三，诗中流露出对历史变迁、人事沧桑的深刻思考，感叹物是人非、繁华易逝。第四，诗人通过对诸葛亮等忠臣义士的赞美，表达了对理想政治的向往和对现实政治的失望。整组诗以咏史怀古为形式，以抒发个人情怀和表达政治见解为内容，体现了杜甫"沉郁顿挫"的诗风和忧国忧民的情怀，展现了诗人将个人命运与国家命运紧密联系的思想境界。
## 🏗️ 结构
《咏怀古迹》是杜甫创作的五首五言律诗组诗，整体结构严谨，层次分明，体现了诗人高超的艺术构思。从整体看，这组诗以历史人物古迹为切入点，按时间顺序从庾信、宋玉、王昭君到刘备、诸葛亮，形成一个历史回溯的序列，情感由个人身世之感逐渐深化到国家兴亡之叹。

第一首咏庾信，章法上采用由己及人的结构。首联"支离东北风尘际，漂泊西南天地间"交代背景，点明自身处境；颔联"三峡楼台淹日月，五溪衣服共云山"具体描绘漂泊之地；颈联"羯胡事主终无赖，词客哀时且未还"由己及人，引出庾信；尾联"庾信平生最萧瑟，暮年诗赋动江关"点明主题，咏叹庾信。层次安排上，前四句写自身漂泊之苦，后四句转向对庾信的咏叹，过渡自然。

第二首咏宋玉，结构上开门见山。首联"摇落深知宋玉悲，风流儒雅亦吾师"直接点题，表达对宋玉的敬仰；颔联"怅望千秋一洒泪，萧条异代不同时"抒发对宋玉的同情；颈联"江山故宅空文藻，云雨荒台岂梦思"描绘宋玉故迹；尾联"最是楚宫俱泯灭，舟人指点到今疑"以古迹今貌作结。层次上，由人对景，由情到理，清晰有序。

第三首咏王昭君，采用由景及人的结构。首联"群山万壑赴荆门，生长明妃尚有村"以景起兴，点明昭君故里；颔联"一去紫台连朔漠，独留青冢向黄昏"概括昭君一生遭遇；颈联"画图省识春风面，环佩空归月夜魂"具体描绘昭君形象；尾联"千载琵琶作胡语，分明怨恨曲中论"以琵琶曲作结。层次安排上，从地点到命运，再到具体形象，最后主题升华，环环相扣。

第四首咏刘备，结构上由人到景再到人。首联"蜀主窥吴幸三峡，崩年亦在永安宫"点明刘备事迹和死亡地点；颔联"翠华想像空山里，玉殿虚无野寺中"描绘刘备遗迹今貌；颈联"古庙杉松巢水鹤，岁时伏腊走村翁"进一步描述庙宇现状；尾联"武侯祠屋常邻近，一体君臣祭祀同"以诸葛亮祠庙作结。这一首既独立成篇，又为下一首咏诸葛亮做铺垫，体现了组诗的整体性。

第五首咏诸葛亮，采用由总到分的结构。首联"诸葛大名垂宇宙，宗臣遗像肃清高"直接赞美诸葛亮的历史地位；颔联"三分割据纡筹策，万古云霄一羽毛"评价诸葛亮的功绩；颈联"伯仲之间见伊吕，指挥若定失萧曹"将诸葛亮与历史名臣比较；尾联"运移汉祚终难复，志决身歼军务劳"总结诸葛亮的一生。层次上，从总括地位到具体功绩，再到历史比较，最后总结评价，逻辑严密。

整体而言，这组诗章法布局严谨，每首都遵循律诗的章法，首联点题，中间两联展开，尾联总结或升华。层次安排有序，过渡自然，情感递进，从个人命运到国家兴亡，视野逐渐开阔。五首诗之间有内在联系，形成一个有机整体，充分体现了杜甫"诗史"的特点和深厚的艺术功底。
## ✍️ 语言风格
《咏怀古迹》是杜甫晚年在夔州时期创作的一组五言律诗，共五首，分别咏叹庾信、宋玉、王昭君、刘备和诸葛亮五位历史人物。这组诗展现了杜甫诗歌艺术的成熟与精湛，其语言风格特点主要体现在以下几个方面：

在语言风格上，这组诗呈现出典型的"沉郁顿挫"特点，情感深沉而含蓄。如"支离东北风尘际，漂泊西南天地间"中，诗人通过简洁有力的语言表达了自身漂泊之感，同时暗含对国家动荡的忧虑。整体风格凝练精工，每个字都经过精心推敲，如"三分割据纡筹策，万古云霄一羽毛"中，"纡"字准确表达了诸葛亮运筹帷幄的复杂过程，而"一羽毛"则形象地比喻了诸葛亮在历史长河中的轻盈与超然。诗中既有对历史人物功业的赞美，也有对他们悲剧命运的同情，形成了雄浑而又悲壮的风格。

在用词方面，杜甫大量运用历史典故，如"羯胡事主"指安禄山叛乱，"庾信"是南北朝时期的文学家，"宋玉"是战国时期的楚辞作家，"明妃"即王昭君等，这些典故的运用使诗歌内容更加丰富厚重。诗中意象鲜明，如"风尘"、"三峡"、"五溪"、"云山"、"紫台"、"朔漠"、"青冢"等，既有地域特色，又富有象征意义。作为律诗，诗中的对仗极为工整，如"支离东北风尘际，漂泊西南天地间"、"三峡楼台淹日月，五溪衣服共云山"等，对仗不仅形式工整，内容也相互呼应。杜甫用词极为精炼，一字千金，如"摇落深知宋玉悲"中的"摇落"一词，既指秋天的落叶，也暗喻宋玉的悲凉处境，一词双关，意蕴丰富。

在修辞手法上，杜甫大量运用对比手法，如"支离东北"与"漂泊西南"的对比，"一去紫台"与"独留青冢"的对比，通过对比突出了历史人物的命运和历史的变迁。诗人善于通过景物描写来表达情感，如"三峡楼台淹日月，五溪衣服共云山"通过描写三峡和五溪的景色，表达了诗人漂泊异乡的孤寂感。诗中多处使用象征手法，如"万古云霄一羽毛"中的"羽毛"象征诸葛亮的高洁品格和超然地位，"青冢"象征王昭君的悲剧命运等。此外，诗中还运用了拟人化手法，如"群山万壑赴荆门"中的"赴"字，赋予山壑以人的行动能力，使画面更加生动；以及反问手法，如"云雨荒台岂梦思"，通过反问加强了语气，表达了诗人对历史传说的质疑。

总体而言，《咏怀古迹》通过精湛的语言艺术，不仅表达了对历史人物的敬仰和同情，也寄托了杜甫对国家命运的忧虑和对个人遭遇的感慨。这组诗在思想内容的深刻性和艺术形式的完美性上，都堪称中国古典诗歌的典范之作，充分展现了杜甫作为"诗圣"的艺术成就和思想深度。
## 📚 文化札记
杜甫《咏怀古迹》五首是其在夔州时期的重要作品，通过咏怀历史古迹和人物，抒发个人感慨。以下是这组诗中重要的文化背景注释：

第一首：
"支离东北风尘际"中的"东北"指中原地区，"风尘"指安史之乱的战乱。"西南"指蜀地，杜甫自安史之乱后漂泊至此。"三峡"指瞿塘峡、巫峡、西陵峡，位于长江上游。"五溪"指古代少数民族居住地区，在今湖南西部、贵州东部一带。"羯胡"指安禄山，他是突厥族后裔，发动了安史之乱。"庾信"是南北朝著名文学家，初仕南朝梁，后出使北朝被留，仕于北周。其《哀江南赋》等作品表达对故国的思念，风格沉郁悲凉。"江关"指长江一带的关隘，代指故国。

第二首：
"宋玉"是战国时期楚国人，屈原的学生，著名辞赋家，作品有《九辩》《高唐赋》等。"摇落"出自宋玉《九辩》："悲哉秋之为气也，萧瑟兮草木摇落而变衰。"表达对秋景萧瑟的感伤。"故宅"指宋玉故居，在今湖北秭归。"云雨荒台"指宋玉《高唐赋》中描写的楚王游高唐，梦见巫山神女的故事，"云雨"成为男女情爱的典故。"楚宫"指楚国宫殿，已荒废。"舟人指点"指当地船夫仍能指出宋玉故宅和楚宫遗址，但已难以确定。

第三首：
"荆门"指荆门山，在今湖北宜昌市西北。"明妃"指王昭君（王嫱），西汉时期宫女，后嫁匈奴呼韩邪单于。"明妃"是她入宫后的封号。"村"指昭君故里，在今湖北秭归县。"紫台"指汉代皇宫。"朔漠"指北方沙漠地区，匈奴居住地。"青冢"指昭君墓，传说中她的墓草常青，在今内蒙古呼和浩特市南郊。"画图"指传说中汉元帝凭画像选宫女，王昭君因不愿贿赂画师而被画丑，未被元帝宠幸。"琵琶"相传是王昭君出塞时携带的乐器，她创作了《昭君怨》等曲子，表达思乡之情。

第四首：
"蜀主"指刘备（161-223），三国时期蜀汉开国皇帝。"窥吴"指刘备为报关羽之仇，攻打东吴。"永安宫"在今重庆奉节县，刘备在此病逝。"翠华"指皇帝的车驾和仪仗。"玉殿"指宫殿，这里指刘备的宫殿。"伏腊"指古代夏天的伏日和冬天的腊日，是重要的祭祀节日。"村翁"指当地村民，在节日时前来祭祀。"武侯"指诸葛亮（181-234），蜀汉丞相，封武乡侯。武侯祠与刘备庙相邻，表示君臣一体，受到后人共同祭祀。

第五首：
"宗臣"指一代功勋卓著的大臣。"三分割据"指诸葛亮策划三国鼎立的局面。"万古云霄一羽毛"比喻诸葛亮像高飞的羽毛一样，超然卓绝。"伊吕"指伊尹和吕尚（姜子牙），都是商朝和周朝的开国功臣。"萧曹"指萧何和曹参，汉朝开国功臣。这句是说诸葛亮的才能可以与伊尹、吕尚相比，指挥若定的能力超过萧何、曹参。"汉祚"指汉朝的国运。诸葛亮虽然决心恢复汉室，但最终因军务劳累而病逝五丈原（今陕西岐山县）。

这组诗通过历史人物庾信、宋玉、王昭君、刘备、诸葛亮等，以及相关历史事件和地理景观，表达了杜甫对国家兴衰的忧思和对自身遭遇的感慨，体现了他深沉的历史感和家国情怀。
## 🌅 创作背景
《咏怀古迹》创作于唐代宗大历元年（766年）左右，当时杜甫居住在夔州（今重庆奉节一带）。这是杜甫晚年漂泊西南时期的代表作，由五首五言律诗组成，分别咏叹庾信、宋玉、王昭君、刘备和诸葛亮等历史人物及相关古迹。

创作这组诗时，杜甫已年过五旬，处于人生晚年。他经历了安史之乱的动荡，目睹了唐朝由盛转衰的过程，自己的一生也充满坎坷和挫折。在夔州期间，他依靠都督柏茂琳的资助维持生活，虽然物质条件有所改善，但精神上却十分孤独和苦闷。年老多病的他，仍心系国事，忧国忧民，这种复杂的情感在诗中得到了充分体现。

当时的历史背景是安史之乱（755-763年）后的恢复阶段。虽然叛乱已被平定，但唐朝国力已大不如前，藩镇割据局面开始形成，中央政府权威受到严重削弱。同时，吐蕃等外族不断侵扰边境，国家面临内忧外患。社会动荡不安，民生凋敝，这一切都让深感忧虑的杜甫借古讽今，表达对国家命运的关切。

杜甫创作这组诗的动机主要源于夔州地区丰富的历史遗迹，如宋玉故宅、王昭君村、刘备永安宫、诸葛亮祠等，这些古迹触发了他对历史的追忆和思考。他晚年的漂泊生活和坎坷经历，使他更容易与历史上的悲剧人物产生共鸣。通过咏叹历史人物和古迹，杜甫实际上是在借古抒怀，表达对国家兴衰、人生际遇的深刻感悟，体现了他作为"诗圣"的历史责任感和人文关怀。
## 💭 情感基调
《咏怀古迹》的情感基调以悲凉怅惘为主，兼有哀怨敬仰。五首诗分别咏叹庾信、宋玉、王昭君、刘备和诸葛亮，通过对历史人物的追忆，表达了诗人深沉的历史感慨和人文关怀。

氛围特点呈现出浓厚的历史沧桑感和忧国忧民情怀。诗中既有对历史变迁的无奈感叹，也有对文人命运的深切同情，更有对英雄人物的崇高敬仰。杜甫借古迹咏怀，实则抒发自己对时局的忧虑和对理想的追求，展现出沉郁顿挫而又不失崇高的艺术风格。

整组诗在悲凉中见深沉，在怅惘中显情怀，形成了独特的艺术感染力，体现了杜甫"诗史"的特质和深厚的人文精神。
## 🔍 赏析
杜甫的《咏怀古迹》组诗是其在夔州时期的代表作，通过对历史古迹和人物的咏叹，展现了诗人深沉的历史感慨和现实忧思。在意境营造上，诗人巧妙地将自身处境与历史人物相联系，创造出时空交错的独特意境。如"支离东北风尘际，漂泊西南天地间"勾勒出漂泊处境，与庾信遭遇形成呼应；"一去紫台连朔漠，独留青冢向黄昏"通过对比手法，营造出王昭君命运的悲凉壮阔。诗人运用对比、借景抒情、用典等多种技巧，使意境深远而富有感染力。

语言特色方面，这组诗体现了杜甫"沉郁顿挫"的风格，语言凝练而富有表现力。诗中多用对仗工整的句式，如"三分割据纡筹策，万古云霄一羽毛"，不仅形式整齐，内涵亦丰富。修辞上运用比喻、拟人、借代等手法，如将诸葛亮比作"万古云霄一羽毛"，赋予山壑以"赴荆门"的人性化动作，增强了艺术表现力。诗人还善于化用前人诗句和典故，显示其渊博学识。

情感表达上，这组诗流露出杜甫对历史人物的敬仰、对自身处境的感慨以及对国家命运的忧虑。"怅望千秋一洒泪，萧条异代不同时"抒发了与宋玉相隔千年的惆怅；"运移汉祚终难复，志决身歼军务劳"表达了对诸葛亮壮志未酬的惋惜。主题思想涉及文人的命运与责任、君臣关系、历史变迁与人事无常等，体现了诗人深刻的历史观和人文关怀。

文学价值方面，这组诗代表了杜甫晚年诗歌创作的艺术成就，体现了他"诗史"的创作特征。它将咏史与抒怀完美结合，开创了咏史诗的新境界，对后世产生深远影响。思想上，表达了杜甫对历史和现实的深刻思考，体现了他忧国忧民的情怀；艺术上，结构严谨，语言凝练，意境深远，代表了唐代七言律诗的最高成就，成为中国古典诗歌中的珍品。
