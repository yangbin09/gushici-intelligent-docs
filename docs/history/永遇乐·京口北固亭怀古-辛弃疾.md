## 📜 原文
```
千古江山，英雄无觅孙仲谋处。舞榭歌台，风流总被雨打风吹去。斜阳草树，寻常巷陌，人道寄奴曾住。想当年，金戈铁马，气吞万里如虎。

元嘉草草，封狼居胥，赢得仓皇北顾。四十三年，望中犹记，烽火扬州路。可堪回首，佛狸祠下，一片神鸦社鼓。凭谁问：廉颇老矣，尚能饭否？
```
## 🎯 主题
《永遇乐·京口北固亭怀古》主要表达了辛弃疾深沉的爱国情怀和壮志难酬的悲愤。词人通过追忆孙权、刘裕等历史英雄的丰功伟绩，对比南宋朝廷的软弱无能，抒发了对国家命运的忧虑。词中"元嘉草草"一句暗含对当时朝廷轻率北伐策略的批评，而"廉颇老矣，尚能饭否"的自问，则道出了自己虽年老却仍有报国之志却无人问津的悲凉。整首词以怀古为表，实为伤今，既是对历史兴衰的感慨，也是对现实国事的深切关怀，体现了作者强烈的忧国忧民精神和收复失地的坚定信念。
## 🏗️ 结构
《永遇乐·京口北固亭怀古》是辛弃疾的代表作之一，其结构特点体现了精心的章法布局和层次安排。

一、章法布局

1. 上下片分工明确，上片写景怀古，下片抒情言志。上片以"千古江山"起笔，描绘北固亭景色并追忆孙权、刘裕等历史英雄；下片则转入现实，先写刘义隆北伐失败的教训，继而回忆自己年轻时的抗金经历，最后以廉颇自喻。

2. 首尾呼应，结构完整。开头以"千古江山"起兴，引出对历史英雄的追忆；结尾以"廉颇老矣，尚能饭否"自问，形成首尾呼应，使整首词从怀古到伤今，形成一个完整的情感发展脉络。

二、层次安排

1. 上片层次分明：
   第一层（"千古江山，英雄无觅孙仲谋处"）：以江山永恒起兴，点明怀古主题。
   第二层（"舞榭歌台，风流总被雨打风吹去"）：写历史遗迹变迁，抒发物是人非之感。
   第三层（"斜阳草树，寻常巷陌，人道寄奴曾住"）：由孙权转向刘裕，继续怀古。
   第四层（"想当年，金戈铁马，气吞万里如虎"）：描绘刘裕北伐的雄壮气势，为下片做铺垫。

2. 下片层层递进：
   第一层（"元嘉草草，封狼居胥，赢得仓皇北顾"）：写刘义隆北伐失败的教训。
   第二层（"四十三年，望中犹记，烽火扬州路"）：回忆自己43年前的抗金经历。
   第三层（"可堪回首，佛狸祠下，一片神鸦社鼓"）：写金人占领区的现状。
   第四层（"凭谁问：廉颇老矣，尚能饭否"）：以廉颇自喻，抒发壮志难酬的悲愤。

三、结构特点

1. 对比手法运用：词中多处运用对比，如"千古江山"与"英雄无觅"的对比，"金戈铁马"与"元嘉草草"的对比，突出历史兴衰之感。

2. 虚实结合：既有实景描写（"斜阳草树，寻常巷陌"），又有虚景想象（"想当年，金戈铁马"），增强了艺术感染力。

3. 时间线索交织：词中交织着多条时间线索，从古代孙权、刘裕时代，到刘义隆元嘉年间，再到作者记忆中的四十三年前，以及写作的当下，形成立体结构。

4. 情感递进：从怀古的感慨，到对现实的忧虑，再到个人壮志难酬的悲愤，情感逐步深入，形成内在张力。

这首词通过严谨的章法布局和层次安排，充分体现了辛弃疾词作的雄浑豪放风格和深沉的爱国情怀。
## ✍️ 语言风格
《永遇乐·京口北固亭怀古》是辛弃疾词作的代表作，其语言风格鲜明，体现了辛弃疾词作的典型特点。

在用词方面，这首词具有凝练有力、气势恢宏的特点。开篇"千古江山"四字即气势磅礴，概括了历史的悠久与江山的壮丽。词中大量使用历史典故，如"孙仲谋"（孙权）、"寄奴"（刘裕）、"元嘉"（宋文帝年号）、"封狼居胥"（霍去病典故）、"佛狸"（北魏太武帝拓跋焘）和"廉颇"等，通过借古喻今的方式表达情感和观点。这些典故的运用不仅丰富了词的内涵，也体现了作者深厚的学养。"金戈铁马"、"气吞万里如虎"等词语形象生动，描绘出战争的壮阔场面；"仓皇北顾"、"烽火扬州路"则简洁有力地勾勒出战败时的狼狈和战火纷飞的情景。

在修辞手法上，这首词运用了多种技巧。借景抒情是其中突出的手法，如"斜阳草树，寻常巷陌"描绘眼前景象，引出对历史的追忆；"佛狸祠下，一片神鸦社鼓"则通过描绘北魏太武帝祠堂前的景象，暗讽南宋朝廷的苟安。比喻手法运用精妙，如"气吞万里如虎"将刘裕北伐的气势比作猛虎，生动形象。词中还运用了设问手法，"凭谁问：廉颇老矣，尚能饭否？"以设问方式表达感慨，增强情感表达。对比手法贯穿全词，上阕中孙权、刘裕的英雄事迹与下阕中刘义隆的失败形成对比；历史上的英雄与当今朝廷的苟安形成对比；自己的报国之志与朝廷的不重用形成对比，这些对比增强了词作的艺术感染力。此外，"雨打风吹去"象征历史的无情和英雄事迹的消逝，"烽火扬州路"象征战争的残酷和国家面临的危机，体现了象征手法的运用。

整体而言，这首词的语言风格雄浑豪放，气势恢宏，如"千古江山"、"金戈铁马"等句体现了辛弃疾词作的豪放风格；同时又沉郁悲愤，"英雄无觅"、"风流总被雨打风吹去"等句表达了作者对国家命运的忧虑和个人遭遇的悲愤。词作用词简练而意蕴丰富，情感真挚深沉，尤其是结尾"廉颇老矣，尚能饭否"，表达了作者虽年老却仍渴望报效国家的赤诚之心，感人至深。
## 📚 文化札记
《永遇乐·京口北固亭怀古》是南宋著名词人辛弃疾的代表作之一，写于开禧元年（1205年），当时作者已66岁，任镇江知府。这首词通过怀古伤今，表达了作者对国家命运的忧虑和自己报国无门的悲愤。以下是词中重要的文化背景注释：

"千古江山，英雄无觅孙仲谋处"中的孙仲谋指三国时期吴国开国皇帝孙权，字仲谋。孙权曾建都京口（今江苏镇江），是江南地区的杰出英雄。作者感叹江山依旧，但像孙权这样的英雄人物已无处可寻，暗指南宋朝廷缺乏能抵抗金兵入侵的英雄人物。

"斜阳草树，寻常巷陌，人道寄奴曾住"中的寄奴是南朝宋武帝刘裕的小名。刘裕出身贫寒，曾在京口居住，后来成为南朝宋的开国皇帝，并多次北伐，收复失地。作者通过提及刘裕，表达了对英雄人物的怀念和对现实的失望。

"想当年，金戈铁马，气吞万里如虎"描述的是刘裕北伐时的雄壮气势。"金戈铁马"形容军队装备精良，气势雄壮；"气吞万里如虎"形容刘裕北伐时的威猛气势。

"元嘉草草，封狼居胥，赢得仓皇北顾"中的元嘉是南朝宋文帝刘义隆的年号（424-453年）；"封狼居胥"指汉代霍去病曾率军追击匈奴至狼居胥山（在今蒙古国境内），封禅祭天，表示武功显赫；"仓皇北顾"指南朝宋文帝北伐失败，仓皇南逃的情景。元嘉年间，宋文帝多次北伐，但都失败了，特别是元嘉二十七年（450年）的北伐，遭到北魏太武帝拓跋焘的反击，几乎导致南朝灭亡。作者借古讽今，警告南宋朝廷不要轻率北伐。

"四十三年，望中犹记，烽火扬州路"中的四十三年指辛弃疾从金国占领区南归到写这首词时的年数。辛弃疾于1162年南归，这首词写于1205年，相隔43年。"烽火扬州路"指辛弃疾年轻时在金国占领区参加抗金义军，以及后来南归时经过扬州等地看到的战火景象。

"可堪回首，佛狸祠下，一片神鸦社鼓"中的佛狸是北魏太武帝拓跋焘的小名。他曾在元嘉二十七年（450年）率军南下，攻打南朝宋，一度打到长江北岸。"佛狸祠"是为纪念拓跋焘而建的祠庙，这里指金人占领区内的祠庙。"神鸦社鼓"指祭祀时的乌鸦和社日的鼓声，表示金人占领区内的祭祀活动，暗示金人对汉地的统治已经根深蒂固。

"凭谁问：廉颇老矣，尚能饭否"中的廉颇是战国时期赵国的名将。据《史记·廉颇蔺相如列传》记载，廉颇晚年遭谗言逃到魏国，赵王想重新起用他，派使者去看望。廉颇为了表示自己还能打仗，在使者面前一顿吃了一斗米、十斤肉，并披甲上马。但使者被廉颇的仇人收买，回去报告说廉颇虽然能吃饭，但一会儿就拉了三次屎。赵王以为廉颇真的老了，就没有再起用他。这里辛弃疾以廉颇自比，表示自己虽然年老，但仍有报国之志，只是无人赏识。

这首词通过丰富的历史典故和人物，表达了作者对国家命运的忧虑和自己报国无门的悲愤，是辛弃疾爱国词作的代表作之一。
## 🌅 创作背景
《永遇乐·京口北固亭怀古》创作于南宋开禧三年（1207年），地点在京口（今江苏镇江）的北固亭。当时辛弃疾已届六十七岁高龄，正值人生暮年。这一年，他被朝廷重新起用，出任镇江知府，但身体状况已经大不如前，且对国家前途充满忧虑。

南宋时期，北方大片国土已被金国占领，朝廷偏安江南，政治环境复杂。主战派与主和派斗争激烈，辛弃疾作为坚定的主战派，一生主张抗金收复失地，却屡遭排挤，仕途坎坷。开禧年间，权臣韩侂胄主持北伐，辛弃疾虽被重新起用，但对这次准备不足的北伐并不乐观，认为难以成功。后来的历史发展也印证了他的担忧，这次北伐最终以失败告终。

辛弃疾登上京口北固亭，俯瞰长江，怀古伤今。他想到了历史上的英雄人物，如东吴的孙权（字仲谋）和南朝宋的刘裕（小名寄奴），他们曾在这片土地上建功立业。同时，他也想到了历史上北伐失败的教训，如宋文帝元嘉年间的草率北伐。这些历史人物的兴衰成败，以及南宋当时的困境，激发了他的创作灵感。词中"凭谁问：廉颇老矣，尚能饭否"一句，既是对自己年老体衰、壮志未酬的感慨，也是对国家前途的深切忧虑，体现了他忧国忧民的赤子之心。
## 💭 情感基调
《永遇乐·京口北固亭怀古》的情感基调可概括为悲壮激愤、苍凉无奈。词人通过对历史英雄的追忆与现实的对比，流露出强烈的爱国情怀和壮志难酬的苦闷。氛围特点表现为雄浑深沉、沧桑感慨。上阕以"千古江山"起笔，追慕孙权、刘裕等英雄，气势恢宏；下阕转入对国事的忧虑，"四十三年"的回忆与"佛狸祠下"的现实形成强烈反差，营造出沉重压抑的氛围。结尾以廉颇自问，更显悲壮不甘。整首词在怀古中伤今，在追忆中抒怀，情感跌宕起伏，既有豪迈之气，又有悲凉之音，体现了辛弃疾词作特有的沉郁顿挫风格。
## 🔍 赏析
《永遇乐·京口北固亭怀古》是辛弃疾豪放词的代表作，通过登临怀古，抒发深沉爱国情怀。在意境营造上，词人以"千古江山"开篇，气势恢宏，随即以"英雄无觅孙仲谋处"点明主题，营造出苍凉悲壮的氛围。词中巧妙运用对比手法，如"舞榭歌台，风流总被雨打风吹去"与"想当年，金戈铁马，气吞万里如虎"形成鲜明对照，突显历史沧桑与英雄气概。"斜阳草树，寻常巷陌"的萧瑟景象与"人道寄奴曾住"的历史记忆交织，增强了时空交错感。下片通过"佛狸祠下，一片神鸦社鼓"的凄凉画面，暗示国土沦丧之痛，意境深远。

语言特色上，此词雄浑豪放，气势磅礴，多用四字短语如"金戈铁马"、"气吞万里"等，节奏铿锵有力。修辞技巧丰富，大量运用典故如孙仲谋、寄奴、元嘉、佛狸、廉颇等，不仅增强历史厚重感，也寄托了词人的政治抱负。词中借"雨打风吹"比喻时光流逝，以"神鸦社鼓"象征国土沦丧后的凄凉，形象生动。结构上，上片写景怀古，下片抒情议论，层次分明，结尾"凭谁问：廉颇老矣，尚能饭否"以典故设问，含蓄有力，余味无穷。

情感表达方面，词人通过对历史英雄的缅怀，表达了对英雄时代的向往和对现实的不满。"元嘉草草，封狼居胥，赢得仓皇北顾"借古讽今，批评南宋朝廷抗金不力。"四十三年，望中犹记，烽火扬州路"流露出对国土沦丧的痛苦记忆。词中既有对民族危亡的忧虑，也有个人报国无门的悲愤，更有"廉颇老矣，尚能饭否"所体现的虽老犹壮的爱国情怀。

文学价值上，此词将怀古、抒情、议论融为一体，艺术成就卓著，充分展现了辛弃疾豪放词风的独特魅力。它不仅反映了南宋时期的社会现实和知识分子的心态，也表达了中华民族的爱国情怀和民族精神，成为后世爱国文学的经典范本。词中所体现的将个人命运与国家兴亡紧密相连的创作思想，对后世文学产生了深远影响，在中国文学史上占有重要地位。
