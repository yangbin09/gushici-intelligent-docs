## 📜 原文
```
白发三千丈，缘愁似个长。
不知明镜里，何处得秋霜。
```
## 🎯 主题
《秋浦歌》主要表达了诗人因忧愁而早生白发的感慨，抒发了对时光流逝、青春不再的忧伤。李白以"白发三千丈"的夸张手法，生动展现了内心愁绪之深重，而"秋霜"的比喻则暗示了岁月无情、生命短暂的思考。诗人在镜前困惑于自己早生的白发，实则是对人生际遇和命运无常的深刻感悟，体现了李白诗歌中常见的豪放与忧思并存的复杂情感。
## 🏗️ 结构
《秋浦歌》是李白创作的一首五言绝句，虽然仅有四句，但结构精巧，章法严谨，体现了李白诗歌的艺术特色。

从章法布局看，此诗遵循了传统绝句"起承转合"的结构模式。首句"白发三千丈"为"起"，开门见山，以夸张手法提出白发之长的形象，震撼人心。次句"缘愁似个长"为"承"，承接上句解释白发长的原因是因为愁绪，形成明确的因果关系。第三句"不知明镜里"为"转"，转向对镜自照的场景，引入新的视角和思考。末句"何处得秋霜"为"合"，以"秋霜"比喻白发，点明主题，完成全诗的收束。

在层次安排上，全诗可分为两个层次。第一层次是前两句，描写外在形象（白发）及其原因（愁绪），直接表达诗人因愁而早衰的状态。第二层次是后两句，转向内心感受，通过镜中形象引发对白发来源的思考，由表及里，由实到虚，层层深入。

此诗的结构特点还体现在：首先，简洁凝练，四句诗包含了形象描写、原因解释、场景转换和深刻思考，结构紧凑而内涵丰富。其次，前后呼应，首尾相扣，从"白发"到"秋霜"，意象转换自然，形成完整的艺术整体。再次，因果关系明确，前两句形成因果，后两句形成问答，结构严谨。最后，由外而内，从外在的白发描写，逐渐深入到内心的愁绪感受，层次分明。

通过这种精巧的结构安排，李白在短短二十字中，既表达了自己深沉的愁绪和对生命短暂的感慨，又展现了丰富的想象力和深刻的艺术感染力，体现了其诗歌"言简意丰"的艺术特色。
## ✍️ 语言风格
李白《秋浦歌》一诗虽然仅有短短二十字，却充分展现了李白诗歌独特的语言风格特点。这首诗在用词和修辞手法上都有鲜明的个性特征，体现了李白的浪漫主义色彩和豪放风格。

在用词方面，李白善于运用夸张的数字表达。"白发三千丈"中的"三千丈"并非实指，而是一种极言其多的夸张表达，这种数字夸张在李白诗歌中屡见不鲜，如"飞流直下三千尺"、"桃花潭水深千尺"等，展现了诗人丰富的想象力和豪迈不羁的个性。同时，诗中"白发"、"秋霜"等词语形象生动，"白发"直接点明诗人因忧愁而早衰的形象，"秋霜"则巧妙地比喻白发，暗示岁月无情和生命短暂。这些词语选择精准，既富有画面感，又饱含情感色彩。特别是"愁"字，作为全诗的情感核心，直接表达了诗人内心的忧愁，而"不知"则带有迷茫困惑的情感色彩，表现了诗人对自己早衰的无奈和困惑。

在修辞手法方面，这首诗运用了多种技巧。首先是夸张手法，"白发三千丈"是典型的夸张表达，通过这种不合常理的夸张，生动表现了诗人因忧愁而头发变白，以及忧愁之深重。其次是比喻，"缘愁似个长"中的"似"字表明这是一个比喻，将抽象的忧愁具象化，使读者能直观感受诗人忧愁的深重；"何处得秋霜"中的"秋霜"也是比喻，用来比喻白发，秋霜洁白寒冷，象征着岁月无情和生命凋零，与白发意象相契合。再次是设问，"不知明镜里，何处得秋霜"是一个设问句，通过设问表达对镜中白发的惊讶和困惑，增强了诗歌的节奏感和表现力，深化了情感表达。此外，诗中还运用了借景抒情的手法，"明镜"和"秋霜"作为景物，诗人通过它们抒发了自己因忧愁而早衰的感慨，使诗歌意境更加深远。

在句式结构上，这首五言绝句结构紧凑，节奏鲜明。首句"白发三千丈"直接点题，开门见山；次句"缘愁似个长"解释原因，承接自然；第三句"不知明镜里"转折设问，引人深思；末句"何处得秋霜"以问作结，余味无穷。这种结构安排使诗歌层次分明，情感递进，艺术效果强烈。

总体而言，李白《秋浦歌》的语言风格特点是夸张大胆、想象奇特，直抒胸臆、情感真挚，语言简洁凝练而意境深远。多种修辞手法的灵活运用，使诗歌表达生动形象，艺术感染力强。这首诗充分体现了李白诗歌豪放飘逸、想象奇特、语言生动、情感真挚的艺术风格，是中国古典诗歌中的杰作。
## 📚 文化札记
《秋浦歌》是李白创作的组诗之一，共十七首，这是其中的第十五首。秋浦是唐代池州的一个县名，位于今天的安徽省贵池县西南，是一个以产铜、银等矿产闻名的地方，同时风景秀丽。李白曾多次游历秋浦，并在此创作了大量诗歌。

"白发三千丈"运用了夸张的修辞手法，"三千丈"并非实际长度，而是形容忧愁之深重导致白发极长。这种夸张手法在唐诗中很常见，如李白的"飞流直下三千尺"等。在中国传统文化中，白发常用来象征年老、忧愁或悲伤。

"缘愁似个长"中的"缘"意为因为，"愁"指忧愁，"似个长"意为像这样长。这句直接点明了白发如此之长的原因。

"明镜"指古代的铜镜，在中国古代文化中，镜子不仅是日常用品，也常被用来象征自省、反思。诗人照镜见白发，引发对生命、时光的思考，这是中国古典诗歌中常见的主题。

"秋霜"比喻白发，如同秋天的霜一样洁白。这一比喻形象生动，富有诗意，是中国古典诗歌中常见的意象。秋霜洁白、寒冷，象征着岁月的无情和生命的衰老。

这首诗创作于李白晚年时期，大约在他五十多岁的时候。当时李白经历了政治上的挫折，被排挤出朝廷，漂泊各地。他曾短暂受到唐玄宗赏识，但很快被排挤出宫，晚年又因安史之乱中站错队伍而被流放。诗中表达的忧愁可能与他的政治失意、人生感慨有关。

这首诗语言简洁，意境深远，通过夸张的修辞和生动的比喻，表达了诗人对时光流逝、生命衰老的感慨，体现了李白诗歌豪放中见深沉的艺术风格，也蕴含着对生命、时光的哲理思考。
## 🌅 创作背景
《秋浦歌》创作于唐玄宗天宝十二年（753年）左右，地点在今安徽省池州市贵池区一带的秋浦地区。当时李白已年过五旬，正处于人生的后半阶段。这首诗创作于他离开长安后的漫游时期，此时他虽已名满天下，但政治理想却屡屡受挫。李白曾短暂受到唐玄宗赏识，入长安任翰林供奉，却因性格傲岸不羁，不善逢迎，加之权贵排挤，最终被迫离开朝廷，开始了长期的漫游生活。

在创作这首诗时，李白内心充满忧愁与失落。诗中"白发三千丈，缘愁似个长"正是他内心愁绪的真实写照，表明他因忧国忧民、怀才不遇而愁绪满怀，以至于白发丛生。而"不知明镜里，何处得秋霜"则反映了他对生命流逝、年华老去的感慨与无奈。

从历史背景看，这首诗创作于唐朝表面繁荣实则危机四伏的天宝年间。唐玄宗后期沉迷声色，朝政被杨国忠等人把持，政治腐败，社会矛盾日益尖锐。安史之乱即将爆发，唐朝由盛转衰的转折点即将到来。李白作为一位有政治抱负的文人，对这种社会状况有着敏锐的感知，内心充满忧虑。

李白创作《秋浦歌》的灵感主要来源于秋浦地区的自然风光和他对人生的深刻思考。秋浦山水秀丽，引发了他对生命、时间和衰老的哲学思考。同时，他对自己的人生际遇感到忧愁和无奈，政治抱负无法实现，只能寄情山水，借诗抒怀。这首诗既是对个人命运的感叹，也是对国家前途的忧虑，体现了李白作为诗人的社会责任感和历史使命感。
## 💭 情感基调
《秋浦歌》的情感基调可概括为：愁绪深沉、感伤无奈。诗人以"白发三千丈"的夸张手法，将内心的忧愁具象化，表达出因愁绪而早生华发的深沉情感。"不知明镜里，何处得秋霜"一句更流露出对时光流逝、年华老去的无奈与惊讶。

氛围特点可概括为：凄凉萧瑟、孤寂沉重。诗中"秋霜"的意象营造出凄凉的氛围，而"明镜"前的独自沉思又增添了孤寂感。整首诗通过白发与秋霜的意象，形成了一种沉重而萧瑟的氛围，使读者能感受到诗人面对岁月无情时的内心凄凉。
## 🔍 赏析
这首《秋浦歌》是李白五言绝句中的经典之作，短短二十字却意蕴深远。在意境营造上，诗人以"白发三千丈"的夸张意象开篇，立即营造出一种深沉忧伤的氛围，将抽象的愁绪具象化为可测量的长度。后两句通过"明镜"这一媒介，形成内外对照的艺术效果，使读者仿佛看到诗人对镜自照、感叹岁月的情景。"秋霜"一词既实指白发如霜，又暗含秋天的萧瑟之感，与"秋浦"地名相呼应，构建了时空统一的意境。

语言特色方面，此诗简练而富有表现力，运用了多种修辞技巧。夸张手法最为突出，"白发三千丈"将愁绪的深度和广度以视觉化方式呈现；比喻修辞将愁绪比作可测量的长度；设问修辞"不知明镜里，何处得秋霜"以自问方式表达困惑，增强了抒情效果。"秋霜"的双关含义体现了李白语言的精妙和丰富。

情感表达上，这首诗深刻展现了诗人因忧愁而早衰的状态，以及对时光流逝的感慨。诗人对镜自照，发现已白发苍苍却不知变化何时发生，这种对时光流逝的无意识感受，强化了诗歌的悲剧色彩。主题上探讨了愁绪与生命、时间的关系，折射出李白在政治失意、人生坎坷中的心理状态。

文学价值方面，这首诗体现了李白"诗仙"的艺术风格，是盛唐诗歌艺术的典范。"白发三千丈"这一夸张意象成为后世文学中表达愁绪的经典手法，影响深远。作为短小精悍却意蕴丰富的作品，它是中国古典诗歌中以简驭繁的典范，对后世诗歌创作产生了重要影响，也成为中国文化中表达愁绪的经典文本。
