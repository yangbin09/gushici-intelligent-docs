## 📜 原文
```
风急天高猿啸哀，渚清沙白鸟飞回。
无边落木萧萧下，不尽长江滚滚来。
万里悲秋常作客，百年多病独登台。
艰难苦恨繁霜鬓，潦倒新停浊酒杯。
```
## 🎯 主题
《登高》是杜甫晚年代表作，主要表达了诗人深沉的悲秋情怀和生命感慨。通过描绘秋日登高所见的萧瑟景象——风急天高、猿啸鸟飞、落木萧萧、长江滚滚，诗人抒发了自己漂泊异乡、年老多病的孤独与无奈。诗中"万里悲秋常作客，百年多病独登台"直接点明了诗人客居他乡、体弱多病的凄凉处境；而"艰难苦恨繁霜鬓，潦倒新停浊酒杯"则进一步表现了诗人对生活的困顿与对命运的感慨。整首诗既有对自然景象的细腻描绘，又有对人生哲理的深刻思考，体现了杜甫忧国忧民、感时伤世的博大情怀，以及面对生命无常的深沉思考。这首诗以高度凝练的语言和深沉的情感，展现了诗人晚年复杂而丰富的内心世界。
## 🏗️ 结构
《登高》是杜甫七言律诗的代表作，其结构特点体现了诗人高超的艺术技巧和深厚的思想内涵。从章法布局和层次安排来看，这首诗具有以下特点：

首先，从章法布局上看，全诗严格遵循律诗"起承转合"的结构模式。首联"风急天高猿啸哀，渚清沙白鸟飞回"为"起"，以自然景象起兴，点明登高所见所闻，营造萧瑟凄清的氛围。颔联"无边落木萧萧下，不尽长江滚滚来"为"承"，承接首联进一步描绘秋景，视野由近及远，气势恢宏。颈联"万里悲秋常作客，百年多病独登台"为"转"，由景入情，转向诗人自身境遇。尾联"艰难苦恨繁霜鬓，潦倒新停浊酒杯"为"合"，总结全诗，表达诗人内心的悲苦与无奈。这种起承转合的结构安排，使得全诗层次分明，逻辑严密。

其次，从层次安排上看，全诗可分为三个层次：第一层次（首联、颔联）为写景，描绘登高所见的秋景，构成一幅动态的秋景图；第二层次（颈联）为抒情，由景入情，表达诗人身处异乡、悲秋伤怀、年老多病的凄凉处境；第三层次（尾联）为言志，进一步深化情感，表达诗人的人生感悟。这种由景到情、由外到内的层次安排，使得全诗内容丰富，意蕴深远。

在艺术技巧方面，这首诗的对仗极为工整。颔联"无边落木萧萧下"对"不尽长江滚滚来"，颈联"万里悲秋常作客"对"百年多病独登台"，无论是词性、结构还是意义都严格对应，体现了杜甫精湛的诗歌技巧，增强了诗歌的节奏感和音乐美。

从意象安排上看，全诗自然意象（风、天、猿、渚、沙、鸟、落木、长江）与人文意象（悲秋、作客、多病、登台、艰难、苦恨、霜鬓、潦倒、浊酒杯）相结合，构成了一幅情景交融的艺术画面，既有宏大的自然背景，又有深刻的人文内涵。

情感表达上，全诗呈现出明显的递进关系：从首联的含蓄，到颔联的深化，再到颈联的直接抒发，最后到尾联的情感高潮，层层深入，逐步推向高潮，给读者留下深刻印象。

空间转换上，全诗由近处的景象（风、天、猿等）到远处的景象（无边落木、不尽长江），再转向内心感受，最后深入到人生感悟，这种由外到内、由远及近的空间转换，使得全诗的空间层次丰富多变。

时间维度上，全诗既有当下的秋景描绘，也有长期漂泊的感慨，还有对岁月流逝的叹息，这种时间维度的变化，增强了诗歌的思想深度。

综上所述，《登高》在结构上章法严谨，层次分明，对仗工整，意象丰富，情感递进，空间转换自然，时间维度多样，体现了杜甫高超的艺术技巧和深刻的思想内涵，是中国古典诗歌中的经典之作。
## ✍️ 语言风格
杜甫的《登高》是其晚年代表作，充分体现了"沉郁顿挫"的艺术风格。从语言风格特点来看，主要表现在以下几个方面：

在用词方面，杜甫极为精炼凝练，每个字都经过精心选择。如"风急天高"四字简洁勾勒出秋日高远天空和急劲秋风，营造出开阔萧瑟的氛围。诗中色彩词运用鲜明，"渚清沙白"、"繁霜鬓"等冷色调词语与悲凉情感基调相呼应。同时，动态词语如"急"、"啸"、"飞回"、"萧萧下"、"滚滚来"等使画面充满动感，增强了诗歌的生动性。对比手法也贯穿全诗，"无边"与"不尽"、"萧萧"与"滚滚"、"万里"与"百年"等形成强烈对比，增强了艺术张力。

修辞手法上，这首七言律诗中间两联对仗工整。颔联"无边落木萧萧下，不尽长江滚滚来"中，"无边"对"不尽"，"落木"对"长江"，"萧萧下"对"滚滚来"；颈联"万里悲秋常作客，百年多病独登台"中，"万里"对"百年"，"悲秋"对"多病"，"常作客"对"独登台"。叠词"萧萧"和"滚滚"形象描绘落叶飘零和江水奔流，增强了音乐性和形象感。"猿啸哀"中的"啸"字既是拟声又传达悲凉情感；"繁霜鬓"暗喻白发如霜，形象表现衰老；"浊酒杯"借代饮酒习惯，暗示诗人因年老多病而不得不放弃的慰藉。

句式结构上，全诗遵循七言律诗格式，起承转合清晰。首联描写登高所见秋景；颔联继续描写自然景象，视野更为开阔；颈联由景入情，转向对自身遭遇的感慨；尾联总结全诗，抒发深沉悲凉之情。诗中前后呼应，如首联的"哀"与尾联的"苦恨"形成情感呼应；"风急天高"与"无边落木"形成空间呼应。

意象运用丰富多样，包括"风"、"天"、"猿"、"渚"、"沙"、"鸟"、"落木"、"长江"等自然意象，共同构成秋日登高的壮阔画面。"悲秋"作为传统意象，充分表达诗人凄凉心境。"万里"和"百年"从空间和时间两个维度展开，增强诗歌的时空感和哲理性。"繁霜鬓"和"浊酒杯"作为生命意象，前者象征衰老，后者象征生活的慰藉与无奈。

情感表达方式上，诗人通过描写秋景抒发内心悲凉，实现情景交融。颈联和尾联中直接抒发情感，如"悲秋"、"多病"、"艰难苦恨"、"潦倒"等，表达深沉悲凉之情。整体上仍保持含蓄风格，通过意象和场景描绘间接传达情感，呈现沉郁顿挫的特点。

这首诗充分体现了杜甫"语不惊人死不休"的创作态度，通过精炼的语言、丰富的修辞、严谨的结构、多样的意象和深沉的情感，展现了诗人对国家命运和个人遭遇的深刻思考，是中国古典诗歌中的经典之作。
## 📚 文化札记
《登高》是杜甫晚年的代表作，创作于唐代宗大历二年（767年），时年56岁的杜甫身处夔州（今重庆奉节）。这首诗被誉为"七律第一"，蕴含丰富的文化背景。

"风急天高猿啸哀"中的"猿啸哀"与夔州地理环境密切相关。夔州位于长江三峡地区，当地多猿，其叫声凄厉。三峡猿啼在古代文学中常作为悲凉意象，如郦道元《水经注》中记载："巴东三峡猿鸣悲，夜鸣三声泪沾裳。"这一自然现象被文人赋予了浓厚的情感色彩。

"渚清沙白鸟飞回"描绘了夔州长江边的秋景。"渚"指水中小块陆地，夔州地处长江边，江中多沙洲。而"鸟飞回"则暗示了秋季候鸟南迁的自然现象，也暗合诗人漂泊异乡的处境。

"无边落木萧萧下，不尽长江滚滚来"中的"落木"与"长江"形成鲜明对比。落叶萧萧象征凋零与衰败，而长江滚滚则代表永恒与不息。长江作为中国第一大河，在传统文化中常被赋予时间流逝、历史变迁的象征意义，如苏轼"大江东去，浪淘尽，千古风流人物"。

"万里悲秋常作客"中的"悲秋"源自中国古代文人的悲秋传统，可追溯至宋玉《九辩》："悲哉，秋之为气也。"秋季万物凋零，容易引发文人对生命短暂的感慨。"万里作客"则指杜甫长期漂泊异乡，远离故土河南巩县。

"百年多病独登台"中的"登台"借用古代文人登高抒怀的传统。重阳节（农历九月初九）有登高避灾的习俗，文人登高则多为抒发情怀、远望思乡。杜甫此处"独登台"既指实际登高行为，也象征其孤独处境。

"艰难苦恨繁霜鬓"中的"繁霜鬓"指鬓发斑白如霜，象征年老和忧愁。杜甫经历了安史之乱（755-763年），目睹国家动荡，个人生活颠沛流离，这些经历加速了他的衰老。

"潦倒新停浊酒杯"中的"停酒"反映杜甫晚年生活状况。他生前喜爱饮酒，与李白、陶渊明等文人一样，将酒作为情感寄托。因病停酒不仅说明其身体状况极差，也暗示生活无奈和精神打击。

整体而言，这首诗创作于安史之乱后的动荡时期，杜甫借秋日登高所见所感，抒发了个人老病潦倒之悲与忧国忧民之情，体现了杜甫"沉郁顿挫"的诗风，也反映了那个时代的苦难与动荡。
## 🌅 创作背景
《登高》创作于唐代宗大历二年（公元767年）重阳节，地点在夔州（今重庆市奉节县）。此时杜甫已年届五十六岁，寓居夔州近两年。这一时期，杜甫的生活极为困顿，依靠友人接济度日，经济拮据。同时，他饱受疾病折磨，患有肺病、风痹（风湿病）等多种疾病，视力也严重衰退，身体状况每况愈下。他携家带口，漂泊异乡，远离故土，内心充满孤独与思乡之情。

从历史背景看，安史之乱（755-763年）后，唐朝国力由盛转衰，中央政权衰弱，藩镇割据，宦官专权，政治腐败，社会动荡不安。战乱虽已结束，但其影响深远，民生凋敝，赋税繁重，百姓生活困苦。杜甫作为一位忧国忧民的诗人，亲历了唐朝由盛转衰的历史变迁，目睹了战乱带来的苦难，内心充满忧虑和悲愤。

重阳节是中国传统节日，有登高望远的习俗。杜甫在这一天登高望远，触景生情。眼前的秋景——风急天高、猿啸、落叶、长江等，引发了他对个人命运和国家前途的深沉思考。他将自己的人生经历、对国家的忧虑以及对自然的感悟融为一体，创作了这首千古绝唱。诗中既有对自然景色的描绘，也有对个人命运的感慨，更有对国家前途的忧虑，体现了杜甫"沉郁顿挫"的诗风和"诗史"的特点。
## 💭 情感基调
《登高》的情感基调以悲凉、孤寂为核心，融入无奈与沉郁。诗中"悲秋"直抒胸臆，"猿啸哀"奠定全诗哀伤基调，"独登台"凸显孤独之感，"艰难苦恨"表达对命运的无奈，"繁霜鬓"和"潦倒"则流露出沉郁心境。

氛围特点呈现萧瑟、壮阔、凄清与苍凉。风急天高、落木萧萧营造秋日萧瑟；无边落木、不尽长江展现壮阔气势；渚清沙白、鸟飞回增添凄清意境；而诗人年老多病、漂泊他乡的处境，又使全诗笼罩在苍凉氛围中。

这首诗通过自然景象与个人境遇的交织，形成一种壮阔中的悲凉，永恒中的凄清，展现了杜甫晚年面对人生困境时的复杂情感世界。
## 🔍 赏析
《登高》是杜甫晚年创作的代表作，通过登高远望的视角，营造出悲壮苍凉的意境。首联以"风急天高猿啸哀，渚清沙白鸟飞回"勾勒秋日江景，既有视觉开阔又有听觉凄凉；颔联"无边落木萧萧下，不尽长江滚滚来"以对仗和叠词手法，展现自然永恒与人生短暂的对比，形成宏大时空感；颈联则转向个人境遇，表达漂泊异乡、年老多病的孤独；尾联以"繁霜鬓"和"浊酒杯"点明诗人年老体衰、生活困顿的现实。全诗采用由远及近、由景入情的手法，层次分明，意境深远。

语言上，诗中大量运用对仗如"风急"对"天高"，"无边"对"不尽"等，使结构严谨，节奏鲜明。叠词"萧萧"、"滚滚"增强音韵美和形象感。对比手法如自然永恒与人生短暂、壮阔景色与个人渺小处境的对比，强化感染力。借景抒情、情景交融的运用，使诗歌既有客观描绘，又有主观抒发，达到情景合一的艺术效果。

情感表达上，诗人抒发了深沉的悲秋之情和人生感慨。"悲秋"不仅是对季节的伤感，更是对个人命运和国家前途的思考。诗中透露出忧患意识和悲天悯人情怀，体现杜甫"沉郁顿挫"的诗风特点，同时也蕴含对生命意义的思考和对人生价值的追求，反映了诗人面对困境时的坚韧品格。

《登高》作为中国古典诗歌经典，在艺术上达到情景交融、形神兼备的境界，展现了杜甫独特的诗歌风格。思想内容上，它不仅表达个人情感体验，更折射出动荡时代的社会现实和知识分子的精神困境，具有深刻的历史意义和人文价值。这首诗成为"登高望远"主题的典范，为后世诗人提供艺术表现范例，其忧国忧民情怀和坚韧不拔的精神也成为中华民族精神文化的重要组成部分。
