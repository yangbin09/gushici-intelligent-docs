## 📜 原文
```
风急天高猿啸哀，渚清沙白鸟飞回。
无边落木萧萧下，不尽长江滚滚来。
万里悲秋常作客，百年多病独登台。
艰难苦恨繁霜鬓，潦倒新停浊酒杯。
```
## 🎯 主题
杜甫的《登高》通过描绘秋天萧瑟壮阔的景色，抒发了诗人漂泊异乡、年老多病的悲苦之情。诗中"无边落木萧萧下，不尽长江滚滚来"展现了生命短暂与宇宙永恒的对比，体现了诗人对人生的哲学思考。同时，"万里悲秋常作客，百年多病独登台"表达了诗人长期漂泊、孤独无依的处境，而"艰难苦恨繁霜鬓，潦倒新停浊酒杯"则反映了诗人历经困苦、贫病交加的凄凉晚景。整首诗不仅表达了个人悲情，也蕴含着对国家命运的忧虑，体现了杜甫深沉的忧国忧民情怀和崇高的人格境界。
## 🏗️ 结构
《登高》是杜甫晚年创作的七言律诗代表作，其结构特点体现了严谨的章法布局和精妙的层次安排。

从章法布局看，此诗遵循律诗的典型结构，分为首联、颔联、颈联和尾联四部分，每联两句，共八句。全诗采用"先景后情"的布局方式，前两联集中写景，后两联侧重抒情，形成由外而内、由物及我的结构脉络。这种布局使诗歌在形式上整齐划一，在内容上层层递进，展现出律诗的严谨美。

从层次安排看，诗歌可分为四个层次：
第一层次为首联"风急天高猿啸哀，渚清沙白鸟飞回"，描绘登高所见的具体景物，通过"风急""天高""猿啸哀"与"渚清""沙白""鸟飞回"的对比，营造出凄凉萧瑟的秋日氛围，奠定全诗基调。

第二层次为颔联"无边落木萧萧下，不尽长江滚滚来"，视野由近及远，由具体到宏大。"无边"与"不尽"形成空间上的广阔感，"落木萧萧"与"长江滚滚"则形成动静对比，既表现了秋天的萧瑟，又暗示了时间的流逝与生命的无常。

第三层次为颈联"万里悲秋常作客，百年多病独登台"，由景入情，直接抒发诗人内心感受。"万里"与"百年"形成空间与时间的对比，"悲秋常作客"与"多病独登台"则表达了诗人漂泊无依、年老多病的凄凉境遇。

第四层次为尾联"艰难苦恨繁霜鬓，潦倒新停浊酒杯"，进一步深化情感，通过"繁霜鬓"和"浊酒杯"两个具体意象，表现诗人晚年生活的艰难与无奈，将全诗情感推向高潮。

在对仗方面，全诗四联均严格对仗，如"风急天高"对"渚清沙白"，"无边落木"对"不尽长江"，"万里悲秋"对"百年多病"，"艰难苦恨"对"潦倒新停"，体现了律诗的严谨性和艺术性。

此外，诗歌在结构上还体现了情景交融的特点，前两联的景物描写为后两联的情感表达做了铺垫，使景中有情，情中有景，形成完美的艺术整体。通过这种精心的结构安排，杜甫成功地将个人遭遇与时代背景、自然景观与内心情感融为一体，创作出这首千古传诵的佳作。
## ✍️ 语言风格
杜甫的《登高》是其晚年代表作，体现了沉郁顿挫、悲壮苍凉的语言风格。这首诗写于杜甫漂泊西南时期，语言凝练深刻，意境开阔悲凉，情感真挚深沉。

在用词方面，诗人选词精准而富有表现力。动词如"急"、"高"、"啸"生动描绘了自然景象的动态；"清"、"白"、"飞回"则勾勒出鲜明对比的画面。名词意象丰富，"风"、"天"、"猿"、"渚"、"沙"、"鸟"等勾勒出秋日景象，而"落木"、"长江"则象征时光流逝和历史长河，"霜鬓"、"浊酒杯"暗示诗人年老体衰的处境。程度副词"无边"、"不尽"、"万里"、"百年"强调了空间的广阔和时间的无限，增强了诗歌的气势。形容词如"哀"、"悲"、"艰难"、"苦恨"、"潦倒"等直接表达了诗人内心的情感。

修辞手法上，这首诗运用了多种技巧。对比手法贯穿全诗，如"风急天高"与"渚清沙白"的动静对比，"无边落木"与"不尽长江"的上下对比，"万里悲秋"与"百年多病"的时空对比。拟人手法如"猿啸哀"赋予自然物以人的情感，"落木萧萧下"则将落叶声音拟人化。叠字"萧萧"模拟落叶声，"滚滚"形容江水湍急，增强了音韵美和意境感。象征手法丰富，"落木"象征时光流逝，"长江"象征历史长河，"霜鬓"象征年老体衰。借景抒情的手法使前四句的景物描写与后四句的情感抒发相互映衬，形成情景交融的艺术效果。对偶手法在诗中尤为突出，四联均形成工整对偶，增强了诗歌的节奏感和艺术感染力。

句式结构上，前四句以写景为主，后四句以抒情为主，形成由景入情的层次。每句七字，句式整齐，节奏鲜明。首联描绘秋日凄凉景象，颔联表现时光流逝，颈联转入抒情表达漂泊心境，尾联深化情感表达生活无奈。

音韵格律方面，这首七言律诗押平声"灰"韵（哀、回、来、台、杯），韵脚整齐，音韵和谐。平仄符合律诗要求，声调起伏有致，读来抑扬顿挫。中间两联对仗工整，体现了律诗的艺术特色。

通过这些语言特点的运用，杜甫成功地营造了悲壮苍凉的艺术氛围，抒发了身世飘零、壮志难酬的悲愤之情，使《登高》成为唐代七言律诗中的经典之作。
## 📚 文化札记
《登高》是杜甫晚年漂泊西南时期的代表作，创作于唐代宗大历二年（767年）重阳节，时年56岁的诗人寓居夔州（今重庆奉节）。此诗被誉为"七律之冠"，体现了杜甫沉郁顿挫的艺术风格。

"风急天高猿啸哀"中的"猿啸哀"反映了夔州地区的地理特点。三峡地区自古多猿，其叫声凄厉，常被文人用来表达哀愁。北魏郦道元《水经注》中就有"三峡猿鸣，声泪俱下"的记载，这一意象也成为三峡文学的重要标志。

"渚清沙白鸟飞回"描绘了长江中的沙洲景象。"渚"指水中的小块陆地，夔州地处长江上游，江中多沙洲，秋季候鸟南迁，盘旋飞回的景象既是写实，也暗含了诗人思乡之情。

"无边落木萧萧下，不尽长江滚滚来"两句对仗工整，气势磅礴。"落木"指秋季落叶的树木，"萧萧"是象声词，形容落叶声。长江在夔州段水流湍急，气势磅礴，这一景象既展现了大自然的壮美，也暗含了时光流逝、生命短暂的哲理思考。

"万里悲秋常作客"中的"悲秋"是中国古典文学的重要主题，源于宋玉《九辩》："悲哉，秋之为气也，萧瑟兮草木摇落而变衰。"秋天万物凋零，容易引发人的悲愁情绪，尤其对于漂泊在外的游子。"作客"指漂泊在外，寄人篱下。安史之乱后，杜甫长期漂泊西南，远离家乡，心情悲凉。

"百年多病独登台"中，"百年"指一生，古人常用"百年"表示人的一生。"登台"指登高望远，与重阳节习俗相关。重阳节登高的习俗源于汉代《续齐谐记》中记载的费长房教桓景避灾的故事，后来逐渐演变为文人登高抒怀的传统。

"艰难苦恨繁霜鬓"中，"繁霜鬓"形容鬓发斑白如霜，表现诗人年老的形象。杜甫晚年生活困顿，仕途不顺，多病缠身，这些人生际遇都在诗中有所体现。

"潦倒新停浊酒杯"中，"潦倒"指失意、不得志，"新停浊酒杯"指因病戒酒。杜甫晚年多病，不得不停止饮酒，这一细节反映了诗人生活的艰难和身体的衰弱。

此诗创作于安史之乱（755-763年）后，唐朝国力衰退，社会动荡的背景下。杜甫亲历了唐朝由盛转衰的历史巨变，其诗作多反映社会现实和人民疾苦，被称为"诗史"。晚年漂泊西南，虽然生活困顿，但创作达到高峰，此时期的作品风格沉郁顿挫，艺术成就极高。

《登高》一诗前四句写景，后四句抒情，情景交融，语言凝练，意境开阔，格律严谨，对仗工整，展现了杜甫高超的艺术技巧和深厚的思想感情，被后人誉为"七律第一"。明代胡应麟《诗薮》称："杜七言律，通体完美者，'风急天高'第一。"清代沈德潜《唐诗别裁》评："气象雄盖宇宙，法律细入毫芒。"
## 🌅 创作背景
《登高》是杜甫于唐代宗大历二年（767年）秋在夔州（今重庆奉节）所作。当时杜甫已年届五十六岁，正值人生暮年。夔州地处长江三峡，山势险峻，风景壮阔，杜甫在此地担任检校工部员外郎的闲职，生活相对安定但内心依然忧思重重。

杜甫此时的生活状况十分艰难。他长期漂泊异乡，远离故土，身体多病，饱受风疾、肺病等疾患折磨。诗中"百年多病独登台"正是他当时身体状况的真实写照。作为一个经历了安史之乱的诗人，他目睹了唐朝由盛转衰的全过程，内心充满对国家命运的忧虑和对人民苦难的同情，这种忧国忧民的情怀使他的心境常常处于悲凉之中。

从历史背景看，此时的唐朝已经历了安史之乱（755-763年）的严重打击，虽然战乱已经平息，但国家元气大伤，藩镇割据，宦官专权，政治腐败，民生凋敝。社会动荡不安，百姓生活困苦。作为一个有强烈社会责任感的诗人，杜甫对这种状况深感痛心。

杜甫创作《登高》的动机和灵感来源主要有三：一是登高望远，面对夔州壮阔的自然景观，触景生情；二是晚年漂泊异乡，身体多病，对生命和时光流逝有着更为敏感的感受；三是对国家命运的忧虑和对人民苦难的同情。诗中"无边落木萧萧下，不尽长江滚滚来"既是对自然景观的描绘，也暗含了对历史变迁的深刻感慨，体现了诗人沉郁顿挫的风格和忧国忧民的情怀。
## 💭 情感基调
《登高》的情感基调主要表现为悲凉孤苦。诗中通过"猿啸哀"、"悲秋"等词语直接流露出哀伤之情，"独登台"表达了诗人深刻的孤独感，"百年多病"和"艰难苦恨"则展现了其内心的悲苦与苦闷，最后"潦倒新停浊酒杯"更透露出一种绝望情绪。

氛围特点则可概括为苍凉壮阔。首联"风急天高猿啸哀，渚清沙白鸟飞回"营造出凄清萧瑟的秋日景象；颔联"无边落木萧萧下，不尽长江滚滚来"以宏大的自然景象反衬人生的渺小与无常，形成苍凉之感；整首诗在壮阔的自然背景下，透露出诗人面对生命凋零、时光流逝的沉重感慨，呈现出一种悲壮的艺术氛围。
## 🔍 赏析
《登高》是杜甫晚年漂泊西南时期的代表作，创作于唐代宗大历二年（767年）秋，时年56岁，客居夔州。这首七言律诗以其深沉的情感和精湛的艺术技巧，成为中国古典诗歌中的不朽名篇。

在意境营造和艺术手法上，诗人通过登高远望，将眼前之景与心中之情完美结合，创造出宏大悲壮的艺术境界。首联"风急天高猿啸哀，渚清沙白鸟飞回"以简练笔触勾勒秋日景象，听觉与视觉相结合，使读者身临其境。颔联"无边落木萧萧下，不尽长江滚滚来"拓展空间感，"无边"与"不尽"将视野推向无限远，落木萧萧与长江滚滚形成动静对比，既表现秋天萧瑟，又暗示时光流逝和生命无常。诗人巧妙运用对比、映衬、象征等手法，如"万里"与"百年"的空间时间对比，增强了诗歌的艺术感染力。

在语言特色和修辞技巧方面，诗歌语言凝练而富有表现力，体现了杜甫"语不惊人死不休"的创作态度。诗人善于用精炼词语描绘生动画面，如"风急天高"四字勾勒秋日高远天空和凛冽秋风。全诗四联均为对仗工整的对偶句，结构严谨，音韵和谐。叠词和拟声词的运用，如"萧萧"形容落叶声，"滚滚"形容江水声，增强了诗歌的音乐性和表现力。"繁霜鬓"象征年老体衰，"浊酒杯"象征借酒消愁的无奈，这些修辞手法使诗歌内涵更加丰富。

情感表达和主题思想方面，诗歌表达了诗人深沉的悲秋之情和人生感慨。"万里悲秋常作客"直接点明长期漂泊异乡的处境，"百年多病独登台"中"百年"暗示人生短暂，"多病"表明身体状况不佳，"独登台"突出孤独感。"艰难苦恨繁霜鬓，潦倒新停浊酒杯"将悲凉情绪推向高潮，表达了诗人对个人命运和国家前途的忧虑，体现了杜甫深沉的忧国忧民情怀。

从文学价值和影响意义看，《登高》是杜甫"诗史"风格的重要体现，将个人情感与国家命运紧密结合，展现了"沉郁顿挫"的风格特点。特别是"无边落木萧萧下，不尽长江滚滚来"一联，被誉为"千古名句"。诗中表达的忧国忧民情怀和崇高人格精神，对后世文人产生了深远影响，成为中华民族精神文化的重要财富，也是中国古典诗歌艺术的高峰之作。
