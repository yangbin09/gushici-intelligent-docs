## 📜 原文
```
早岁那知世事艰，中原北望气如山。
楼船夜雪瓜洲渡，铁马秋风大散关。
塞上长城空自许，镜中衰鬓已先斑。
出师一表真名世，千载谁堪伯仲间。
```
## 🎯 主题
《书愤》主要表达了陆游强烈的爱国情怀和壮志难酬的悲愤。诗人回忆年轻时北望中原、收复失地的豪情壮志，以及亲身参与抗金战斗的经历，与现实中年华已逝却理想未成的处境形成鲜明对比。诗中"塞上长城空自许"一句，既体现了诗人自比国家栋梁的抱负，又流露出壮志未酬的无奈。最后引用诸葛亮《出师表》的典故，既是对先贤的敬仰，也抒发了自己报国无门的悲愤。整首诗充满了忧国忧民的情怀和对南宋朝廷偏安一隅、不思收复失地的不满，是陆游爱国思想的集中体现。
## 🏗️ 结构
陆游的《书愤》作为一首七言律诗，其结构特点体现了宋诗严谨的章法布局和精心的层次安排。

从章法布局看，此诗遵循了律诗"起承转合"的经典结构。首联"早岁那知世事艰，中原北望气如山"为"起"，点明主题，奠定全诗基调，回顾年轻时的豪情壮志。颔联"楼船夜雪瓜洲渡，铁马秋风大散关"为"承"，具体展开诗人曾经参与的抗金战斗场景，承接首联的豪情。颈联"塞上长城空自许，镜中衰鬓已先斑"为"转"，转向现实处境，表达壮志未酬的悲愤。尾联"出师一表真名世，千载谁堪伯仲间"为"合"，引用诸葛亮《出师表》典故，升华主题，收束全诗。

在层次安排上，此诗可分为三个层次。第一层次（首联）是诗人对年轻时期心态的回顾，表达了对国家统一的向往和收复中原的雄心。第二层次（颔联和颈联）形成鲜明对比，颔联描绘过去战斗的英勇场景，颈联则描述现实的衰老与无奈，通过时空对比强化了壮志未酬的悲愤。第三层次（尾联）通过引用历史典故，将个人情感上升到历史层面，表达了虽年老但志向不灭的精神。

在时空安排上，此诗也有独特之处。时间上，从"早岁"到"衰鬓已先斑"，再到"千载"，形成了从个人经历到历史长河的时间跨度。空间上，从"中原"到"瓜洲渡"、"大散关"，再到"塞上"，构建了广阔的地理背景，使诗歌意境更加开阔。

情感表达上，此诗层次分明，从年轻时的豪情壮志，到战斗中的英勇无畏，再到现实中的悲愤无奈，最后到对历史英雄的敬仰和自我激励，情感层层递进，最终达到高潮。

总的来说，《书愤》结构严谨，层次分明，章法布局符合律诗要求，通过时空转换和情感递进，深刻表达了陆游壮志未酬的悲愤和对国家的赤诚之心，体现了陆游诗歌"沉郁顿挫"的艺术特色。
## ✍️ 语言风格
陆游的《书愤》在语言风格上体现了精炼有力、意象鲜明、对仗工整、情感深沉等特点，通过丰富的修辞手法和精心的词语选择，表达了诗人对国家命运的忧虑和个人理想无法实现的悲愤。

在用词方面，诗人选词精准而富有表现力。首句"早岁那知世事艰"中的"那知"以反问语气表达了对过去天真的反思；"气如山"以简洁的比喻形象描绘了年轻时的雄心壮志。中间两联"楼船夜雪瓜洲渡，铁马秋风大散关"选用军事意象与自然景象相结合的词语，如"楼船"、"铁马"、"夜雪"、"秋风"等，营造出浓厚的战争氛围和边塞景象。后两联中的"空自许"、"已先斑"等词语则简洁有力地传达了诗人内心的失落与衰老之感。

在修辞手法上，这首诗运用了多种技巧。首先是对仗工整，尤其是颔联"楼船夜雪瓜洲渡，铁马秋风大散关"和颈联"塞上长城空自许，镜中衰鬓已先斑"，结构严谨，音韵和谐。其次是比喻手法的运用，如"气如山"将豪情比作高山，"塞上长城"自比为国家的守卫者。再次是对比手法，诗中多次形成对比：年轻时的豪情与年老时的衰颓、理想与现实、过去与现在等，通过对比强化了悲愤之情。此外，诗末"出师一表真名世，千载谁堪伯仲间"运用了用典和反问的修辞手法，引用诸葛亮的《出师表》这一典故，通过反问表达了对古代忠臣的敬仰，同时也暗示了自己的报国之志。

整体而言，这首诗语言风格沉郁顿挫，既有对往事的追忆，又有对现实的感慨，情感起伏跌宕。诗人通过精炼的语言和丰富的修辞手法，将个人命运与国家命运紧密结合，展现了陆游诗歌特有的忧国忧民情怀和悲壮风格。抒情与叙事相结合的表现手法，使这首诗既有具体的历史内容，又有深刻的情感体验，增强了诗歌的艺术感染力。
## 📚 文化札记
《书愤》是南宋著名爱国诗人陆游的作品，表达了诗人对国家命运的忧虑和自己壮志未酬的悲愤。以下是诗中重要的文化背景注释：

"中原北望气如山"中的"中原"指黄河中下游地区，是北宋时期的政治、经济、文化中心，当时已被金国占领。陆游北望中原，表达了对收复失地的强烈愿望。"气如山"形容诗人年轻时豪情壮志如山一般雄壮，体现了坚定的爱国情怀。

"楼船夜雪瓜洲渡"中的"瓜洲渡"位于今江苏省扬州市南部长江北岸，是长江的重要渡口，也是南宋与金国对峙的前线之一。"楼船"是古代建有楼台的大型战船，这句描绘了南宋军队在雪夜中守卫瓜洲渡的情景。

"铁马秋风大散关"中的"大散关"位于今陕西省宝鸡市西南，是南宋与金国交界的重要关隘，历史上曾多次发生战斗。"铁马"指披着铁甲的战马或精锐骑兵，这句展现了南宋军队在秋风中守卫大散关的英勇形象。

"塞上长城空自许"运用了典故，源自《史记·蒙恬列传》中秦始皇派蒙恬修筑长城抵御匈奴的故事。陆游自比为国家的边防屏障，有保卫国家的志向，但"空自许"表示这只是自己的期许，实际上未能实现。

"镜中衰鬓已先斑"指诗人看到镜中自己斑白的鬓发，暗示年事已高，壮志未酬的悲凉。

"出师一表真名世"中的"出师一表"指三国时期蜀汉丞相诸葛亮的《出师表》，是诸葛亮北伐前上给后主刘禅的表文，表达了"鞠躬尽瘁，死而后已"的决心。"千载谁堪伯仲间"意为千年来谁能与诸葛亮相比，表达了陆游对诸葛亮的崇敬，以及自己渴望像诸葛亮一样为国效力的愿望。

陆游生活在南宋与金国对峙的时期，一生主张抗金收复失地，但南宋朝廷主和派占上风，他的理想始终未能实现，这首诗正是他内心悲愤的真实写照。
## 🌅 创作背景
《书愤》创作于南宋淳熙年间，约公元1186年左右，当时陆游已年过六旬，闲居于家乡山阴（今浙江绍兴）。此时的陆游经历了多次政治起落，正处于人生的暮年阶段。他曾经积极主张抗金复国，却因与朝廷主和派政见不合而屡遭贬谪，壮志难酬。诗中"镜中衰鬓已先斑"正是对自身衰老、功业未就的真实写照，流露出深沉的悲愤与无奈。

从历史背景看，陆游生活在南宋与金国对峙的时期。自靖康之变后，宋朝被迫南迁，形成南北分治局面。南宋朝廷内部主和派占据主导，对金采取妥协政策，而陆游等主战派则力主北伐收复失地。这种政治环境下，陆游的爱国理想与现实政治形成尖锐矛盾，导致他一生郁郁不得志。

这首诗的创作动机源于陆游对国家命运的深切忧虑和个人壮志未酬的悲愤。诗题"书愤"直接点明了主题。诗中回顾了青年时代意气风发的抗金抱负，以及曾经参与的军事活动，如"楼船夜雪瓜洲渡，铁马秋风大散关"。最后引用诸葛亮《出师表》，既是对古代忠臣的敬仰，也是对自己报国无门的悲叹，表达了千载之下知音难觅的孤独感。整首诗充满了对国家命运的忧思和对个人遭遇的愤懑，是陆游爱国情怀的典型体现。
## 💭 情感基调
《书愤》的情感基调可概括为悲愤豪迈。诗中既有"中原北望气如山"的雄心壮志，又有"塞上长城空自许"的壮志难酬之愤，形成了悲愤交织的情感基调。同时，诗人对年轻时的豪情回忆和对理想的执着追求，又体现出豪迈之气。

氛围特点可概括为悲壮沉郁。"楼船夜雪瓜洲渡，铁马秋风大散关"描绘了金戈铁马的战场场景，营造出悲壮氛围；而"镜中衰鬓已先斑"则表现了岁月流逝、理想无法实现的沉重感，使全诗笼罩在沉郁的氛围中。这种悲壮沉郁的氛围，正是诗人报国无门、壮志难酬心境的真实写照。
## 🔍 赏析
《书愤》是陆游爱国诗篇的代表作，通过精炼的语言和深沉的情感，展现了诗人对国家命运的忧虑和个人壮志难酬的愤慨。在意境营造上，诗人巧妙运用对比手法，将"早岁那知世事艰"的豪情与"镜中衰鬓已先斑"的现实形成强烈反差，营造出悲壮慷慨的艺术氛围。颔联"楼船夜雪瓜洲渡，铁马秋风大散关"通过意象组合，勾勒出壮阔的战场画面，既有视觉冲击力，又蕴含时空感，使读者仿佛置身于那个风雪交加、战马嘶鸣的抗金战场。

语言特色方面，这首诗凝练有力，气势磅礴。"气如山"的比喻生动形象地表现了诗人年轻时的豪情壮志；颔联采用对偶修辞，结构对称，音韵和谐，通过名词并列形成紧凑有力的节奏感；"塞上长城空自许"中的"空自许"与"已先斑"形成鲜明对比，突出了理想与现实的矛盾。尾联运用反问手法，既表达了对诸葛亮的崇敬，也暗示了自己报国无门的愤慨。

情感表达上，诗歌流露出强烈的爱国情怀和壮志难酬的悲愤。"中原北望"表现了对收复失地的热切盼望；"塞上长城空自许"直接抒发了报国无门的痛苦；对《出师表》的赞美则体现了诗人对历史英雄的崇敬和自我期许。这些情感交织在一起，构成了诗人复杂而丰富的内心世界，也反映了那个时代知识分子的普遍心态。

文学价值方面，这首诗是南宋爱国主义的典范之作，代表了南宋诗歌的最高成就之一。它将个人情感与家国情怀紧密结合，既有鲜明的个性特征，又有深刻的时代烙印。特别是颔联被誉为千古名句，其凝练有力的语言和鲜明的意象组合，成为中国古典诗歌艺术的典范。这首诗对后世产生了深远影响，在民族危亡时刻常常被后人引用，激励人们为国家和民族的利益而奋斗，具有永恒的文学价值和深远的影响意义。
