## 📜 原文
```
北风卷地白草折，胡天八月即飞雪。
忽如一夜春风来，千树万树梨花开。
散入珠帘湿罗幕，狐裘不暖锦衾薄。
将军角弓不得控，都护铁衣冷难着。
瀚海阑干百丈冰，愁云惨淡万里凝。
中军置酒饮归客，胡琴琵琶与羌笛。
纷纷暮雪下辕门，风掣红旗冻不翻。
轮台东门送君去，去时雪满天山路。
山回路转不见君，雪上空留马行处。
```
## 🎯 主题
《白雪歌送武判官归京》主要描绘了边塞地区严寒壮美的自然景象和送别友人的深情场景。诗中通过"胡天八月即飞雪"、"千树万树梨花开"等生动描写，展现了边塞独特的自然风光和严酷环境；同时以"中军置酒饮归客"到"雪上空留马行处"的送别过程，表达了诗人对友人的依依不舍之情。全诗将边塞的壮美、严寒与离别的伤感融为一体，既反映了边塞将士的艰苦生活和坚韧精神，也抒发了诗人深厚的友情和对边塞生活的复杂感受，体现了盛唐边塞诗特有的豪迈与深情并存的风格特点。
## 🏗️ 结构
《白雪歌送武判官归京》是岑参边塞诗的代表作，其结构特点主要体现在章法布局和层次安排两个方面，展现出严谨而灵活的艺术构思。

一、章法布局

1. 起承转合的完整结构
全诗遵循古典诗歌传统的"起承转合"结构：开篇两句"北风卷地白草折，胡天八月即飞雪"直接点明边塞环境，为"起"；中间六句从不同角度描写严寒景象和边塞生活，为"承"；"中军置酒饮归客，胡琴琵琶与羌笛"两句转入送别主题，为"转"；最后四句描写送别过程和离别之情，为"合"。这种结构使全诗层次分明，线索清晰。

2. 景情结合的叙事布局
诗歌采用"景-情-景"的交替布局：先描绘壮丽雪景，再写严寒中的边塞生活，最后回到送别场景和离别之情。这种布局使得景物描写与情感表达相互映衬，情景交融，既展示了边塞的壮美风光，又表达了深厚的离别之情。

3. 视角转换的技巧布局
诗歌视角变化呈现出由远及近、再由近及远的规律：从广阔的边塞雪景，到军营内部的细节，再到送别的具体场景，最后又拉远视角，描写远去的身影。这种视角的转换增强了诗歌的画面感和立体感。

二、层次安排

1. 空间层次
诗歌在空间上形成三个层次：宏观层面描绘整个边塞大地的雪景（"北风卷地"、"胡天飞雪"、"瀚海阑干"）；中观层面描写军营中的情景（"散入珠帘"、"将军角弓"、"都护铁衣"）；微观层面聚焦送别的细节（"中军置酒"、"风掣红旗"、"雪上马行处"）。这种空间层次的处理使诗歌既有宏大的气势，又不失细腻的描写。

2. 时间层次
诗歌在时间上呈现清晰的推进：从雪景的静态描写，到严寒天气的持续，再到送别宴会的场景，最后是送别过程的动态描写。这种时间层次的安排使诗歌情节发展自然流畅，富有节奏感。

3. 情感层次
诗歌情感表达层层递进：先是对边塞奇景的惊叹，然后是对严寒环境的感受，接着是送别时的复杂情感，最后是离别后的惆怅和思念。尤其是结尾"山回路转不见君，雪上空留马行处"，以景结情，余韵悠长，表达了深沉的离别之情。

4. 意象层次
诗歌在意象安排上丰富多样：自然意象（北风、白草、飞雪、冰、愁云等）展现边塞环境；人文意象（珠帘、狐裘、角弓、胡琴等）反映边塞生活；动态意象（卷地、飞雪、散入、掣、冻等）增强画面感。这些意象的层次安排使诗歌意境丰富，形象生动。

总体而言，《白雪歌送武判官归京》结构严谨而不失灵活，层次分明而富于变化，充分展现了岑参边塞诗的艺术特色和结构成就。
## ✍️ 语言风格
岑参的《白雪歌送武判官归京》是唐代边塞诗的代表作，其语言风格特点鲜明，艺术成就突出。以下从多个方面分析其语言风格特点：

在用词方面，诗人选用了大量形象鲜明的边塞景物词汇，如"北风"、"白草"、"飞雪"、"瀚海"、"愁云"等，直接勾勒出边塞严酷的自然环境。"白草折"、"飞雪"等词语简洁有力，一语道破边塞的独特风光。同时，诗人巧妙运用冷暖对比的词汇，如"春风"与"飞雪"、"梨花开"与"冻不翻"等，形成强烈反差，增强了诗歌的艺术效果。诗中还使用了"珠帘"、"罗幕"、"狐裘"、"锦衾"、"角弓"、"铁衣"等具有边塞特色的器物词汇，丰富了诗歌内容，增强了地域特色。此外，"愁云惨淡"、"空留"等带有明显情感色彩的词语，表达了诗人对友人的不舍之情。

在修辞手法上，比喻的运用最为精妙，特别是"忽如一夜春风来，千树万树梨花开"这一千古名句，将雪景比作梨花盛开，生动形象地展现了诗人丰富的想象力和独特的审美视角。夸张手法也多处运用，如"千树万树梨花开"、"百丈冰"、"万里凝"等，通过夸张的数字和空间描述，强化了边塞环境的广阔和严酷。对比手法贯穿全诗，包括冷暖对比、动静对比（"风掣红旗"与"冻不翻"）、内外对比等，使诗歌更加丰富多彩。拟人手法如"愁云惨淡万里凝"中的"愁云"，赋予自然物以人的情感，增强了诗歌的抒情性。借景抒情的手法则体现在最后两句"山回路转不见君，雪上空留马行处"，通过雪地上留下的马蹄印，含蓄地表达了诗人的离别之情。

在句式特点上，诗歌对仗工整，如"将军角弓不得控，都护铁衣冷难着"、"中军置酒饮归客，胡琴琵琶与羌笛"等，句式整齐，音韵和谐。整体以七言为主，但灵活运用句式变化，使诗歌节奏感强，读来抑扬顿挫。首尾呼应也是其特点，开头写"北风卷地白草折，胡天八月即飞雪"，结尾写"山回路转不见君，雪上空留马行处"，都以雪景为背景，形成完整结构。

在音韵方面，全诗押韵工整，平仄交替，声调变化丰富，使诗歌读来朗朗上口，富有节奏感。

从整体风格来看，这首诗雄浑豪放，气势磅礴，描绘的边塞风光壮阔，展现了诗人开阔的胸襟和豪迈的气概。诗人将边塞的雪景与送别之情完美融合，达到情景交融的艺术境界。想象丰富，能够将边塞的严酷环境转化为美丽的艺术形象，体现了诗人独特的艺术视角和创造力。语言精炼而不失生动，用词准确而富有表现力，展现了诗人高超的语言驾驭能力。

综上所述，岑参的《白雪歌送武判官归京》在语言风格上具有形象鲜明、对比强烈、修辞丰富、句式工整、音韵和谐等特点，整体风格雄浑豪放，情景交融，想象丰富，语言精炼，是唐代边塞诗的杰出代表，充分展现了中国古典诗歌的艺术魅力。
## 📚 文化札记
《白雪歌送武判官归京》是唐代著名边塞诗人岑参的代表作，创作于天宝年间，当时岑参在安西都护府任职。这首诗描绘了西北边塞的严寒景象和送别武判官归京的场景，充满了浓郁的边塞风情和深厚的友情。

"胡天"中的"胡"是古代中原对北方少数民族的称呼，"胡天"即指北方边塞地区的天空，反映了唐代边疆与中原的文化差异。"白草"是北方边塞特有的一种草，秋季变白，耐寒耐旱，是边塞地区常见的植物。

"瀚海"一词在古代有不同解释，有时指北方的大湖泊，有时指广阔的沙漠，这里应指西北地区的大沙漠。"阑干"形容纵横交错的样子，描绘了冰封沙漠的壮观景象。

"都护"是唐代设置在边疆地区的最高军政长官，负责管理边防事务，安西都护府是唐代管理西域地区的重要机构。"角弓"是用牛角加固的弓，是古代重要的军事武器，在寒冷天气中因手冻僵而难以拉开。

"中军"指军队主帅所在的营地，"辕门"是军营的门，反映了唐代军队的营地设置。"胡琴、琵琶与羌笛"都是少数民族乐器，胡琴源于北方游牧民族，琵琶最初来自西域，羌笛是羌族的乐器，这些乐器的出现体现了边塞地区多元文化交融的特点。

"轮台"是地名，在今新疆维吾尔自治区轮台县一带，是唐代西北边疆的重要军事据点和丝绸之路上的重镇。"天山"是横亘新疆中部的大山脉，是西域地理的重要标志。

诗中"狐裘"和"锦衾"分别是贵重的皮袍和锦缎被子，即使在这样保暖的衣物下，边塞的寒冷依然难以抵挡，突显了边塞地区气候的严酷。"铁衣"指铠甲，"不得控"和"冷难着"生动地表现了极寒天气对军事活动的影响。

这首诗通过对边塞严寒气候的描写和对送别场景的刻画，不仅展现了唐代边塞将士的生活状况，也反映了唐代边疆地区的地理风貌和民族文化特色，是边塞诗中的经典之作。
## 🌅 创作背景
《白雪歌送武判官归京》创作于唐玄宗天宝十三载（754年）左右，地点在安西都护府（今新疆库车附近）的轮台。当时岑参正随封常清出任安西北庭节度判官，这是他第二次出塞。诗中所描写的"胡天八月即飞雪"正是西域地区独特的气候特征。

岑参（约715-770）是唐代著名边塞诗人，此时他已年近四十，身处边塞军旅，生活条件艰苦，远离家乡和亲人。在边塞的岁月里，他亲身体验了戍边将士的艰辛生活，也感受到了边疆壮阔的自然风光和独特的民族风情。这种经历使他的诗歌形成了雄浑豪放、气势磅礴的风格。在送别武判官时，他既有对友人的不舍，也有对边塞生活的深刻感悟。

唐朝天宝年间（742-756年），表面繁荣昌盛，实则内部矛盾已经积累。在边疆地区，唐朝与吐蕃、大食（阿拉伯帝国）等势力争夺西域控制权。安西都护府作为唐朝在西域的重要军事和行政机构，承担着保卫国家领土、管理西域事务的重任。边塞将士们常年驻守在环境恶劣的边疆，为国家安全默默奉献。

这首诗的创作动机是送别一位姓武的判官返回京城。判官是唐代地方官府中的属官，负责处理文书、司法等事务。武判官可能是与岑参同在安西都护府任职的同事。在送别的场合，岑参被边塞特有的自然景观（如八月飞雪的奇特景象）和军营生活所触动，将离别的情感与边塞的壮美风光融为一体，创作出了这首千古传诵的边塞诗名篇。诗中既有对边塞严寒气候的生动描绘，也有对军营生活的真实写照，更有对友人离别的深情表达，充分展现了岑参边塞诗的艺术特色。
## 💭 情感基调
《白雪歌送武判官归京》的情感基调可概括为：严寒壮阔、凄清豪迈、忧愁不舍。

氛围特点表现为：荒凉壮美、孤寂热烈、凄清雄浑。

诗中通过"北风卷地"、"胡天八月即飞雪"等描绘边塞严寒；以"忽如一夜春风来，千树万树梨花开"展现壮美雪景；用"愁云惨淡万里凝"直抒胸臆；借"中军置酒饮归客"的热烈场面与"山回路转不见君"的孤寂形成对比，整体呈现出边塞特有的壮阔与凄清交织的独特氛围。
## 🔍 赏析
《白雪歌送武判官归京》是唐代边塞诗人岑参的代表作，以其独特的艺术魅力和深厚的情感内涵成为边塞诗的经典之作。在意境营造上，诗人通过对比手法将边塞严寒与内地温暖形成鲜明对照，如"忽如一夜春风来，千树万树梨花开"以春景喻雪景，既形象生动又暗含思乡之情。同时，诗人运用夸张手法描绘"瀚海阑干百丈冰，愁云惨淡万里凝"的壮阔景象，以及动静结合的技巧如"风掣红旗冻不翻"，立体构建了边塞特有的苍凉壮美意境。

语言特色方面，诗歌雄浑豪放，用词精准有力。"卷"、"折"、"掣"等动词生动传神，"白草"、"梨花"、"红旗"等名词色彩鲜明。修辞上，比喻最为突出，尤其是将雪景比作梨花盛开，成为千古名句；拟人如"愁云惨淡万里凝"赋予自然以情感；对偶如"将军角弓不得控，都护铁衣冷难着"工整有力，增强了诗歌的节奏感和表现力。

情感表达上，诗人既有对边塞严寒环境的惊叹与不适，如"狐裘不暖锦衾薄"，又有对友人的深情厚谊与不舍，如"山回路转不见君，雪上空留马行处"。主题上，诗歌不仅表现了边塞的壮阔与严酷，反映了军旅生活的艰辛，也表达了深厚的友情，体现了中国传统文化中的重情重义精神，同时隐含对和平的向往和对归乡的渴望。

作为唐代边塞诗的典范，这首诗将边塞景色与军旅生活、友情送别完美结合，展现了边塞诗的独特魅力。其名句被广泛传诵，对后世文学创作产生深远影响，丰富和发展了中国古代诗歌中的边塞题材。同时，作为研究唐代边塞历史、军事和文化的重要文献，这首诗也体现了中国古代文人的家国情怀和人文精神，是中华优秀传统文化的重要组成部分。
