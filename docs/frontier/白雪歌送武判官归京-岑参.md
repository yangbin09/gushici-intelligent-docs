## 📜 原文
```
北风卷地白草折，胡天八月即飞雪。
忽如一夜春风来，千树万树梨花开。
散入珠帘湿罗幕，狐裘不暖锦衾薄。
将军角弓不得控，都护铁衣冷难着。
瀚海阑干百丈冰，愁云惨淡万里凝。
中军置酒饮归客，胡琴琵琶与羌笛。
纷纷暮雪下辕门，风掣红旗冻不翻。
轮台东门送君去，去时雪满天山路。
山回路转不见君，雪上空留马行处。
```
## 🎯 主题
《白雪歌送武判官归京》的主要主题思想是通过描绘边塞严寒壮美的自然环境和艰苦的军旅生活，表达对即将归京友人的深情送别。诗中既展现了边塞"忽如一夜春风来，千树万树梨花开"的壮美雪景，又刻画了"狐裘不暖锦衾薄"、"将军角弓不得控"的严寒艰苦，形成鲜明的对比。诗人以"雪上空留马行处"作结，含蓄表达了依依惜别之情，同时也反映了边塞将士的坚毅精神和对家乡的眷恋，体现了岑参作为边塞诗人对边塞生活的深刻体验和家国情怀。
## 🏗️ 结构
《白雪歌送武判官归京》是岑参边塞诗的代表作，其结构特点精妙而严谨，体现了诗人高超的艺术构思。

从章法布局来看，全诗可分为四个层次，层层递进，环环相扣：

第一层为自然环境描写（第1-4句）。开篇"北风卷地白草折，胡天八月即飞雪"直接点明边塞的严寒气候，为全诗奠定基调。接着"忽如一夜春风来，千树万树梨花开"以绝妙的比喻描绘雪景，将严寒的冬雪比作春日盛开的梨花，既形象生动又富有美感，展现了诗人独特的艺术视角。

第二层为人物感受描写（第5-8句）。诗人从室外转向室内，从自然环境转向人的感受。"散入珠帘湿罗幕，狐裘不暖锦衾薄"写室内寒冷；"将军角弓不得控，都护铁衣冷难着"写将士的艰难处境；"瀚海阑干百丈冰，愁云惨淡万里凝"则将视野再次扩大，描写广阔天地间的严寒景象。这一层次由近及远，由小到大，层层展开，全面展现了边塞的严寒环境。

第三层为送别场景描写（第9-10句）。"中军置酒饮归客，胡琴琵琶与羌笛"两句从自然描写转向人的活动，点明送别的主题。军中设宴，多种乐器齐奏，既表现了军中生活的特点，也暗示了送别的隆重气氛。

第四层为离别情感描写（第11-14句）。"纷纷暮雪下辕门，风掣红旗冻不翻"描绘送别时的环境；"轮台东门送君去，去时雪满天山路"写送别的过程；"山回路转不见君，雪上空留马行处"则以深情的笔触描绘离别后的情景，余韵悠长，令人回味。

从结构安排上看，这首诗采用了"由景到人、由外到内、再由内到外"的章法布局。开头描写边塞的自然环境，然后转向室内的人物感受，再转向送别场景，最后又回到室外的送别过程和离别情景，形成了一个完整的结构循环。

此外，全诗还体现了首尾呼应的特点。开头写"胡天八月即飞雪"，结尾写"雪上空留马行处"，都以雪为背景，形成首尾呼应，使全诗结构更加完整。

在表现手法上，诗人运用了情景交融、虚实结合、动静结合等多种艺术手法。如将雪景与送别情感紧密结合，既有对实际景象的描写，也有想象和比喻，既有动态的描写（如"北风卷地"），也有静态的描写（如"瀚海阑干百丈冰"），使诗歌更加生动丰富。

总的来说，《白雪歌送武判官归京》结构严谨，层次分明，章法布局巧妙，艺术手法多样，充分展现了岑参边塞诗的艺术特色和诗人深厚的艺术功底。
## ✍️ 语言风格
《白雪歌送武判官归京》是岑参边塞诗的代表作，其语言风格具有鲜明的特色，主要表现在以下几个方面：

在用词方面，诗人选用了大量具有边塞特色的词汇，如"胡天"、"瀚海"、"都护"、"轮台"等，直接点明了诗歌的边塞背景。同时，诗中色彩对比鲜明，"白草"、"白雪"与"红旗"形成强烈的视觉冲击。动词运用生动有力，如"卷地"、"飞雪"、"散入"、"掣"等，准确描绘了边塞风雪的猛烈态势。此外，诗中还运用了"梨花"、"珠帘"、"狐裘"、"锦衾"、"角弓"、"铁衣"等具体物象，构建了丰富的意象系统。

在修辞手法上，比喻的运用尤为出色，"忽如一夜春风来，千树万树梨花开"将雪景比作梨花盛开，既形象又富有美感，成为千古名句。夸张手法在诗中也屡见不鲜，如"百丈冰"、"万里凝"等，通过夸张表现边塞严寒的极致。对比手法增强了诗歌的表现力，"狐裘不暖锦衾薄"中的冷暖对比，"风掣红旗冻不翻"中的动静对比，都形成了强烈的艺术效果。拟人手法如"愁云惨淡万里凝"，将云赋予人的情感，增强了诗歌的抒情性。

句式特点上，全诗以七言为主，但句式灵活多变，如"忽如一夜春风来，千树万树梨花开"这样的对偶句，增强了诗歌的节奏感。首尾呼应也是这首诗的重要特点，开头写"北风卷地白草折，胡天八月即飞雪"，结尾写"山回路转不见君，雪上空留马行处"，首尾都以雪景呼应，形成完整结构。

意境营造方面，诗人将壮阔与细腻巧妙结合，既有"瀚海阑干百丈冰，愁云惨淡万里凝"的壮阔景象，又有"散入珠帘湿罗幕"的细腻描写。情景交融是这首诗的重要特色，将送别之情与边塞雪景融为一体，如"纷纷暮雪下辕门，风掣红旗冻不翻"，既写景又抒情。虚实结合的手法也运用得恰到好处，既有实景描写，如"轮台东门送君去"，又有想象，如"山回路转不见君，雪上空留马行处"。

在声音效果上，诗中运用了"胡琴琵琶与羌笛"等乐器的描写，增强了诗歌的音乐性。全诗押韵工整，读来朗朗上口，音韵和谐。

情感表达方面，诗人没有直接抒发离别之情，而是通过对雪景的描写和送别场景的刻画，含蓄地表达了不舍之情。整体风格豪放，但在细节描写上又极为细腻，体现了诗人情感的丰富性。

总之，岑参的这首诗通过精湛的语言艺术，成功地将边塞风雪的壮美与送别友人的深情融为一体，成为唐代边塞诗的经典之作。
## 📚 文化札记
《白雪歌送武判官归京》是唐代边塞诗人岑参的代表作，创作于作者在安西、北庭都护府任职期间。以下是诗中重要的文化背景注释：

"胡天"指北方少数民族地区的天空，"胡"是古代中原对北方民族的泛称，这里指西域地区。"白草"是边塞地区的一种坚韧草木，秋天变白，被强劲北风吹折，突显了风势之大。

"千树万树梨花开"是著名的比喻，将雪花比作梨花，既形象又美丽，表现了诗人面对严寒依然保持的乐观态度和丰富想象力。

"珠帘"、"罗幕"指军帐中的装饰，表明即使是高级军官的住所也难以抵御严寒。"狐裘"是狐狸皮制成的贵重冬衣，"锦衾"是锦缎被子，两者在严寒中都失去了保暖效果，突显边塞气候之恶劣。

"角弓"是用牛角装饰的弓，是古代重要武器。"都护"是唐代边疆地区的军事长官，负责管理边疆地区的军事和行政事务。"铁衣"即铁甲，古代将士的防护装备。

"瀚海"指西北地区的大沙漠或戈壁，有时也指湖泊。"阑干"形容纵横交错的样子，描绘冰裂纹的景象。"愁云"既形容阴沉的云层，也暗示送别时的伤感情绪。

"中军"指古代军队主帅所在部队，这里指主帅的营帐。"归客"指即将归京的武判官。"胡琴"、"琵琶"、"羌笛"都是古代乐器，其中胡琴和羌笛是西北少数民族的乐器，反映了边塞地区多民族文化交流的特点。

"辕门"是古代军营的出入口。"风掣红旗冻不翻"生动描绘了边塞严寒，风虽大但红旗被冻住不能飘扬的细节。

"轮台"是地名，在今新疆轮台县附近，唐代是重要的军事据点，岑参曾在此任职。"天山"指今天的新疆天山山脉，是唐代西域的天然屏障。

诗题中的"武判官"指武元衡，唐代官员，在边塞地区任职，此时要返回京城。"判官"是唐代官职，在节度使或都护府中负责文书和司法等工作。

这首诗通过描绘边塞严寒和壮丽景色，以及送别友人的场景，表现了边塞生活的艰苦和诗人对友人的深厚情谊，是边塞诗的代表作。
## 🌅 创作背景
《白雪歌送武判官归京》创作于唐代天宝十三年（754年）左右，地点在北庭都护府（今新疆吉木萨尔县北）的轮台（今新疆轮台县）。当时岑参约四十岁，正任北庭都护府判官，这是他第二次出塞任职。在边塞的生活虽然艰苦，但也是他创作的高峰期。他经历了边塞的严酷环境，对边塞生活有了深刻认识，同时也因长期离家而思乡情切。

天宝年间是唐玄宗统治后期，表面繁荣，实则暗藏危机，安史之乱即将爆发。在边疆地区，唐朝与吐蕃、大食等势力争夺西域控制权，边境时有战事。唐朝在西域设有安西都护府和北庭都护府，岑参所在的北庭都护府是唐朝在西域的重要军事和政治中心，负责管理天山以北地区。

这首诗的创作动机是送别一位姓武的判官返回京城。武判官可能是岑参的同僚或上级，两人建立了深厚友谊。诗中描绘了边塞的严寒雪景和送别场景，表达了作者对友人的不舍之情。灵感来源于边塞特有的自然环境和军旅生活。边塞的八月就开始飞雪，这种严酷气候与内地形成鲜明对比，给诗人留下深刻印象。诗中"忽如一夜春风来，千树万树梨花开"等名句，生动描绘了边塞雪景的壮美；而"中军置酒饮归客，胡琴琵琶与羌笛"等场景，则展现了边塞军营中的生活，成为创作的重要素材。
## 💭 情感基调
《白雪歌送武判官归京》的情感基调可概括为：离别惆怅、边塞苦寒、友情深厚、壮美感慨。氛围特点表现为：严寒壮阔、凄凉奇幻、离愁别绪、苍茫孤寂。

全诗以边塞严寒的自然环境为背景，通过"北风卷地"、"胡天飞雪"等描写营造出苦寒氛围，又以"千树万树梨花开"的比喻展现雪景的壮美奇幻。诗中"愁云惨淡万里凝"一句直接点明了凄凉氛围，而"中军置酒饮归客"则表现了深厚的友情。结尾"山回路转不见君，雪上空留马行处"以景结情，将离别的惆怅与苍茫孤寂之感推向高潮，使整首诗在壮美的边塞雪景中蕴含着深沉的离愁别绪。
## 🔍 赏析
《白雪歌送武判官归京》是岑参边塞诗的代表作，以其雄奇壮丽的艺术风格和深刻的情感表达，成为唐代边塞诗的经典之作。

在意境营造上，诗人通过"北风卷地白草折，胡天八月即飞雪"开篇，直接展现北方边塞的严酷环境，与中原形成鲜明对比。尤其"忽如一夜春风来，千树万树梨花开"这一绝妙比喻，将飞雪比作盛开的梨花，在严寒中创造出美丽壮观的意境。"瀚海阑干百丈冰，愁云惨淡万里凝"通过夸张手法，强化了边塞的荒凉感，而"风掣红旗冻不翻"和"雪上空留马行处"等细节描写，则生动再现了边塞的严寒与离别的凄凉，使读者如临其境。

语言特色上，岑参善用精炼语言描绘宏大景象，修辞技巧丰富多样。比喻是本诗最突出的特点，尤其是梨花喻雪，形象生动且富有诗意。夸张手法如"百丈冰"和"万里凝"，增强了诗歌的震撼力。对偶如"将军角弓不得控，都护铁衣冷难着"，排比如"胡琴琵琶与羌笛"，使诗歌节奏鲜明，音韵和谐。拟人手法如"愁云惨淡"，则赋予自然景物以人的情感，增强了感染力。

情感表达方面，诗人既表达了对友人的惜别之情，也反映了边塞将士的艰苦生活和家国情怀。"中军置酒饮归客"展现了军中送别的热闹与伤感，而"山回路转不见君，雪上空留马行处"则直接表达了依依不舍之情。主题上，诗中通过描绘边塞环境和军中生活，展现了边塞将士的艰辛，表达了诗人的同情与敬意，同时也隐含了对战争的思考和对和平的向往。

从文学价值看，这首诗艺术成就极高，丰富了边塞诗的题材和表现手法，将自然环境、军中生活和送别情感有机结合。诗中对边塞气候、地理和军中生活的描写，也具有重要的史料价值。对后世而言，岑参的雄奇风格和真实描绘成为边塞诗创作的典范，其爱国情怀和对和平的向往也对后世文学创作和思想观念产生了深远影响，使这首诗成为中国古典诗歌中的不朽经典。
