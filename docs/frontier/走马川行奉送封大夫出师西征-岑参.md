## 📜 原文
```
君不见走马川行雪海边，平沙莽莽黄入天。
轮台九月风夜吼，一川碎石大如斗，随风满地石乱走。
匈奴草黄马正肥，金山西见烟尘飞，汉家大将西出师。
将军金甲夜不脱，半夜军行戈相拨，风头如刀面如割。
马毛带雪汗气蒸，五花连钱旋作冰，幕中草檄砚水凝。
虏骑闻之应胆慑，料知短兵不敢接，车师西门伫献捷。
```
## 🎯 主题
《走马川行奉送封大夫出师西征》主要描绘了西域战场的恶劣环境和戍边将士的英勇形象。诗人通过生动刻画风沙肆虐、严寒刺骨的极端自然条件，展现了唐军将士不畏艰险、夜以继日行军作战的坚韧精神。诗中将军金甲不脱、风如刀割的描写，以及马毛带雪、砚水凝冰的细节，突出了戍边之艰辛。全诗洋溢着对唐军必胜的坚定信念，期待在车师西门迎来胜利捷报，体现了诗人深厚的爱国主义情怀和对将士们的崇高敬意。这首边塞诗气势磅礴，意境雄浑，真实反映了唐代西北边疆的军事形势和将士们的英雄气概。
## 🏗️ 结构
《走马川行奉送封大夫出师西征》是岑参边塞诗的代表作，其结构特点鲜明，章法布局严谨而灵活，层次安排有序而富有变化。

从章法布局看，这首诗采用古风体，不拘泥于固定格律，全诗十五句，句式长短不一，形成自由奔放的风格。开篇以"君不见"起势，气势磅礴，结尾以"伫献捷"作结，首尾呼应，结构完整。这种自由体的形式与边塞诗的豪放风格相得益彰，为诗人抒发情感提供了广阔空间。

从层次安排看，全诗可分为三个清晰的层次：

第一层次为环境描写，包括前五句："君不见走马川行雪海边，平沙莽莽黄入天。轮台九月风夜吼，一川碎石大如斗，随风满地石乱走。"这一层次以夸张手法描绘边塞荒凉景象，黄沙漫天，狂风怒吼，碎石遍地，为全诗奠定了雄浑悲壮的基调。

第二层次交代出师背景和行军场景，包括中间六句："匈奴草黄马正肥，金山西见烟尘飞，汉家大将西出师。将军金甲夜不脱，半夜军行戈相拨，风头如刀面如割。"这一层次既交代了出师原因，又展现了将士们夜行的艰苦场景，将军不卸甲，士兵连夜行军，突显了将士们的英勇精神。

第三层次进一步描写行军艰苦和预期胜利，包括最后四句："马毛带雪汗气蒸，五花连钱旋作冰，幕中草檄砚水凝。虏骑闻之应胆慑，料知短兵不敢接，车师西门伫献捷。"这一层次从马匹出汗结冰、砚水冻结的细节描写，转向对胜利的预期展望，形成了情感的升华。

在结构艺术上，这首诗还体现了动静结合、虚实结合、对比等手法。既有静态的环境描写，也有动态的场景刻画；既有对现实环境的真实描写，也有对胜利的预期展望；通过边塞环境的恶劣与将士们的英勇形成对比，突出了将士们的精神风貌。

这种结构安排既符合送别诗的特点，又体现了边塞诗的豪放风格，使全诗在形式和内容上达到了高度统一，展现了岑参边塞诗的独特魅力。
## ✍️ 语言风格
《走马川行奉送封大夫出师西征》是岑参边塞诗的代表作，其语言风格特点鲜明，主要体现在以下几个方面：

在用词方面，诗人大量运用边塞特色词汇，如"走马川"、"雪海"、"轮台"、"金山"、"匈奴"、"汉家大将"、"虏骑"、"车师"等，营造出浓郁的西域战争氛围。动词选择生动有力，如"吼"、"乱走"、"飞"、"出师"、"拨"、"割"、"蒸"、"凝"等，增强了诗歌的动态感和紧张感。形容词使用形象鲜明，如"莽莽"、"黄"、"大如斗"、"肥"等，使读者能够直观感受到边塞环境的恶劣。同时，"黄"（沙）、"金"（甲、山）等色彩词的运用，形成鲜明对比，增强了视觉效果。

在修辞手法上，比喻手法运用巧妙，如"一川碎石大如斗"以"斗"作比，形象描绘碎石巨大；"风头如刀面如割"将寒风比作刀，生动表现风力猛烈。夸张手法突出，如"平沙莽莽黄入天"夸张描绘沙漠广阔无垠；"随风满地石乱走"夸张表现风势之猛烈。对偶手法如"风头如刀面如割"增强了语言的节奏感。排比手法在"将军金甲夜不脱，半夜军行戈相拨，风头如刀面如割"三句中形成递进，增强紧张感。拟人手法如"轮台九月风夜吼"赋予风以人的动作，生动表现风声凄厉。开篇"君不见"采用设问手法，引起读者注意。

句式上，诗歌以整齐的七言句为主，节奏鲜明，朗朗上口。诗句之间衔接紧密，一气呵成，展现战争场面的紧张感和连续性。末尾三句"虏骑闻之应胆慑，料知短兵不敢接，车师西门伫献捷"形成递进关系，表达对战争胜利的坚定信心。

整体风格雄浑豪放，气势磅礴，语言刚劲有力，展现了唐朝军队的雄壮气势。描写形象生动，通过具体细节使读者如临其境，感受边塞荒凉、战争残酷和将士英勇。意境开阔，描绘广阔边塞风光和宏大战争场面。情感真挚，对边塞将士艰苦生活深表同情，对出征西征充满敬意和期待。

这首诗以其鲜明的边塞特色、生动有力的用词、丰富的修辞手法和雄浑豪放的风格，成为唐代边塞诗的代表作，充分展现了岑参对边塞生活的深刻体验和对战争场面的艺术再现能力。
## 📚 文化札记
《走马川行奉送封大夫出师西征》是唐代边塞诗人岑参的代表作，创作于唐玄宗天宝年间（约742-756年）。这首诗是为送别封常清（封大夫）出征西域而作，展现了西域艰苦的自然环境和唐军将士英勇作战的精神。

走马川是古代丝绸之路上的重要通道，位于今新疆维吾尔自治区境内。雪海指新疆地区的高山雪原或冰湖，被视为西域险恶之地。诗中"平沙莽莽黄入天"描绘了西域荒凉壮阔的沙漠景观。

轮台是古代西域重要地名，位于今天的新疆轮台县一带，汉代曾为西域都护府所在地，唐代时是重要的军事据点。诗中"轮台九月风夜吼，一川碎石大如斗，随风满地石乱走"生动描述了西域地区恶劣的气候条件。

诗中"匈奴"是古代北方游牧民族的泛称，这里指唐代西部的游牧民族（如突厥等）。"草黄马正肥"指秋季草黄，马匹膘肥体壮，正是游牧民族发动战争的季节。金山可能指阿尔泰山，位于今天的新疆北部和蒙古西部，是唐代西北边疆的重要山脉。

"汉家大将"指唐朝将领封常清，古代诗歌中常用"汉家"代指唐朝。封常清是唐代著名将领，曾任安西副都护、安西节度使等职，在西域多次征战，抵御外敌，维护了唐朝在西域的统治。岑参曾在其幕府中任职，对封常清十分敬重。

"五花连钱"指名贵的马匹，马身上的毛色呈五花或连钱状（圆形斑纹）。"幕中草檄"指在军帐中起草征讨文书，"砚水凝"形容砚台中的墨水都结冰了，突显西域严寒的气候。

车师是古代西域国名，位于今天的新疆吐鲁番一带，唐代时是安西都护府辖区。"车师西门伫献捷"表达了诗人对唐军必胜的信心，预计唐军很快就会在车师取得胜利。

这首诗体现了盛唐边塞诗的典型特色：雄浑豪放的风格、生动的边塞风光描写、对将士英勇精神的赞美以及对战争的乐观态度。通过对比西域恶劣的自然环境与唐军将士的英勇无畏，诗人展现了唐朝军队的强大战斗力和必胜信念，同时也反映了唐朝在西域的军事活动和文化影响。
## 🌅 创作背景
《走马川行奉送封大夫出师西征》创作于唐玄宗天宝十三载(754年)左右，地点在安西都护府(今新疆库车一带)。当时岑参作为安西节度使封常清的幕僚，随军出征，亲历了西域的军事生活。

岑参此时已是第二次出塞。他于天宝八载(749年)首次到达安西，天宝十载(751年)返回长安，天宝十三载再次出塞。作为一位有着丰富边塞生活经验的诗人，岑参对西域的自然环境和军事生活有着深刻的了解和体验。此时的他，既是一位亲历者，也是一位记录者，对边塞将士的艰苦生活和英勇精神充满敬意，同时也怀着对国家边疆安全的关切。

这首诗创作于唐朝由盛转衰的关键时期。天宝年间，唐朝虽表面繁荣，但内部危机四伏。在西域，唐朝与吐蕃、大食(阿拉伯帝国)等势力争夺控制权。天宝十载，唐将高仙芝在怛罗斯之战中败于大食，唐朝在西域的势力受到严重削弱。为巩固西北边疆，唐朝加强了军事部署，封常清作为安西节度使，肩负着保卫西域的重任。诗中提到的"匈奴草黄马正肥，金山西见烟尘飞"，正是当时边疆紧张局势的真实写照。

岑参创作此诗的直接动机是送别封常清出征西征。作为幕僚，他亲身参与了军事行动，对西域恶劣的自然环境和艰苦的军事生活有着切身体会。诗中"轮台九月风夜吼，一川碎石大如斗，随风满地石乱走"等生动描写，都源于作者的亲身经历和细致观察。同时，诗人也希望通过这首诗表达对唐军必胜的信心和对封常清的敬意与祝福，展现了盛唐时期边塞诗的典型特征——豪迈激昂、气势恢宏。
## 💭 情感基调
这首诗的情感基调可概括为雄浑悲壮，昂扬向上。诗人通过描绘边塞壮阔景象和严酷环境，展现出一种面对艰难险阻仍奋发向前的精神风貌。

氛围特点则表现为艰苦严酷，紧张激烈。诗中"风夜吼"、"风头如刀"等意象营造出边塞的恶劣自然环境；"半夜军行"、"金甲夜不脱"等描写则突显了战前的紧张气氛。然而，诗的结尾却透露出必胜的信心，使全诗在悲壮中不失豪迈，在艰苦中见精神，体现了唐代边塞诗特有的阳刚之美。
## 🔍 赏析
《走马川行奉送封大夫出师西征》是岑参边塞诗的代表作，以雄浑豪放的笔调描绘了西域征战的壮阔场景。在意境营造上，诗人通过"平沙莽莽黄入天"等宏大景象，勾勒出边塞的苍凉壮美；"一川碎石大如斗，随风满地石乱走"的动态描写，生动展现了自然环境的险恶。诗中巧妙运用写实与夸张相结合的手法，如"风头如刀面如割"的比喻，既夸张又形象，使读者仿佛身临其境。

语言特色方面，全诗气势磅礴，动词运用丰富有力，"吼"、"走"、"飞"、"出"、"拨"等充满动态感。修辞技巧多样，夸张手法如"一川碎石大如斗"强化了环境险恶；"马毛带雪汗气蒸，五花连钱旋作冰"的冷热对比，突显极端气候；排比句式增强了节奏感和军旅气势。这些语言特点形成了诗歌鲜明的艺术风格。

情感表达上，诗人通过对艰苦环境和将士顽强意志的描写，间接表达了对封大夫及将士的敬佩之情。主题思想展现了唐代将士不畏艰险、奋勇向前的英雄气概，反映了开疆拓土、保家卫国的时代精神。结尾"虏骑闻之应胆慑，料知短兵不敢接，车师西门伫献捷"表达了诗人对战争胜利的坚定信心。

作为唐代边塞诗的典范，此诗将壮阔的边塞景色与军旅生活融为一体，既有宏大场面又有细腻细节，艺术成就卓著。它记录了唐代边塞战争的历史场景，展现了唐人精神风貌，对后世文学创作产生深远影响，其表现的爱国主义精神和英雄气概，成为中华民族精神文化的重要组成部分。
