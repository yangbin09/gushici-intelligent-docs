## 📜 原文
```
轮台城头夜吹角，轮台城北旄头落。
羽书昨夜过渠黎，单于已在金山西。
戍楼西望烟尘黑，汉兵屯在轮台北。
上将拥旄西出征，平明吹笛大军行。
四边伐鼓雪海涌，三军大呼阴山动。
虏塞兵气连云屯，战场白骨缠草根。
剑河风急雪片阔，沙口石冻马蹄脱。
亚相勤王甘苦辛，誓将报主静边尘。
古来青史谁不见，今见功名胜古人。
```
## 🎯 主题
《轮台歌奉送封大夫出师西征》的主要主题思想是歌颂封大夫出征西征的英勇壮举和忠诚报国精神。诗中通过描绘紧张的军情、壮阔的出征场面、残酷的战争环境以及边塞的艰苦条件，展现了封大夫"勤王甘苦辛，誓将报主静边尘"的坚定决心和爱国情怀。同时，诗人以"古来青史谁不见，今见功名胜古人"的赞颂，表达了对封大夫功绩将超越前人的高度评价。整首诗既表现了战争的残酷与边塞的艰苦，也彰显了唐军的威武气势和必胜信念，体现了岑参边塞诗特有的豪迈风格和爱国情怀。
## 🏗️ 结构
《轮台歌奉送封大夫出师西征》是岑参边塞诗的代表作，其结构特点严谨而富有变化，体现了诗人高超的艺术构思。

从章法布局来看，全诗采用典型的起承转合结构。起首两句"轮台城头夜吹角，轮台城北旄头落"点明地点和战争气氛，为全诗奠定基调。承接部分"羽书昨夜过渠黎"至"平明吹笛大军行"，描述敌情紧急和我军准备，情节推进自然。转的部分从"四边伐鼓雪海涌"到"沙口石冻马蹄脱"，转向对战争场面和环境的描绘，展现军威和战争残酷。最后两句"古来青史谁不见，今见功名胜古人"作为合，总结全诗，赞美统帅功绩。

在层次安排上，诗歌可分为四个层次：第一层次（第1-2句）描绘战争背景，通过"夜吹角"、"旄头落"等意象营造紧张氛围；第二层次（第3-4句）描述我军准备和统帅出征，展现军队井然有序；第三层次（第5-7句）刻画战争场面和严酷环境，既有"伐鼓雪海涌"、"大呼阴山动"的军威描写，又有"战场白骨缠草根"的残酷现实；第四层次（第8-9句）直接赞美封大夫的忠诚和功绩，表达崇敬之情。

诗歌结构上还体现出时空转换的特点，从夜晚到黎明，从轮台城头到西征战场，时空逐渐扩展。同时，诗人巧妙运用动静结合的手法，既有"夜吹角"、"伐鼓"、"大呼"等动态描写，又有"旄头落"、"烟尘黑"等静态刻画，使画面生动立体。虚实相生也是其结构特点，实写战争场面和环境，虚写功绩评价，增强了艺术感染力。

整体而言，这首诗结构严谨，层次分明，通过起承转合的章法和四个层次的安排，从战争背景到功绩赞美，层层递进，气势磅礴，充分展现了岑参边塞诗的艺术特色。
## ✍️ 语言风格
《轮台歌奉送封大夫出师西征》是岑参边塞诗的代表作，其语言风格特点鲜明，主要体现在以下几个方面：

一、雄浑豪放的风格特点
全诗气势磅礴，意境开阔，展现了唐代边塞诗特有的雄浑之美。诗人以宏大的视角描绘军队出征的壮阔场面，如"四边伐鼓雪海涌，三军大呼阴山动"一句，通过夸张手法表现军威之盛，气势恢宏。同时，诗中又不乏悲壮情怀，如"战场白骨缠草根"揭示了战争的残酷，形成了慷慨悲壮的艺术风格。

二、用词特点
1. 军事色彩浓厚：诗中大量使用军事词汇，如"角"、"羽书"、"单于"、"戍楼"、"汉兵"、"上将"、"大军"、"三军"、"虏塞"等，营造出浓郁的战争氛围。

2. 边塞地域特色鲜明：运用"轮台"、"渠黎"、"金山"、"剑河"、"沙口"等西域地名，以及"旄头"、"雪海"、"阴山"等具有边塞特色的自然景观名称，增强了诗歌的地域特色和真实感。

3. 动词简洁有力：诗中动词如"吹"、"落"、"过"、"望"、"屯"、"拥"、"征"、"行"、"伐"、"涌"、"呼"、"动"、"缠"、"急"、"阔"、"冻"、"脱"等，大多简练有力，增强了诗歌的动感和节奏感。

4. 意象生动鲜明：诗中的意象如"夜吹角"、"旄头落"、"烟尘黑"、"雪海涌"、"阴山动"、"白骨缠草根"、"风急雪片阔"、"石冻马蹄脱"等，都极为鲜明生动，给人留下深刻印象。

三、修辞手法
1. 对偶：诗中多处运用对偶手法，如"轮台城头夜吹角，轮台城北旄头落"、"四边伐鼓雪海涌，三军大呼阴山动"、"剑河风急雪片阔，沙口石冻马蹄脱"等，整齐对称，增强了诗歌的节奏感和气势。

2. 夸张：运用夸张手法增强表现力，如"三军大呼阴山动"夸张地表现了军威之盛；"雪海涌"将雪比作海，也是夸张手法，增强了诗歌的艺术感染力。

3. 比喻：如"烟尘黑"比喻战争带来的尘土飞扬，"雪海涌"将雪地比作翻涌的大海，"白骨缠草根"形象地比喻战场的惨状，使诗歌更加生动形象。

4. 衬托：通过环境的严酷衬托将士的英勇，如"剑河风急雪片阔，沙口石冻马蹄脱"描写了西域的严寒环境，衬托出将士们不畏艰险的精神。

5. 对比：诗中存在多处对比，如"古来青史谁不见，今见功名胜古人"中古今对比，突出当代功业的辉煌；战争场面的壮丽与残酷的对比等，增强了诗歌的思想深度。

6. 象征：如"旄头落"象征敌军败亡的征兆，"烟尘黑"象征战争的临近和残酷，增强了诗歌的象征意义。

7. 借代：用"汉兵"借指唐朝军队，用"单于"借指敌军首领，都是典型的借代手法，使诗歌表达更加简洁有力。

岑参的这首边塞诗通过雄浑豪放的语言风格，富有军事色彩和边塞特色的用词，以及多种修辞手法的灵活运用，成功塑造了一幅气势恢宏的边塞出征图，展现了盛唐时期边塞诗的典型风格特征，也体现了诗人对国家统一的关切和对将士英勇精神的赞美。
## 📚 文化札记
《轮台歌奉送封大夫出师西征》是唐代著名边塞诗人岑参的代表作，创作于唐玄宗天宝年间。当时岑参在安西都护府任职，而封常清（即诗中的"封大夫"）受命出征西域，岑参写下此诗为他送行。

诗中"轮台"指古代西域地名，在今新疆轮台县一带，是唐代安西都护府辖区内的重要军事据点。"角"是古代军中吹奏的乐器，用于发出信号或鼓舞士气。"旄头"指北斗七星中的第六星和第七星，古代星象学认为旄星坠落预示着战争的胜利，此处暗示唐军将取胜。

"羽书"是古代紧急军情文书，因插有羽毛以示紧急而得名。"渠黎"为古代西域地名，在今新疆库尔勒市一带。"单于"本是匈奴君主的称号，这里泛指北方游牧民族的首领。"金山"可能指阿尔泰山，古代称为金山，位于今新疆北部和蒙古国西部。

"戍楼"指边防哨所的瞭望楼。诗中"汉兵"实指唐朝军队，唐代诗人常以"汉"代指唐。"上将"指封常清，"拥旄"意为持旌节，古代高级将领出征时持旌节作为权力的象征。

"雪海"可能指天山山脉中的雪峰或积雪覆盖的高原。"三军"古代指左、中、右三军，泛指全部军队。"阴山"是位于今内蒙古自治区中部的重要山脉，古代为北方军事防线。

"剑河"为西域河流名，具体位置不详，可能在今新疆境内。"沙口"是地名，应在西域沙漠地区。"石冻马蹄脱"形容天气极其寒冷，石头都冻得坚硬，马蹄都可能脱落，极言边塞环境的恶劣。

"亚相"是对封常清的尊称，"勤王"意为为君主效力，保卫国家。"静边尘"指平定边疆的战乱。"青史"指史书，古代用竹简记录历史，竹简呈青色，故称史书为"青史"。

封常清是唐朝名将，唐玄宗时期曾任安西副都护、北庭都护等职，多次率军出征西域，战功卓著。天宝十四年（755年），安史之乱爆发，封常清被任命为范阳、平卢节度使，率军抵抗安禄山叛军，但因寡不敌众而战败，后被唐玄宗处死。

此诗真实反映了唐代与西域的关系以及唐代边塞战争的面貌。唐代是中国历史上对西域控制力最强的朝代之一，设立了安西都护府和北庭都护府，管辖西域广大地区。诗中描绘的战争场景和边塞环境，展现了唐代边塞诗的雄浑豪放风格，也体现了诗人对国家统一的关注和对将士的赞美。
## 🌅 创作背景
《轮台歌奉送封大夫出师西征》创作于唐玄宗天宝年间（约751-754年），地点在唐代西域的轮台地区，即今天的新疆轮台县一带。当时岑参正担任安西节度使高仙芝的幕僚，随军驻守西域，这是他第一次出塞的经历。

岑参此时年约三十多岁，正值壮年。作为幕府文人，他亲历了边塞生活的艰辛与残酷，目睹了边疆战争的惨烈。这一时期，他既感受到了边疆的壮阔风光，也体验到了战争的血腥无情。在艰苦环境中，他的心境复杂而矛盾：一方面对战争的残酷有清醒认识，另一方面又对将士们的英勇精神和唐朝开疆拓土的壮举充满敬意。

天宝年间是唐朝由盛转衰的关键时期。唐玄宗统治前期国力鼎盛，但危机已逐渐显现。在西北边疆，唐朝与吐蕃、大食（阿拉伯帝国）等强国争夺西域控制权。751年，唐军与大食在怛罗斯之战中失利，边疆局势紧张。封大夫（据考证可能为封常清）率军西征，正是这一系列边疆冲突的延续。唐朝在西域设立安西、北庭等都护府，屯驻重兵，维护丝绸之路畅通和边疆安全，这种军事实力和扩张政策为边塞诗的繁荣提供了社会背景。

这首诗的创作动机主要是为送别封大夫西征而作。岑参作为亲历边塞生活的诗人，对边疆战争、自然环境和士兵生活有着深刻了解，这些都成为他创作的丰富素材。他希望通过诗歌表达对封大夫西征的支持和对唐朝开疆拓土事业的赞美。诗中既描写了战争的残酷场面，也歌颂了将士们的英勇精神，反映了当时社会对功名的追求态度和对边疆英雄的崇敬之情。
## 💭 情感基调
《轮台歌奉送封大夫出师西征》的情感基调和氛围特点可概括为雄浑悲壮、紧张激烈、崇高豪迈。诗中通过"四边伐鼓雪海涌，三军大呼阴山动"等夸张手法展现军威浩大的雄浑气势；通过"战场白骨缠草根"和"剑河风急雪片阔，沙口石冻马蹄脱"等描写战争残酷与边地艰苦，营造悲壮苍凉之感；从"夜吹角"到"单于已在金山西"的紧急军情，形成紧张激烈的氛围；最后对封大夫"今见功名胜古人"的赞美，体现了崇高豪迈的英雄气概和报国情怀，整体展现了唐代边塞诗特有的豪放风格与悲悯情怀的融合。
## 🔍 赏析
《轮台歌奉送封大夫出师西征》是岑参边塞诗的代表作，通过雄浑壮阔的战争场景和鲜明的意象，展现了唐代将士出征西域的壮烈画卷。

在意境营造和艺术手法上，诗人巧妙运用时空转换，从夜晚的军号到天亮的行军，从轮台城到金山、阴山，构建了一幅宏大的战争图景。"四边伐鼓雪海涌，三军大呼阴山动"采用夸张手法，表现军威之盛；"战场白骨缠草根"则以具体意象展现战争残酷；"剑河风急雪片阔，沙口石冻马蹄脱"通过对恶劣自然环境的描写，暗示战争的艰难。这些艺术手法共同营造出一种雄浑悲壮的边塞战争意境。

语言特色方面，全诗气势磅礴，语言雄浑豪放，具有鲜明的边塞诗特色。诗人多用对仗，如"轮台城头夜吹角，轮台城北旄头落"，增强了诗歌的节奏感和音乐美。同时，善用夸张修辞，如"雪海涌"、"阴山动"等，表现战争的声势；运用象征手法，如"旄头落"象征军队出发，"烟尘黑"象征敌军来袭，使诗歌意蕴丰富。

情感表达上，诗人既表达了对封大夫出征的送别之情，也有对战争残酷的忧虑，更有对将士英勇抗敌的赞美。诗的主题思想是歌颂将士保家卫国的英勇精神，表达对和平的向往。结尾"古来青史谁不见，今见功名胜古人"更是表达了对封大夫建功立业的期许和赞美，体现了诗人对国家安危的深切关怀。

作为唐代边塞诗的代表作，这首诗具有重要文学价值。其雄浑豪放的风格、生动的意象描绘、丰富的修辞手法，对后世边塞诗创作产生深远影响。同时，诗中表现的爱国主义精神和英雄气概，也具有积极的教育意义，成为中华民族精神文化的重要组成部分。
