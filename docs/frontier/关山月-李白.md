## 📜 原文
```
明月出天山，苍茫云海间。
长风几万里，吹度玉门关。
汉下白登道，胡窥青海湾。
由来征战地，不见有人还。
戍客望边邑，思归多苦颜。
高楼当此夜，叹息未应闲。
```
## 🎯 主题
李白《关山月》主要表达了边塞征战的悲苦与征人的思乡之情。诗以壮阔的边塞景色开篇，"天山"、"云海"、"长风"等意象勾勒出苍茫辽阔的边地风光；中间部分揭示战争的残酷本质，"由来征战地，不见有人还"直指战争的血腥无情；最后通过戍客"思归多苦颜"和家中妻子的"叹息"，展现了战乱中的离别之痛。整首诗以明月贯穿，既描绘了边塞的壮美，又抒发了对战争的厌恶和对和平生活的向往，体现了诗人深厚的人文关怀和对普通民众疾苦的同情。
## 🏗️ 结构
《关山月》是李白边塞诗的代表作，其结构特点体现了李白诗歌的严谨章法与巧妙构思。

从章法布局看，全诗采用传统"起承转合"的四段结构，每两句为一层，层次分明，结构工整。前两句"明月出天山，苍茫云海间"为起，以壮阔景象开篇；三四句"长风几万里，吹度玉门关"为承，进一步拓展空间；五六句"汉下白登道，胡窥青海湾。由来征战地，不见有人还"为转，转入历史回顾；末两句"戍客望边邑，思归多苦颜。高楼当此夜，叹息未应闲"为合，以人情收束全篇。

从层次安排看，全诗可分为三个递进层次：第一层次为前四句，描绘边塞自然景观，从天山到玉门关，空间跨度极大，营造出苍茫辽远的意境；第二层次为五六句，转入对边塞征战历史的回顾，点出边塞自古以来的悲剧命运；第三层次为末两句，将视角转向人的情感世界，表达戍边将士及其亲人的相思之苦。

在艺术手法上，诗歌运用了空间转换、时间跨越和情感递进等技巧。空间上从天山到玉门关，再到内地的"高楼"，视角不断变化；时间上从眼前景象追溯到汉朝征战历史，再回到当夜景况；情感上从景物描绘到历史思考，再到人事情感，层层深入，结构紧凑而富有张力。

特别值得注意的是，诗中自然景物、历史回顾和现实情感三者有机融合，形成了一个完整的艺术整体。首尾呼应，以明月起兴，以叹息收束，结构严谨，意境深远，充分体现了李白高超的艺术构思能力。
## ✍️ 语言风格
《关山月》是李白边塞诗的代表作，其语言风格具有鲜明的个人特色和时代特征。

从语言风格看，此诗雄浑壮阔，气势恢宏。开篇"明月出天山，苍茫云海间"便以宏大的视野拉开边塞夜景的帷幕，"长风几万里，吹度玉门关"更以夸张的笔法拓展了空间感，展现出盛唐气象的壮美。同时，诗歌语言又具有清新自然的特点，虽写边塞，但不用艰深晦涩之词，而是以流畅自然的语言描绘景象，读来朗朗上口。后半部分"戍客望边邑，思归多苦颜"等句情感真挚，语言朴实却直击人心，体现了李白诗歌中的真情实感。

在用词方面，诗人选用的意象极具边塞特色，如"明月"、"天山"、"云海"、"玉门关"等，这些词语的选用使诗歌画面感极强。对比性词语的运用也很突出，"汉下白登道，胡窥青海湾"中的"汉"与"胡"形成鲜明对比，一南一北，构成对峙之势。诗中动词的选用精准有力，如"出"、"吹度"、"下"、"窥"、"望"、"思归"、"叹息"等，准确传神地表达了动作和情感。

修辞手法上，诗歌运用了多种技巧。"长风几万里"是典型的夸张手法，虽不合常理却极好地表现了边塞的辽阔和风势的强劲，体现了李白诗歌中常见的浪漫主义色彩。"汉下白登道，胡窥青海湾"形成了工整的对偶，增强了诗歌的节奏感。借景抒情也是此诗的重要特点，前四句描绘边塞夜景，后四句转入对戍边士兵的描写，通过景与情的对比，抒发了对战乱的不满和对士兵的同情。此外，诗中的"明月"不仅是自然景物，还可能象征着和平与团圆，与后文戍边士兵的思归之情形成呼应，增强了诗歌的象征意义。

总体而言，《关山月》语言雄浑壮阔又清新自然，用词精准鲜明，修辞手法丰富多样，既展现了边塞的壮美景象，又表达了对战争的思考和对戍边士兵的同情，体现了李白诗歌的独特艺术魅力和深厚人文关怀。
## 📚 文化札记
《关山月》是李白创作的边塞诗代表作，其中蕴含丰富的历史文化元素，现对其中的重要典故、地名等作如下注释：

"天山"在诗中指的是祁连山，位于今甘肃和青海交界处，是唐代西北边疆的重要自然屏障。祁连山在古代文献中常被称为"南山"或"天山"，与今天新疆的天山不同。

"玉门关"是古代西北边疆的重要关隘，位于今甘肃省敦煌市西北，是丝绸之路上的必经之地，也是唐代边防体系中的重要一环。玉门关因其特殊的地理位置，在古代诗词中常作为边塞征战的象征。

"汉下白登道"引用了汉代著名的历史典故"白登之围"。公元前200年，汉高祖刘邦率军亲征匈奴，在白登山（今山西大同市东）被匈奴军队围困七天七夜，最终以重金贿赂匈奴单于之妻才得以脱险。李白借汉代历史影射唐代边疆战事。

"胡窥青海湾"中，"胡"是古代对北方和西北少数民族（如匈奴、突厥、吐蕃等）的泛称。"青海湾"指青海湖地区，唐代时这里是唐朝与吐蕃争夺的战略要地，战争频仍。唐玄宗时期曾多次在青海湖一带与吐蕃发生大规模军事冲突。

"戍客"指驻守边疆的士兵，唐代实行府兵制，士兵需长期戍边，远离家乡。"高楼"在古代诗词中常用来指代思妇所居之处，这里暗指远在故乡的军人妻子或亲人。

整首诗通过描绘边关壮阔的自然景观与厚重的历史背景，表现了战争的残酷与戍边士兵的思乡之情，同时也反映了战争给普通家庭带来的痛苦。诗中融入的历史典故和地理知识，展现了李白对边塞问题的深刻关注和人文关怀。
## 🌅 创作背景
《关山月》创作于唐玄宗天宝年间，约公元746年左右，此时正值李白第二次漫游时期。地点可能在长安或西北边塞地区，诗中提到的"天山"、"玉门关"、"青海湾"等地理特征，表明李白对西北边防有深入了解，或亲历边塞，或基于对边塞战争的想象而作。

当时李白刚经历了长安政治生活的失意，天宝三年(744年)离开长安后，他开始四处游历。这一时期，李白虽然失去了政治上的机会，但创作了大量优秀诗篇。他既怀有对政治理想的失落，又饱含对国家命运的关切，同时保持着豪放不羁的个性和对自由的向往。这种复杂的心理状态反映在他的边塞诗创作中。

历史背景方面，天宝年间是唐朝由盛转衰的关键时期。表面上唐朝仍然繁荣昌盛，但实际上已暗藏危机。唐玄宗后期沉溺享乐，朝政被杨国忠、李林甫等奸臣把持。同时，唐朝与吐蕃、突厥等周边民族在西北地区的冲突频繁，形成长期军事对峙。"汉下白登道，胡窥青海湾"正反映了这种紧张局势。唐朝实行府兵制和募兵制相结合，大量士兵被派往边关长期戍守，导致"由来征战地，不见有人还"的悲惨局面。

李白的创作动机多源于对国家命运的关切和对士兵疾苦的同情。他受到当时边塞诗流行的影响，如高适、岑参等人的创作，同时结合自身游历经历和对边塞风物的观察。"关山月"作为乐府旧题，本身就有描写边塞、思归的传统，李白在此基础上进行了艺术创新，通过壮阔的自然景象和深沉的人文关怀，表达了对战争的反思和对和平的向往。
## 💭 情感基调
李白《关山月》的情感基调可概括为壮阔苍凉、悲怆哀伤，氛围特点表现为边塞辽阔与思乡愁绪的交织。诗歌以天山明月、万里长风开篇，构建出苍茫壮阔的边塞画卷，随后笔锋转向，描绘"不见有人还"的战争残酷，流露出深沉的悲怆感。后半部分通过"戍客望边邑"与"高楼叹息"两相对应，强化了思乡愁绪和离别之痛。全诗在雄浑壮美的景象中渗透着历史沧桑与人情悲凉，形成独特的苍凉美感，既有对边塞战事的冷静审视，也有对戍边将士的人文关怀，展现了李白边塞诗特有的壮美与哀婉并存的艺术风格。
## 🔍 赏析
《关山月》是李白边塞诗的代表作，以其雄浑壮阔的意境和深刻的人文关怀展现了唐代边塞诗的最高成就。

在意境营造上，诗人以"明月出天山，苍茫云海间"开篇，勾勒出一幅宏大辽远的边塞夜景，天山明月与苍茫云海交相辉映，营造出壮阔苍凉的意境。"长风几万里，吹度玉门关"进一步拓展了空间感，几万里长风吹过玉门关，使读者感受到边塞的辽阔与荒凉。李白巧妙运用对比手法，将壮美的自然景观与残酷的战争现实形成鲜明对比，"汉下白登道，胡窥青海湾"描述军事对峙，而"由来征战地，不见有人还"则直指战争本质，增强了诗歌的深刻性。

语言特色方面，全诗简洁有力，气势磅礴。李白善用简练语言表达丰富内容，如"几万里"一词既表现空间广阔，又暗示时间久远。修辞上，诗人运用夸张手法增强气势，如"长风几万里"；运用对偶如"汉下白登道，胡窥青海湾"，使诗歌节奏鲜明，音韵和谐。诗中"明月"、"长风"、"玉门关"等意象都具有丰富的文化内涵，使诗歌含蓄隽永。

情感表达上，诗人对战争的厌恶和对和平的向往贯穿全诗。"由来征战地，不见有人还"揭示战争残酷，"戍客望边邑，思归多苦颜"描绘戍边将士思乡之苦，"高楼当此夜，叹息未应闲"则以家中人的叹息收束，形成边关与内地的情感呼应，使诗歌情感更加丰富立体。

《关山月》的文学价值在于它将边塞诗从单纯的描绘风光提升到对战争与和平、人性与命运的深刻思考，对后世边塞诗创作产生深远影响。同时，诗中提到的地名和历史事件也为研究唐代边塞政策和民族关系提供了重要史料，具有很高的历史文化价值。这首诗充分体现了李白"气势磅礴、想象丰富、情感真挚"的艺术特色，是中国古典诗歌中的瑰宝。
