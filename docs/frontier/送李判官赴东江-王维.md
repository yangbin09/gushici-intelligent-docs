## 📜 原文
```
远树带行客，孤城当落晖。
吾谋适不用，勿谓知音稀。
```
## 🎯 主题
《送李判官赴东江》是一首充满情感的送别诗。王维通过"远树带行客，孤城当落晖"的凄清景象，营造出离别时的伤感氛围，表达了对友人远行的不舍之情。诗的后两句"吾谋适不用，勿谓知音稀"则转向对李判官的慰藉和鼓励，暗示李判官可能仕途不顺，但诗人告诉他不要感到孤独，因为仍有自己这样的知音存在。整首诗既是对友人的送别，也是对失意友人的精神支持，体现了诗人与李判官之间深厚的知己之情。这首诗的主题可以概括为：在离别之际，面对友人仕途不顺，诗人表达不舍之情的同时，更给予了真诚的慰藉和坚定的友谊支持。
## 🏗️ 结构
王维的《送李判官赴东江》是一首五言绝句，虽然篇幅短小，但结构精巧，层次分明。从章法布局和层次安排来看，这首诗体现了以下特点：

在章法布局上，诗歌遵循了五言绝句典型的"起承转合"结构。首句"远树带行客"为起句，点明送别场景；第二句"孤城当落晖"为承句，进一步描绘环境，承接首句的意境；第三句"吾谋适不用"为转句，从景物描写转向抒情议论；末句"勿谓知音稀"为合句，收束全诗，表达对友人的劝慰和期望。这种章法安排使诗歌结构严谨，层次清晰。

从层次安排看，全诗可分为两个层次：前两句为写景层次，后两句为抒情层次。第一层次中，诗人通过"远树"、"行客"、"孤城"、"落晖"等意象，勾勒出一幅凄美的送别图景，营造出孤寂、凄凉的氛围，为后面的抒情做铺垫。第二层次中，诗人从景物描写转向直接表达情感和思想，先是表达自己的政治抱负未能实现的遗憾，然后劝慰友人不要认为知音稀少，暗示了他们之间的深厚友谊。

这首诗在结构上还体现了情景交融的特点。前两句的写景与后两句的抒情在情感基调上形成呼应，都透露出一种淡淡的忧伤和无奈。同时，诗歌采用了由景入情的结构安排，使情感表达更加自然流畅，增强了诗歌的感染力。

此外，作为五言绝句，这首诗结构简洁凝练，仅用二十个字就勾勒出完整的送别图景，并表达了丰富的情感和思想，体现了王维诗歌"诗中有画"的艺术特色。诗中运用的对比手法，如"远树"与"行客"、"孤城"与"落晖"之间的对比，也增强了诗歌的艺术效果。
## ✍️ 语言风格
王维的《送李判官赴东江》体现了其典型的诗歌语言风格，具有清新淡远、含蓄自然的艺术特色。在用词方面，诗人极为精炼，"远树"、"行客"、"孤城"、"落晖"等词语简练却意境深远，尤其是"带"字用得极为精妙，既表示远树陪伴行客，又有牵引连绵之感；"当"字则准确描绘了孤城正对落日余晖的画面感，体现了王维"诗中有画"的艺术特色。

修辞手法上，首联"远树带行客，孤城当落晖"形成了工整的对仗，"远树"对"孤城"，"行客"对"落晖"，结构整齐，音韵和谐。全诗运用借景抒情手法，通过描绘萧瑟的秋景表达离别之情，情景交融。尾联"勿谓知音稀"虽非直接反问，但含有劝慰和反问的意味，表达了对友人的鼓励。

在意象运用上，诗人选取了"远树"、"行客"、"孤城"、"落晖"等典型意象，营造出一种既萧瑟又温暖的送别氛围。"远树"象征距离和离别，"孤城"暗示孤独寂寥，"落晖"则象征时光流逝和人生阶段的结束，这些意象组合构成了一幅意境深远的送别图景。

情感表达上，这首诗含蓄而深沉。首联通过景物描写委婉表达离别之情，尾联则直接抒发情感，"吾谋适不用"表达了诗人政治抱负未能施展的遗憾，"勿谓知音稀"则是对友人的鼓励和安慰，整体情感基调是淡淡的忧伤中带着温暖和鼓励，体现了王维诗歌中常见的超脱和豁达。

总体而言，这首诗语言朴素自然，不事雕琢，却意蕴丰富；结构严谨，前后呼应；情景交融，景中有情，情中有景；情感表达不直露，而是通过意象和暗示来传达，展现了王维作为"诗佛"的独特艺术魅力。
## 📚 文化札记
《送李判官赴东江》是唐代诗人王维创作的一首送别诗，虽然只有短短四句，但蕴含丰富的文化内涵和情感表达。

关于作者王维：王维（701年-761年），字摩诘，号摩诘居士，唐代著名诗人、画家、音乐家，有"诗佛"之称。他历仕唐玄宗、肃宗、代宗三朝，官至尚书右丞，故世称"王右丞"。王维的诗以山水田园见长，风格清新淡远，意境优美，被誉为"诗中有画，画中有诗"。

题目中的"判官"是唐代官职名，是节度使、观察使等地方军政长官的属官，负责处理文书、参谋军事等事务。"东江"的具体位置尚有争议，多数学者认为指长江下游地区，可能是今江苏、安徽一带的长江流域，也有人认为是今江西境内的某条江流。这首诗是王维为即将前往东江任职的李姓判官送别所作。

"远树带行客，孤城当落晖"两句描绘了一幅优美的送别画面。远处的树木伴随着远行的客人，孤城沐浴在夕阳的余晖中。这种景物描写体现了王维"诗中有画"的艺术特色。"孤城"可能指的是长安或当时王维与李判官分别的城市；"落晖"（夕阳）在中国古典诗词中常用来象征离别、暮年或时代的变迁，这里既点明了分别的时间，也暗示了离别的伤感。

"吾谋适不用，勿谓知音稀"两句则转入抒情，表达了复杂的情感。"吾谋适不用"反映了王维的政治处境。王维虽然在官场有一定地位，但在安史之乱期间曾被叛军俘虏，被迫接受伪职，虽然后来被赦免，但政治上可能并不如意。这里的"谋"可能指他的政治建议或抱负未能得到采纳。

"勿谓知音稀"引用了著名的"知音"典故。此典出自《列子·汤问》中伯牙与钟子期的故事。伯牙善鼓琴，钟子期善听。伯牙鼓琴，志在高山，钟子期曰："善哉，峨峨兮若泰山！"志在流水，钟子期曰："善哉，洋洋兮若江河！"伯牙所念，钟子期必得之。后来钟子期死，伯牙破琴绝弦，终身不复鼓琴，以为世无足复为鼓琴者。后世用"知音"比喻了解自己、心意相通的朋友。王维在这里告诉李判官，虽然自己的建议未被采纳，但不要因此认为真正的知音难觅，暗示他们之间的友谊是真挚的。

这首诗的创作时间没有明确记载，但根据诗中流露的情感，可能创作于安史之乱（755年-763年）之后。安史之乱是唐朝由盛转衰的重要转折点，对唐朝的政治、经济、文化都产生了深远影响。王维在乱中的特殊经历可能对他的心理和政治生涯产生了影响，诗中"吾谋适不用"可能反映了他在这段特殊历史时期的个人感受。

作为一首送别诗，它继承了中国古典诗词中送别诗的传统，通过景物描写和情感表达，展现了离别的场景和复杂的心绪，同时也体现了王维诗歌语言简洁、意境深远、情景交融的艺术特色。
## 🌅 创作背景
《送李判官赴东江》创作于安史之乱后，约在唐肃宗至德年间（756-758年）或稍晚时期。当时王维因在安史之乱中被俘并被迫接受伪职，虽后来得到赦免，但政治生涯已受到严重影响，被贬为太子中允，后迁任尚书右丞。创作地点可能在长安或其任职之地。

此时的王维生活境遇颇为复杂。一方面，他虽重新获得官职，但因曾接受伪职的经历而内心充满愧疚和自责；另一方面，他对政治逐渐失去热情，转而潜心佛学，追求精神上的超脱。诗中"吾谋适不用"一句正反映了他在政治上无法施展抱负的无奈心境，而"勿谓知音稀"则体现了他对友情的珍视及对友人的真挚劝慰。

安史之乱是唐朝由盛转衰的转折点，战乱后的唐朝社会动荡不安，藩镇割据现象日益严重，朝廷权威下降。在这样的历史背景下，许多文人官员都面临着政治上的困境和抉择。王维作为一位有才华的文人，也不可避免地卷入这场政治风波，其诗作往往流露出对人生无常的感慨和对友情的珍视。

这首诗的创作动机源于送别友人李判官赴东江任职。在送别的场合，王维不仅表达了对友人的不舍和祝福，也借机抒发了自己政治上的失意之情。"远树带行客，孤城当落晖"的景象源于实际的送别场景，以简洁而富有意境的语言描绘了离别时的凄美画面；而"吾谋适不用，勿谓知音稀"则反映了他内心的感慨，既是对友人的安慰，也是对自己的勉励。王维通过这首诗，既表达了对友人的深情厚谊，也抒发了自己政治抱负无法实现的苦闷，体现了他作为"诗佛"的深刻洞察力和人文关怀。
## 💭 情感基调
《送李判官赴东江》的情感基调以伤感失意为主，却又蕴含豁达慰藉。诗中"远树带行客，孤城当落晖"营造出苍凉孤寂的氛围，通过远树、行客、孤城、落晖等意象勾勒出一幅离别图景，色调黯淡。后两句"吾谋适不用，勿谓知音稀"则流露出政治抱负不得施展的无奈，同时转而劝慰友人不要因离别而感到孤独知音难觅，为整体氛围增添了温暖与希望，形成伤感中见豁达、孤寂中有慰藉的复杂情感层次。
## 🔍 赏析
《送李判官赴东江》是王维的一首五言绝句，虽仅二十字，却意蕴深远，展现了盛唐诗歌的艺术魅力。

在意境营造上，诗人以"远树带行客，孤城当落晖"勾勒出一幅壮阔而凄美的离别图景。"带"字用得极为精妙，既表现了行客渐行渐远的动态，又暗示了树木与行客之间的某种情感联系，仿佛树木在挽留远行的友人。"孤城当落晖"则通过"孤"与"落"二字，既描绘了城池在夕阳中的景象，又暗喻了诗人内心的孤独与时光流逝的感慨。这种情景交融的艺术手法，使得自然景物与人物情感浑然一体，意境开阔而又深沉。

语言特色方面，王维以简洁精炼的语言表达复杂情感，体现了"诗中有画"的艺术特色。"吾谋适不用，勿谓知音稀"运用对偶和对比手法，前者表达政治抱负未能实现的失落，后者则是对友人的劝慰与鼓励。这种语言上的凝练与情感上的丰富形成鲜明对比，展现了王维诗歌的独特魅力。

情感表达上，这首诗超越了普通送别诗的范畴，既有对友人的不舍之情，又有对自身处境的深刻反思。"吾谋适不用"流露出诗人仕途失意的无奈，而"勿谓知音稀"则显示了他豁达的人生态度对友人的关怀。这种复杂而真实的情感表达，使得诗歌具有了更为丰富的人生哲理。

从文学价值看，这首诗不仅代表了王维诗歌的典型风格，也在中国文学史上产生了深远影响。其情景交融的艺术手法、简洁精炼的语言风格，以及深刻的人生感悟，都成为后世诗人学习的典范。尤其是"远树带行客，孤城当落晖"这样的名句，更是被后人广泛引用和模仿，体现了其持久的艺术生命力。
