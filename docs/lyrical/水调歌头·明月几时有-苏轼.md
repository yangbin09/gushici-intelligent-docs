## 📜 原文
```
明月几时有？把酒问青天。
不知天上宫阙，今夕是何年。
我欲乘风归去，又恐琼楼玉宇，高处不胜寒。
起舞弄清影，何似在人间。
转朱阁，低绮户，照无眠。
不应有恨，何事长向别时圆？
人有悲欢离合，月有阴晴圆缺，此事古难全。
但愿人长久，千里共婵娟。
```
## 🎯 主题
《水调歌头·明月几时有》的主要主题思想可概括为对人生无常的哲学思考与对亲情团圆的深切思念。苏轼借月抒怀，以月亮的阴晴圆缺比喻人生的悲欢离合，表达了对世事难全的豁达认知。词中既有"我欲乘风归去"的超然向往，又有"何似在人间"的现实眷顾，体现了作者理想与现实间的矛盾与调和。最终以"但愿人长久，千里共婵娟"收束，表达了对亲人健康长寿的美好祝愿，以及即使相隔千里也能共享明月的情感寄托，展现了中国传统文化中超越时空的精神共鸣和豁达乐观的人生态度。
## 🏗️ 结构
《水调歌头·明月几时有》在结构上展现出严谨的章法布局和清晰的层次安排，体现了苏轼高超的艺术构思能力。

从整体结构看，这首词分为上片和下片两部分，每部分又包含四个层次，形成了起承转合的完整结构。上片主要写作者对明月的疑问和想象，以及向往天上与留恋人间的矛盾心理；下片则描写月亮运行变化，引申出对人生哲理的思考和对亲人的美好祝愿。

上片第一层次以"明月几时有？把酒问青天"开篇，直接设问明月，与自然对话，奠定全词浪漫基调。第二层次"不知天上宫阙，今夕是何年"进一步展开想象，将思绪引向天界，表现对天上世界的好奇。第三层次"我欲乘风归去，又恐琼楼玉宇，高处不胜寒"揭示内心矛盾：一方面向往天上仙境，一方面担忧天上寒冷孤寂。第四层次"起舞弄清影，何似在人间"作为上片小结，通过对人间生活的赞美，表达对现实生活的热爱。

下片第一层次"转朱阁，低绮户，照无眠"描写月亮移动轨迹，展现时间流逝，暗示作者因思念亲人而失眠。第二层次"不应有恨，何事长向别时圆"对明月提出质疑，既是对自然现象的疑问，也是对人生际遇的感慨。第三层次"人有悲欢离合，月有阴晴圆缺，此事古难全"是全词哲理核心，通过对比人的悲欢离合与月的阴晴圆缺，表达对人生无常、世事难全的深刻认识。第四层次"但愿人长久，千里共婵娟"作为结尾，表达对亲人的美好祝愿，情感得到升华。

在章法布局上，这首词呈现出以下特点：一是起承转合结构严谨，上片以设问起，承接以想象和矛盾心理，转到对人间的肯定；下片以月亮移动起，承接以质疑明月，转到哲理思考，最后以祝愿作结。二是情景交融，既有对自然景象的描写，又有情感的表达。三是虚实结合，既有实写如把酒问青天，又有虚写如天上宫阙。四是对比手法运用巧妙，如天上与人间的对比、悲欢离合与阴晴圆缺的对比等。五是层次递进明显，情感从好奇、向往、矛盾、留恋，到疑问、感慨、思考、祝愿，层层深入。

这种结构安排使全词在形式上严谨有序，在内容上丰富多彩，在情感上真挚深沉，在思想上深刻隽永，充分展现了苏轼高超的文学技巧和豁达的人生态度。
## ✍️ 语言风格
苏轼的《水调歌头·明月几时有》语言风格清新自然，意境深远，体现了宋词"以文为词"的特点和苏轼独特的艺术风格。

在用词方面，这首词平易近人却不失典雅。如"明月几时有"、"把酒问青天"等句，用词简单直接却意境深远。"琼楼玉宇"、"清影"、"朱阁"、"绮户"等意象既富有美感又不显雕琢，"婵娟"一词含蓄地指代月亮，体现了作者用词的准确与精妙。整体用词既有文人雅致，又不失通俗流畅，形成了苏轼特有的"清丽"风格。

修辞手法上，拟人是最突出的特点。作者将月亮人格化，如"不应有恨，何事长向别时圆"，直接质问月亮为何总是在人们离别时才圆，赋予月亮人的情感。比喻手法在"人有悲欢离合，月有阴晴圆缺"中运用得极为精妙，将人的情感变化与月亮的阴晴圆缺相比，既形象又深刻。此外，设问如开篇"明月几时有？"和对偶如"人有悲欢离合，月有阴晴圆缺"等修辞手法的运用，增强了词的艺术表现力和音乐美。

句式特点上，长短句结合使节奏富有变化。词中既有短促有力的"明月几时有"，也有舒缓悠长的"不知天上宫阙，今夕是何年"。问句与陈述句交错使用，形成一种对话感，如"何事长向别时圆？"与"此事古难全"的呼应，增强了词的感染力。句式灵活多变，从直抒胸臆到动态描写，再到哲理思考，展现了作者丰富的情感层次。

意境营造上，虚实结合是显著特点。词中既有"把酒问青天"、"转朱阁，低绮户"等现实场景，又有"天上宫阙"、"琼楼玉宇"等想象中的仙境。作者将自己的情感与月亮的景象紧密结合，如"照无眠"既是月光照耀的实景，又是作者因思念而失眠的情感写照，达到了情景交融的艺术效果。

总体而言，这首词语言风格清新自然，不事雕琢；情感真挚，意境深远；理趣结合，哲理性强；豪放与婉约并存。"我欲乘风归去"展现了豪放的气概，而"但愿人长久，千里共婵娟"则体现了婉约的情感。这种风格特点使这首词成为千古传诵的佳作，也体现了苏轼作为"豪放派"代表词人的艺术成就。
## 📚 文化札记
《水调歌头·明月几时有》是苏轼于宋神宗熙宁九年（1076年）中秋所作，当时他在密州（今山东诸城）任知州。这首词以月亮为中心意象，表达了作者对人生哲理的思考和对亲人的思念之情。

"明月几时有？把酒问青天"开篇点明时间背景是中秋之夜，诗人面对明月，举杯向天发问。这里化用了李白《把酒问月》中"青天有月来几时？我今停杯一问之"的意境。

"不知天上宫阙，今夕是何年"中的"天上宫阙"指的是月宫，即嫦娥所居之处。"今夕是何年"表达了对天上时间与人间时间不同的思考，暗示了作者对超然世外的向往。

"我欲乘风归去，又恐琼楼玉宇，高处不胜寒"中，"乘风归去"暗示了作者想要像仙人一样乘风而去，"归去"一词表明作者似乎认为自己原本来自天上，有一种"谪仙人"的自觉。"琼楼玉宇"指的是月中华丽的宫殿，出自《淮南子》中关于月宫的描述。"高处不胜寒"既是对月宫寒冷环境的描述，也暗喻了官场高处不胜寒的政治处境。苏轼当时因反对王安石变法而被贬谪，有政治失意之感。

"起舞弄清影，何似在人间"化用了李白《月下独酌》中"我歌月徘徊，我舞影零乱"的意境，表达了诗人孤独寂寞的情感。"何似在人间"表明诗人虽然向往天上，但最终还是觉得人间更为亲切。

"转朱阁，低绮户，照无眠"中，"朱阁"和"绮户"都是指装饰华丽的楼阁和门窗，描绘了月光移动的景象。"照无眠"表明诗人因思念亲人而无法入眠。

"不应有恨，何事长向别时圆"是对月亮的质问，表达了诗人对月亮在亲人离别时反而圆满的疑惑。这种情感源于中国传统中月圆象征团圆的文化心理。

"人有悲欢离合，月有阴晴圆缺，此事古难全"是全词的哲理核心，表达了诗人对人生无常的感悟和豁达态度。

"但愿人长久，千里共婵娟"中，"婵娟"是月亮的美称，出自南朝谢庄《月赋》："美人迈兮音尘阙，隔千里兮共明月。"意思是即使相隔千里，也能共赏同一轮明月。

这首词的创作背景是苏轼思念远在异地的弟弟苏辙（字子由）。苏轼与苏辙感情深厚，常以诗词唱和。此时兄弟二人因各自任职而相隔千里，不能团聚，苏轼因此写下了这首词。从文化背景来看，这首词融合了中国传统的月亮崇拜、中秋团圆的文化心理以及道教神仙思想，体现了苏轼融合儒、道、佛三家思想的独特人生哲学。
## 🌅 创作背景
《水调歌头·明月几时有》创作于宋神宗熙宁九年（1076年）中秋节，地点在密州（今山东诸城）。当时苏轼任密州知州，已在此任职两年。这一时期，苏轼因反对王安石变法而被排挤出朝廷权力中心，先是被贬为杭州通判，后又调任密州知州。

在密州期间，苏轼的生活相对清贫，但政务之余仍有不少文学创作。此时，他的弟弟苏辙（字子由）在齐州（今山东济南）任职，兄弟二人相隔不远，却因公务繁忙，难以经常相聚。创作这首词时，正值中秋佳节，苏轼独自赏月，心中充满对远方弟弟的思念之情。同时，他因政治上的挫折，对人生有所感悟，表现出一种超然物外的人生态度。

北宋中期，王安石变法（熙宁变法）正在进行中，政治环境复杂，党争激烈。许多文人因政治立场不同而仕途起伏。社会方面，虽然北宋经济文化较为繁荣，但政治上的压抑使得许多文人心怀不满，转而在文学创作中寻求精神寄托。

这首词的创作直接动机是中秋赏月时对弟弟苏辙的思念。苏轼与苏辙感情深厚，两人自幼相伴，后又同科及第，但成年后因仕途奔波，聚少离多。中秋本是团圆之日，苏轼独自赏月，思念远方的弟弟，触景生情。灵感来源于对月亮的观察和对人生的思考。苏轼借月抒怀，把个人的离愁别绪与对宇宙人生的哲理思考结合起来，表达了对亲人的思念和对人生的豁达态度。词中"人有悲欢离合，月有阴晴圆缺，此事古难全"的哲理性思考，体现了苏轼面对人生挫折的达观心态和超越精神。
## 💭 情感基调
《水调歌头·明月几时有》的情感基调可概括为：思念、超脱、矛盾、豁达、哲理。词中既有对远方亲人的深切思念，又有对天上宫阙的向往与顾虑；既有对人生无常的感慨，又有对世事难全的豁达理解；既有个人情感的抒发，又有对宇宙人生的哲理思考。

氛围特点可概括为：清幽、孤独、超然、温情。全词以中秋明月为背景，营造出清幽寂静的夜晚氛围；作者独自把酒问天，透露出孤独感；对天上宫阙的想象和对人生哲理的思考，展现出超然物外的意境；而结尾"但愿人长久，千里共婵娟"的美好祝愿，又为全词注入了温情脉脉的色彩。
## 🔍 赏析
《水调歌头·明月几时有》是苏轼词作中的经典之作，展现了词人卓越的艺术才华和深刻的人生思考。在意境营造上，苏轼以明月为核心意象，构建了一个连接天上与人间的意境空间。开篇"明月几时有？把酒问青天"直接将读者带入仰望星空、思考宇宙的情境中，体现了词人豪迈的情怀和对天地的探询。词人巧妙运用虚实结合的手法，"天上宫阙"、"琼楼玉宇"是对天上的想象，是虚写；而"起舞弄清影"、"转朱阁，低绮户，照无眠"则是实写人间景象，使词作既有超脱尘世的想象，又有扎根现实的情感。对比手法的运用，如天上与人间的对比，悲欢离合与阴晴圆缺的对比，进一步丰富了词的艺术表现力。

语言特色方面，这首词清新自然又富有哲理意味。苏轼善于用平易近人的语言表达深刻的思想，如"人有悲欢离合，月有阴晴圆缺，此事古难全"看似简单，却包含深刻的人生哲理。修辞技巧上，词中运用了设问、对偶、排比等多种手法，开篇设问引出全词思考，中间对偶句工整而富有节奏感，排比句增强了语言的韵律美。拟人手法的运用，如"不应有恨，何事长向别时圆"，将月亮人格化，赋予其情感，增强了词的抒情性和亲切感。此外，"琼楼玉宇"、"婵娟"等典故的运用，丰富了词的文化内涵，体现了词人深厚的文学修养。

情感表达上，这首词蕴含着词人对亲人的深切思念和对人生哲理的思考。词作写于中秋之夜，当时苏轼因政治原因与弟弟苏辙分离，"不应有恨，何事长向别时圆"直接表达了词人对月亮在离别时反而圆满的疑问，流露出内心的无奈和伤感。但词人并未沉溺于伤感，而是通过哲理思考超越了个人情感的局限，达到了豁达的境界。主题思想方面，这首词探讨了人生与自然的关系，以及如何在面对人生聚散离合时保持平和心态的问题。词人认识到悲欢离合如同月亮的阴晴圆缺一样，是自然规律，难以避免，因此提出了"但愿人长久，千里共婵娟"的美好愿望，表达了对亲情友情的珍视和对生命永恒的向往。

文学价值上，这首词在艺术上达到了情景交融、虚实结合的高超境界，将个人情感与宇宙思考完美结合，开创了词作的新境界。它体现了宋代文人的哲学思考和人生态度，展现了"天人合一"的哲学思想。对后世文学产生了深远影响，"千里共婵娟"的意象成为后世文人表达思念之情和团圆之愿的经典表达。在文化传承方面，这首词已成为中国文化的重要组成部分，每逢中秋佳节，人们常常吟诵这首词，表达对团圆的向往和对亲人的思念。它不仅是中国古典文学的瑰宝，也是中华民族文化精神的重要载体，体现了苏轼作为文学大家的艺术成就和思想深度，展示了他豪放派词风的特点，对宋代词风的发展产生了重要影响。
