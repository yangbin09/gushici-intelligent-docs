## 📜 原文
```
洛阳城里见秋风，
欲作家书意万重。
复恐匆匆说不尽，
行人临发又开封。
```
## 🎯 主题
《秋思》主要表达了游子思乡的深切情感。诗人在洛阳城感受到秋风，引发了对家乡的思念，想要写信表达内心的万千思绪。然而，又担心匆忙之间无法说尽心中所想，以致于在送信人即将出发时又拆开信封，想要补充更多内容。这首诗通过描写写家书时的犹豫和反复，生动展现了游子在外对家乡亲人的眷恋之情，以及思念之情难以言表的复杂心理。全诗以简洁的语言，刻画了游子思乡这一普遍情感，体现了中国人重视家庭、思念亲情的传统美德。
## 🏗️ 结构
张籍的《秋思》是一首结构精巧的七言绝句，其章法布局和层次安排体现了唐代绝句的典型特点，同时又有其独特之处。

从章法布局看，这首诗严格遵循了绝句"起承转合"的传统结构模式。首句"洛阳城里见秋风"为"起"，点明地点和时节，奠定全诗基调；次句"欲作家书意万重"为"承"，承接上句引发的思乡之情，将情感具体化为写家书的行动；第三句"复恐匆匆说不尽"为"转"，情感递进，表达担忧；末句"行人临发又开封"为"合"，通过一个具体动作收束全诗，将情感推向高潮。

在层次安排上，诗歌呈现出由外到内、由抽象到具体的递进关系。第一句写外部环境（秋风），后三句则转入内心情感的描写；第二句表达抽象的思念之情，第四句则通过"又开封"这一具体行动将情感具象化。情感层次也逐步深入：从见秋风引发思乡之情，到想要写家书，再到担心表达不尽，最后到又开封，情感层层递进，愈发细腻。

诗歌的时空安排也颇具匠心。空间上，从洛阳城（诗人所在地）延伸至家乡（想象中的收信地）；时间上，从见到秋风的瞬间，到写家书的过程，再到送信人即将出发的时刻，虽时间跨度短，但情感浓度高。

此外，这首诗结构紧凑，四句环环相扣，逻辑严密。每句都是下一句的铺垫，最后一句成为前三句情感积累的自然爆发点。诗人通过借景抒情（秋风）、细节描写（又开封）等手法，将内敛而深沉的思乡之情含蓄地表达出来，使全诗在严谨的结构中蕴含丰富的情感。
## ✍️ 语言风格
张籍的《秋思》体现了唐代诗歌清新自然、朴实无华却又意蕴深远的语言风格。这首诗以简洁的语言表达了丰富的情感，展现了诗人高超的艺术技巧。

在用词方面，诗人选用了平实自然的日常词汇，如"洛阳城"、"秋风"、"家书"、"行人"等，没有生僻字和艰深词汇，使诗歌显得平易近人。特别是"秋风"一词，不仅点明了季节，更在中国传统文化中暗示了思乡之情，成为引发全诗情感的关键意象。"意万重"中的"万重"一词，既表达了思绪之多，又暗示了情感的复杂和深沉，用词精准而富有表现力。诗中动词的运用也尤为生动，"见"、"作"、"恐"、"说"、"发"、"开"等动词的连贯使用，使诗歌具有动态感，生动地描绘了诗人的行为和心理活动。

在修辞手法上，这首诗运用了多种技巧。首句"洛阳城里见秋风"是典型的借景抒情，通过描写秋风这一景物，引出思乡之情。"意万重"运用了夸张的修辞手法，形容思绪之多，情感之复杂。诗中还隐含了"想写"与"怕写不完"的对比，表现了诗人内心的矛盾和纠结。尤其是"行人临发又开封"这一细节描写，生动地刻画了诗人对家书的珍视和对亲人的牵挂，成为全诗的点睛之笔。全诗将思乡之情与写家书、开封的具体情景紧密结合，情感与情景相互交融，增强了诗歌的感染力。

在表达技巧上，诗歌从"见秋风"到"欲作家书"，再到"恐说不尽"，最后到"又开封"，情感和行为层层递进，结构严谨。诗人通过"开封"这一小动作，折射出对家人的深厚感情和思念，体现了"以小见大"的艺术手法。诗中没有直接表达思乡之情，而是通过写家书、开封等行为间接表达，情感表达含蓄而深沉。同时，诗中细腻地描写了诗人的心理活动，从想要写信到担心说不尽，再到最后开封，心理变化刻画得十分细腻。

总体而言，张籍的《秋思》通过平实的词语、鲜明的意象、精准的情感词汇和生动的动词，运用借景抒情、夸张、对比、细节描写等修辞手法，以及层次递进、以小见大、含蓄表达、心理描写等表达技巧，间接而含蓄地表达了思乡之情。特别是通过"行人临发又开封"这一细节描写，生动地传达了诗人对家人的深厚感情和思念，体现了唐代诗歌"言简意丰"的艺术特点。
## 📚 文化札记
《秋思》是唐代诗人张籍的代表作，此诗以简洁平实的语言，生动描绘了游子思乡的复杂心境。以下是诗中重要的文化背景注释：

洛阳：作为唐朝的东都，与长安并称"两京"，是当时政治、经济、文化中心之一。洛阳有着悠久历史，曾为十三朝古都，文化底蕴深厚。诗中"洛阳城里"点明了诗人所在之地，暗示他身处异乡，而非故乡。

秋风：在中国传统文化中，秋天常与思乡、离愁联系在一起。"秋风"不仅是自然现象，更是触发思乡之情的媒介。古人认为"秋气肃杀"，万物凋零的季节最容易引发游子思乡之情，如"秋风起兮白云飞，草木黄落兮雁南归"等诗句都传达了这种情感。

家书：古代通讯不便，书信是远在他乡之人与家人联系的主要方式。家书承载着游子对家人的思念和牵挂，是情感交流的重要载体。在古代文人生活中，家书具有特殊意义，往往经过反复斟酌才能写成。

行人：此处指负责送信的人，即信使。古代有官方驿使、私人信使，或外出时顺便捎信的人。由于交通不便，送信往往需要长途跋涉，十分辛苦，因此送信机会显得格外珍贵。

又开封：这一细节描写极为传神，反映了古代游子对家书的慎重态度。在纸笔珍贵的古代，一旦信使出发，就难以再有修改或补充的机会，因此"临发又开封"这一动作，生动表现了诗人思绪万千、唯恐遗漏重要内容的复杂心境。

张籍（约766-约830）：字文昌，原籍吴郡（今江苏苏州），后迁居和州乌江（今安徽和县）。他是韩愈的学生，与王建齐名，世称"张王"。其诗歌多反映社会现实，风格朴实自然，语言平易近人。

这首诗通过"见秋风"引发思乡之情，到"意万重"的情感积累，再到"说不尽"的忧虑，最后以"又开封"的细节收束，完整展现了游子思乡的复杂心理过程。诗中没有华丽辞藻，却以平实语言和生动细节，深刻表达了人类共通的思乡情感，这正是其千古传诵的原因。
## 🌅 创作背景
张籍的《秋思》创作于唐代中后期，约在唐德宗至唐宪宗时期（公元8世纪末至9世纪初），地点在洛阳城。当时张籍正客居洛阳，远离家乡。他出身贫寒，仕途坎坷，长期漂泊在外，生活并不如意。在这首诗创作时，他可能正经历着仕途上的挫折或是对故乡的深切思念。

唐代中后期，安史之乱后，唐朝国力由盛转衰。虽然仍有"元和中兴"的短暂繁荣，但社会矛盾日益尖锐，藩镇割据、宦官专权等问题严重。许多文人仕途不顺，四处漂泊成为常态。同时，唐代交通虽相对发达，但远距离通信仍主要依靠书信传递，且传递速度慢，信息不畅，这使得离乡在外的游子思乡之情更为浓烈。

这首诗的创作动机主要源于诗人的思乡之情。秋风起时，万物萧瑟，最容易引发人的思乡情绪。诗人看到洛阳城里的秋风，想起了远方的家乡和亲人，想要写信诉说思念之情，却又担心书信难以表达内心的万般思绪。"行人临发又开封"这一生动细节，不仅表现了诗人对家人的深厚感情和言不尽意的惆怅，也反映了当时通信不便的社会现实，体现了游子思乡这一永恒主题的深刻内涵。
## 💭 情感基调
《秋思》的情感基调主要表现为思乡、忧虑与不舍。诗人在洛阳城中感受到秋风，触发对家乡的思念，想要写家书却思绪万千，担心匆忙中无法表达完整。最后"行人临发又开封"一句，生动展现了诗人的犹豫与牵挂，体现出深沉的思乡之情和内心的不安。

氛围特点上，此诗营造出萧瑟、纷乱的秋日氛围。秋风作为传统意象，带来凄清之感；"意万重"和"又开封"的描写，表现出诗人内心的纷乱与不安。整首诗笼罩在一种淡淡的忧愁与牵挂之中，既有离别的伤感，又有对家人的深切思念，情感真挚而内敛。
## 🔍 赏析
这首诗以秋风起兴，勾勒出一幅深沉的思乡画卷。首句"洛阳城里见秋风"点明时节地点，秋风作为触发思乡之情的媒介，被诗人以"见"字具象化，使无形之风有了视觉感受。这种由外物触发内心感受的手法，体现了中国古典诗歌"情景交融"的艺术特色。全诗采用白描手法，不事雕琢，却情真意切，特别是"行人临发又开封"这一细节描写，以小见大，通过一个细微动作展现了诗人内心的复杂情感，体现了"以简驭繁"的艺术境界。

语言上，这首诗简洁朴素，字字珠玑，体现了唐代诗歌"言简意赅"的特点。"意万重"中的"万重"运用夸张手法，表现了诗人思绪的繁复；"又开封"中的"又"字含蓄地暗示了诗人反复斟酌的心理状态。诗人巧妙运用对比手法，将"欲作家书"与"恐匆匆说不尽"形成对照，展现了内心的矛盾。这种平实中见深情的语言风格，使诗歌具有强烈的感染力。

诗歌表达了诗人深沉的思乡之情和对家人的牵挂。秋风萧瑟之际，身处异乡的诗人内心涌起对家乡和亲人的思念，这种情感通过"欲作家书"直接表达。而"复恐匆匆说不尽"则表现了诗人对家人的深厚感情，有千言万语想要倾诉，却又担心无法完全传达。更深层次上，这首诗探讨了游子思乡这一永恒主题，表现了人在异乡的孤独感和对家的眷恋，具有普遍的人性意义。

作为唐代思乡诗的代表作，《秋思》的文学价值在于通过简洁的语言和生动的细节，深刻表达了人类共通的思乡情感。它不仅成为唐代思乡诗的经典之作，也为后世诗人提供了情感表达的范本。诗中"行人临发又开封"的细节描写，成为中国古典诗词中细节描写的典范，影响了无数后世创作。这首诗所表达的思乡之情具有超越时空的普遍意义，使它能够引起不同时代、不同地域读者的共鸣，体现了中国古典诗歌的永恒魅力。
