## 📜 原文
```
东皋薄暮望，徙倚欲何依。
树树皆秋色，山山唯落晖。
牧人驱犊返，猎马带禽归。
相顾无相识，长歌怀采薇。
```
## 🎯 主题
《野望》这首诗主要表达了诗人王绩的孤独迷茫与归隐情怀。诗中通过秋日薄暮时分的远望，描绘了一幅萧瑟凄清的秋景图，同时展现了田园生活的宁静画面。诗人在东皋徘徊，内心无所依靠，面对"树树皆秋色，山山唯落晖"的凄凉景象，更加强化了其内心的孤独感。当他看到牧人驱犊、猎人带禽归家的场景时，却感到"相顾无相识"，最终以"长歌怀采薇"表达了对隐居生活的向往。全诗流露出诗人仕途失意后的失落情绪，以及寻求心灵归宿的愿望，体现了唐代文人在仕途不顺时常见的归隐思想。
## 🏗️ 结构
《野望》作为王绩的代表作，其结构特点体现了唐代早期律诗的成熟艺术。从章法布局看，这首五言律诗严格遵循了律诗的格式要求，分为首联、颔联、颈联和尾联四个部分，各联之间既有独立功能，又紧密联系，形成完整统一的艺术整体。

首联"东皋薄暮望，徙倚欲何依"为全诗定下基调，交代了时间(薄暮)、地点(东皋)和人物状态(徙倚)，同时以设问方式暗示诗人内心的彷徨与不安，为全诗的情感走向埋下伏笔。这一联起到了提纲挈领的作用，统领全诗内容。

颔联"树树皆秋色，山山唯落晖"承接首联的"望"字，描绘诗人所见之景。此联采用对仗手法，"树树"对"山山"，"皆"对"唯"，"秋色"对"落晖"，形式工整，内容上则从宏观角度展现秋天傍晚的自然景观，视野开阔，气势恢宏。

颈联"牧人驱犊返，猎马带禽归"继续描写景物，但视角从自然景观转向人文活动。同样运用对仗，"牧人"对"猎马"，"驱犊"对"带禽"，"返"对"归"，展现了农村生活的具体场景。与颔联相比，这一联的描写更为具体、细致，体现了由远及近、由大到小的空间转换。

尾联"相顾无相识，长歌怀采薇"是全诗的收束，从前面的景物描写转向诗人的内心感受。这一联点明了诗人的孤独处境，以及通过"长歌怀采薇"这一典故表达的超脱情怀，完成了从外在景物到内心情感的转变，形成了情景交融的艺术效果。

在层次安排上，《野望》呈现出多维度的递进结构：时间层次上，从"薄暮"开始，通过"返"、"归"等词语暗示时间的推移，形成完整的时间序列；空间层次上，从远处的"山山"到近处的"树树"，再到具体的"牧人"、"猎马"，最后聚焦到诗人自身，形成由外到内的空间转换；情感层次上，从"徙倚欲何依"的彷徨，到观察景物时的平静，再到"相顾无相识"的孤独，最后以"长歌怀采薇"达到超脱，形成情感的起伏变化。

此外，全诗在意象安排上也遵循了"自然-人文-情感"的递进关系，自然意象(东皋、树、山、落晖)和人文意象(牧人、犊、猎马、禽)共同构建了外在环境，而情感意象(徙倚、相顾、长歌、采薇)则直接表达了诗人的内心世界，形成了从物到我的完整意境。

《野望》的结构特点体现了王绩作为初唐时期诗人的艺术成就，其严谨的章法布局、清晰的层次安排以及情景交融的艺术处理，为后世律诗创作提供了典范。
## ✍️ 语言风格
王绩的《野望》是初唐诗歌的代表作之一，其语言风格体现了清新自然、朴实无华的特点，同时蕴含深厚的情感内涵。

在用词方面，这首诗选词精炼而富有表现力。"东皋"、"薄暮"直接点明时间地点，简洁明了；"秋色"、"落晖"等自然意象词汇营造出秋日黄昏的特定氛围；"牧人"、"犊"、"猎马"、"禽"等农村生活场景的词汇朴实自然，展现了田园生活的真实面貌。特别值得注意的是"采薇"一词的运用，这一典故出自《诗经》，指伯夷、叔齐隐居首阳山采薇而食的故事，体现了诗人深厚的文学修养，也使诗歌内涵更加丰富。

修辞手法上，这首诗运用了多种技巧。对偶是最显著的特点，如"树树皆秋色，山山唯落晖"、"牧人驱犊返，猎马带禽归"，使诗歌结构严谨，节奏鲜明。叠词的运用如"树树"、"山山"，增强了诗歌的节奏感和形象性，强化了秋景的普遍性。首句"徙倚欲何依"则运用了设问的修辞，暗示诗人内心的迷茫，引发读者思考。末句"长歌怀采薇"的用典，含蓄地表达了诗人对隐居生活的向往。

整体而言，这首诗的语言风格具有以下特点：清新自然，不事雕琢；含蓄委婉，通过景物描写和典故运用表达情感；意境深远，简单的语言中蕴含着深刻的情感和哲理；融情于景，诗人将自己的情感融入景物描写之中；朴实中见真情，语言朴实无华，但情感真挚。

《野望》的语言风格既体现了王绩个人的诗歌特色，也反映了初唐诗歌从六朝绮丽风格向盛唐雄浑风格过渡的特点，展现了初唐诗歌的艺术成就。
## 📚 文化札记
东皋：地名，指东边的高地或山丘。"皋"在古代通常指水边的高地。王绩自号"东皋子"，表明此地可能是他常去之处，也是他精神寄托的所在。在唐代文人的地理观念中，东皋常被视为远离尘嚣、适合隐居的地方。

薄暮：傍晚时分，太阳即将落山的时候。在中国古典诗歌中，黄昏时分常被用来表达忧愁、孤独等情感，如李商隐"夕阳无限好，只是近黄昏"。

徙倚：徘徊、来回走动的样子，形容心神不宁、犹豫不决的状态。这一词语生动地表现了诗人内心的彷徨和无所适从。

秋色：秋天的景色，通常指树叶变黄、变红的景象。在中国传统文化中，秋天常与萧瑟、肃杀、离愁等情感联系在一起，如"自古逢秋悲寂寥"。

落晖：夕阳的余晖。在中国古典诗词中，夕阳常被用来象征衰败、离别或时光流逝，如"夕阳西下，断肠人在天涯"。

牧人驱犊返：描绘了农村日常生活的场景。在古代农业社会中，牧人是社会底层劳动者，这一形象反映了诗人对普通民众生活的关注。

猎马带禽归：猎人带着猎物归来的场景。在古代，狩猎既是贵族的娱乐活动，也是普通民众获取食物的方式之一。这一意象与牧人驱犊共同构成了黄昏时分的乡村生活图景。

相顾无相识：环顾四周，没有认识的人，表达了诗人的孤独感和与周围环境的疏离感。这种情感在古代文人中较为常见，特别是那些仕途不顺或思想超脱的诗人。

长歌怀采薇：这是全诗的点睛之笔，包含了一个重要典故。"采薇"出自《诗经·小雅·采薇》，原描写戍边士兵的艰辛生活和对家乡的思念。后来，"采薇"成为隐士生活的象征，特别是指伯夷、叔齐隐居首阳山，采薇而食的典故。伯夷、叔齐是商末周初的隐士，他们反对周武王伐纣，隐居在首阳山，以采集野菜（薇）为食，最终饿死。他们的故事成为后世文人歌颂的高洁隐士典范。王绩在此用"怀采薇"表达了对隐逸生活的向往和对高洁品格的追求。

王绩（约585-644），字无功，号东皋子，唐代诗人。他生活在隋末唐初的动荡时期，曾仕隋，入唐后也做过官，但不久便归隐。他的诗歌多表现隐逸情怀和对自然的热爱，风格清新自然，对唐代山水田园诗有一定影响。这首《野望》正是他归隐后所作，反映了他对隐逸生活的向往和对现实社会的疏离感。
## 🌅 创作背景
王绩的《野望》创作于唐太宗贞观年间（约627-649年），地点在东皋（今河南省境内），这也是王绩隐居之地，他的号"东皋子"即由此而来。诗中"东皋薄暮望"明确点明了创作地点和傍晚时分的具体时间。

创作此诗时，王绩正处于隐居状态。他经历了隋末的战乱和唐朝的建立，曾在隋朝做官，隋亡后归隐，唐初又短暂出仕，最终选择归隐田园。从诗中"徙倚欲何依"和"相顾无相识"等句可见，他虽然身处自然美景之中，内心却充满孤独与彷徨，与周围环境格格不入，表现出一种无所依归的复杂心境。

从历史背景看，此诗创作于隋末唐初的大变革之后。贞观年间虽是政治清明、社会稳定的盛世，但经历过朝代更替和战乱的人们，特别是像王绩这样的知识分子，内心仍充满矛盾和不安。他们对新政权既有期待，又有保留，这种复杂心态在诗中有所体现。

《野望》的创作动机源于王绩对隐居生活的思考和对社会现实的感受。诗中描绘了秋日傍晚的田园景色，但透露出的却是一种孤独情绪。诗末"长歌怀采薇"引用《诗经》典故，既表达了对隐士生活的向往，又暗示了内心的矛盾和不安，反映了他虽隐居山林却无法完全超脱世俗的复杂心理状态。
## 💭 情感基调
《野望》的情感基调主要表现为孤独、迷茫与怀古。诗中"徙倚欲何依"直接表达了诗人内心的无依无靠之感，"相顾无相识"则强化了这种孤独情绪。结尾"长歌怀采薇"流露出对隐士生活的向往，体现了怀古之情。

氛围特点上，这首诗呈现出萧瑟、寂寥与凄凉的特点。"树树皆秋色，山山唯落晖"描绘了秋日黄昏的萧条景象，夕阳与秋色共同营造出凄凉氛围。尽管有"牧人驱犊返，猎马带禽归"的生活场景，但并未打破整体的寂寥感，反而衬托出诗人与周围环境的疏离。整首诗笼罩在一种宁静而略带忧伤的意境之中。
## 🔍 赏析
王绩的《野望》是唐初山水田园诗的代表作，展现了诗人隐居东皋时的复杂心境。从意境营造看，诗人以"薄暮"为时间背景，通过"树树皆秋色，山山唯落晖"的叠词手法，构建了一幅广阔萧瑟的秋日黄昏图景。颔联与颈联的对仗工整，"牧人驱犊返，猎马带禽归"的农村生活场景与自然景色相映成趣，形成动静结合的艺术效果。诗人巧妙运用情景交融手法，使外在景物与内心情感相互渗透，营造出孤独而深远的意境。

语言特色方面，全诗简洁凝练却意蕴丰富。叠词"树树"、"山山"的运用不仅增强了音韵美，也强化了秋色的普遍性和落晖的广泛性。诗中"徙倚欲何依"的设问和"相顾无相识"的直抒胸臆，展现了诗人内心的迷茫与孤独。尾联"长歌怀采薇"引用《诗经》典故，既增添了文化底蕴，又表达了对隐居传统的向往，体现了诗人深厚的文学修养。

情感表达上，诗歌流露出诗人复杂的内心世界。首联"徙倚欲何依"直接点明其无所依靠的迷茫；中间两联描绘的秋景与农村生活，既是对自然的欣赏，也暗含诗人对生活的思考；尾联则表达了诗人在陌生环境中的孤独感，以及对隐居生活的坚守。这种情感的复杂性，反映了唐初文人在仕隐之间的矛盾心态，也体现了诗人对人生意义的深刻思考。

从文学价值看，《野望》作为唐代早期山水田园诗的典范，对后世产生了深远影响。它打破了六朝诗风的浮艳，开创了唐代山水诗的新风格。诗中对自然的细腻观察和真实情感的表达，为王维、孟浩然等盛唐山水诗人提供了借鉴。同时，诗中表现的隐士情怀和文化坚守，也成为中国文学中的重要主题，具有永恒的艺术魅力和人文价值。
