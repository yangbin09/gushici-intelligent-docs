## 📜 原文
```
东皋薄暮望，徙倚欲何依。
树树皆秋色，山山唯落晖。
牧人驱犊返，猎马带禽归。
相顾无相识，长歌怀采薇。
```
## 🎯 主题
《野望》的主要主题思想是表达了诗人对隐逸生活的向往与对现实的疏离感。诗中，王绩通过秋日傍晚登高远望的场景，描绘了一幅萧瑟凄凉的秋景图，同时以牧人、猎人归家的生活场景与自己"相顾无相识"的孤独处境形成鲜明对比，流露出内心的迷茫与孤独。"长歌怀采薇"一句直接点明诗人向往如古代隐士般远离尘世、归隐山林的生活态度。这首诗反映了唐代士人在仕途不顺时普遍存在的归隐情怀，体现了诗人对自然生活的向往和对世俗羁绊的超脱。
## 🏗️ 结构
《野望》是王绩创作的一首五言律诗，其结构特点体现了唐代律诗的典型特征，下面从章法布局和层次安排两方面进行分析。

章法布局方面，这首诗遵循了律诗的典型结构：首联、颔联、颈联、尾联四联八句。首联"东皋薄暮望，徙倚欲何依"为起，点明时间（薄暮）、地点（东皋）和诗人状态（徙倚欲何依），奠定全诗基调。颔联"树树皆秋色，山山唯落晖"为承，承接首联的"望"字，具体描绘所见秋景。颈联"牧人驱犊返，猎马带禽归"为转，由自然景色转向人事活动，增添动态元素。尾联"相顾无相识，长歌怀采薇"为合，收束全诗，表达诗人的孤独与向往。这种起承转合的结构安排使全诗层次分明，脉络清晰。

层次安排上，诗歌呈现多维度的层次感。在空间层次上，颔联描绘远景（树树秋色、山山落晖），颈联则刻画近景（牧人驱犊、猎马带禽），形成由远及近的空间转换，增强了画面的立体感。在动静结合上，颔联的静景与颈联的动景形成对比，使诗歌画面更加生动丰富。情感层次上，从首联的孤独迷茫，到中间两联的客观观察，再到尾联的内心独白，形成了一个由内而外再回归内心的情感循环。

此外，诗歌还体现了由景及情的层次递进，前三联侧重写景，尾联转为抒情，符合古典诗歌"情景交融"的审美特点。在时间层次上，从"薄暮"到"返"、"归"，暗示了黄昏时分的时间流逝，构成了一个完整的时间片段。

值得注意的是，颔联和颈联的对仗极为工整，"树树"对"山山"，"皆秋色"对"唯落晖"；"牧人"对"猎马"，"驱犊返"对"带禽归"，体现了律诗的形式美。同时，尾联的"相顾无相识"与首联的"徙倚欲何依"形成呼应，使全诗结构更加完整。

总的来说，《野望》的结构严谨有序，层次丰富多样，既有律诗的形式美，又有内容上的层次感，展现了王绩作为初唐诗人对律诗形式的熟练掌握和艺术构思的精妙。
## ✍️ 语言风格
王绩的《野望》是一首典型的唐代五言律诗，其语言风格体现了初唐诗歌向盛唐过渡的特点。从用词和修辞手法等方面分析，这首诗具有以下特点：

在用词方面，王绩选择了简洁而富有表现力的词汇。如"薄暮"一词精准地描绘了傍晚时分的朦胧氛围；"徙倚"二字生动地刻画了诗人徘徊踌躇的姿态；"秋色"和"落晖"则营造出萧瑟凄清的意境。这些用词平实自然，不事雕琢，却能准确传达诗人的情感和眼前的景象，体现了诗人深厚的语言功底。

在修辞手法上，这首诗运用了多种技巧。首先是对仗的运用，"树树皆秋色，山山唯落晖"和"牧人驱犊返，猎马带禽归"形成了工整的对仗，增强了诗歌的节奏感和音乐美。其次是叠词的使用，"树树"和"山山"的重复强调了秋色的普遍性和落晖的广泛性，使画面更加立体。此外，诗人还巧妙地运用对比手法，通过牧人、猎马的归家与诗人的"相顾无相识"形成鲜明对比，突出了诗人的孤独感。

诗中还融入了典故运用，末句"长歌怀采薇"引用《诗经·召南·草虫》中的典故，含蓄地表达了对隐逸生活的向往，显示了诗人深厚的文学修养。这种用典方式既典雅含蓄，又丰富了诗歌的文化内涵。

从整体语言风格来看，《野望》呈现出清新自然、含蓄蕴藉的特点。诗人善于将情感融入景物描写之中，使外在景物与内在情感相互呼应，达到情景交融的艺术效果。诗歌语言简约而不简单，短短四十个字包含了丰富的内容和情感，体现了"言有尽而意无穷"的艺术境界。

与盛唐时期李白、杜甫等人的诗歌相比，王绩的《野望》更加内敛、含蓄，情感表达不如盛唐诗人那样奔放，而是更接近魏晋时期隐逸诗人的风格，表现了对自然和隐逸生活的向往。这种语言风格特点，使《野望》成为唐代早期诗歌中的佳作，也为后世诗歌创作提供了有益的借鉴。
## 📚 文化札记
《野望》是唐代诗人王绩的代表作，体现了作者隐居生活的情怀与心境。以下是诗中重要的文化背景注释：

东皋：地名，位于今山西省河津市。王绩晚年隐居于此，并自号"东皋子"。皋指水边高地，东皋即东边的高地。古代文人常以隐居地为号，体现其与自然的亲近及对隐逸生活的认同。

薄暮：傍晚时分，太阳即将落山之时。在古典诗词中，薄暮常带有忧伤、孤独的情感色彩，是文人表达感伤情绪的典型时间背景。

徙倚欲何依：徙倚意为徘徊、踌躇。此句表达了诗人的孤独与迷茫，不知依靠什么，归属何方，反映了隐士内心的矛盾与不安。

树树皆秋色，山山唯落晖：描绘秋日傍晚景色。秋色在传统文化中象征萧瑟凄凉，也暗含人生暮年的感慨；落晖即夕阳余晖，暗示时光流逝、人生将尽的哲思。

牧人驱犊返，猎马带禽归：描写农村生活场景，展现与自然和谐共处的田园生活。这种场景是隐逸诗中常见意象，与诗人的孤独形成对比，突出其"无归"之感。

相顾无相识：表达诗人环顾四周却无熟人的孤独感。这源于中国古代文人"知音难觅"的传统感慨，与"伯牙绝弦"等典故相关，表现高洁之士难以找到志同道合者的孤独。

长歌怀采薇：全诗最重要的典故。"采薇"出自《诗经·小雅·采薇》，描述戍边士卒的辛苦与思乡。后成为隐逸生活的象征，尤其与伯夷、叔齐的故事相关。据《史记》记载，商末周初，伯夷、叔齐因反对周武王伐纣，隐居首阳山，采薇而食，最终饿死。因此"采薇"象征高洁的隐逸品格和不与世俗同流合污的精神。王绩借此表达对古代高士隐逸生活的向往，表明自己归隐的心志。

此诗创作背景：王绩(约585-644年)，字无功，出身官宦世家，但仕途不顺，几次出仕又归隐。他性格孤高，不喜交游，晚年隐居东皋。这首诗反映了隋末唐初动荡时期，部分文人对政治失望而选择归隐的心态，体现了中国古代文人的隐逸情怀和价值观。
## 🌅 创作背景
《野望》创作于唐初贞观年间(627-649年)，地点在东皋，即今山西省河津市附近，是王绩辞官后的隐居之地。王绩出生于隋唐之际的官宦世家，祖父王通是隋代著名学者。他本人曾在隋朝和唐朝初年担任过官职，但仕途不顺，对官场的尔虞我诈感到厌倦，最终选择辞官归隐东皋。诗中"徙倚欲何依"正反映了他内心的彷徨和对人生归宿的思考；"相顾无相识"则表达了他隐居生活中的孤独感。

当时的历史背景是，经历了隋末战乱后，唐朝初年社会正在恢复稳定。唐太宗李世民在位的贞观年间，政治清明，社会安定，文化繁荣。然而，对于像王绩这样经历过隋末动乱的知识分子来说，他们对政治仍保持一定距离，许多文人选择隐居或对官场持观望态度。

王绩创作《野望》的动机主要源于对官场生活的失望和对自由生活的向往。诗中描绘的"树树皆秋色，山山唯落晖"展现了秋日黄昏的壮美景色，体现了他对自然的敏锐观察和深厚感情。"牧人驱犊返，猎马带禽归"则描绘了乡村生活的恬静与和谐。结尾"长歌怀采薇"引用《诗经》典故，表达了他对古代隐士高洁品格的追慕，也反映了他对人生意义的深层思考。这首诗既是对自然景色的描绘，也是作者内心情感的真实流露，体现了唐初隐逸诗的特点。
## 💭 情感基调
《野望》的情感基调可概括为孤独迷茫、忧郁怀古、超脱恬淡。诗中"徙倚欲何依"、"相顾无相识"直接表现了诗人的孤独与迷茫；"长歌怀采薇"则流露出对古代隐士生活的向往，体现怀古之情。整体情感虽带忧郁，却又透露出一种超然物外的恬淡心境。

氛围特点则表现为萧瑟宁静、凄凉悠远。"树树皆秋色，山山唯落晖"勾勒出秋日黄昏的萧瑟凄凉景象；而"牧人驱犊返，猎马带禽归"则描绘了乡村生活的宁静祥和。全诗在时空上营造出一种悠远之感，使读者仿佛置身于那秋日黄昏的东皋，感受诗人内心的孤独与超脱。
## 🔍 赏析
《野望》是王绩隐居东皋时的代表作，展现了他作为初唐重要诗人的艺术才华和思想深度。从意境营造来看，诗人以"东皋薄暮望"开篇，点明时间、地点与动作，为全诗奠定基调。诗中通过"树树皆秋色，山山唯落晖"的叠用手法，强调秋色的普遍性和落晖的广泛性，营造出开阔而宁静的秋日黄昏图景。中间两联远近结合、动静相映，既有静态的自然景色，又有动态的牧归场景，形成一幅生动的田园画卷。

在语言特色方面，这首诗语言简洁凝练却意蕴丰富。"树树"和"山山"的叠用增强了节奏感和画面感；对仗工整如"树树皆秋色，山山唯落晖"与"牧人驱犊返，猎马带禽归"，形成整齐和谐的美感。诗中还运用了对比手法，牧人与猎人、犊与禽、返与归的多层次对比，丰富了诗歌内容。特别是"长歌怀采薇"中的"采薇"典故，出自《诗经·小雅·采薇》，既增添了文化内涵，又深化了主题。

情感表达上，诗人通过"徙倚欲何依"和"相顾无相识"直抒胸臆，表达了内心的孤独与迷茫。当别人都有家可归时，诗人却感到无所依靠，这种反差强化了情感张力。整首诗流露出诗人对动荡时局的失望和对隐居生活的向往，反映了隋末唐初文人的普遍心态。

从文学价值看，《野望》继承了南北朝诗歌精华又有创新，对后世诗歌创作产生深远影响。特别是诗中情景交融的艺术手法，成为后人借鉴的典范。这首诗不仅展现了王绩深厚的诗歌功底，也为研究初唐社会文化提供了重要资料，在中国诗歌史上占有重要地位。
